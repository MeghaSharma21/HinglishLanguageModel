 
 diyah premata megawati setiyavati sukarnoputree,, jinhen megawati sukarnoputree ke naam se janaa jaataa he, indonesian raajnneetijna or vipakshi dal peediai-pi kii netaa he. inhone 23 july 2001 se lekar 20 oktober 2004 tak indonesia ke raashtrapati ke rup main kaarybhaar sambhala tha or desh kii pehali mahila raashtrapati hone kaa gaurav hasil kiya tha. iske alaava ve aisi pehali raajnneetijna hai, jo indonesia ke swatantra hone ke baad paidaa huin. megawati indonesia ke pehle raashtrapati sukarno kii putree hai.
 megawati kaa janm yogyakarta main sukarno tathaa fatmavati ke ghar main hua tha. fatmavati unki nau patniyon main se ek thi. unka laalan paalan unke pita ke hi merdek mahal main hua tha. vah apane pita ke mehmaanon ke saamne nrity karti thi tathaa bagwani ko unhone apana shauk bana liya tha. jab megawati 19 saal kii thi tab unke pita ko desh kii satta chhodani padi tathaa uske baad suhartho ke netritv main sarkaar bani. suhartho ne sukarno tathaa unke parivaar ko raajnneeti ke nepathy main bhej diya.
 megawati ne pradjadeiran vishvidhyalay, bandung main krishi ke adhyayan ke liye pravesh liya parantu unke pita ke rajnatik patan ke kaaran unhen yahan se nikaal diya gaya. 1970 main pita kii mrityu ke baad unhone indonesia vishvidhyalay main manovigyaan padhane ke liye pravesh liya magar do saal baad 1972 main unhen yahan se bhi nikaal diya gaya. vah islam dharm kii anuyaayi hai parantu parmparik java dharm ko bhi maanati hai.
 sukarnoputree unka gotra hai, na ki paariwarik naam; kyonki java ke nivasee ke praayah paariwarik naam nahin hote. unke naam praayah mega yaa megawati kaha jaataa he, jo ki sanskrit bhashaa ke ek shabd he jiskaa arth "badalon kii devi" he. shri satya sai primary skool main apane vyaakhyaan ke dauraan unhone kaha ki unke pita ke aagrah par odisha ke mukhyamantri biju patnayak ne unka yah naam rakhaa he.
 1986 main ek samaaroh ke dauraan suhartho ne sukarno ko ghoshit nayak kaa darja diya. is samaaroh main megawati bhi upasthit thim. is samay tak megawati swayan ko gruhani ke rup main pesh karti thim magar 1987 main unhone indonesian democratic party kii sadsyta lee tathaa devan perwakilan rakyaat kii sadsyta paane main lag gayin. peediai ne bhi megawati ko apni janata ke madhya apni chhavi banane ke uddeshy se party main sweekaar kar liya. apane bhaashan dene kii kala main nipun na hone ke baavjood bhi sukarno kii beti hone ke kaaran unki lokapriyata badhane lagi. party kii karaari haar ke baavjood bhi vah indonesian sansad ke liye chuni gayin.
 disambar 1993 main vah apni party peediai kii adhyaksh bhi bani. unki adhyakshata main party kii lokapriyata main baari oochaal hua.       
 23 july 2001 ko megawati indonesia ke raashtrapati pad par virajman hui. is prakaar se vah kisi muslim bahul desh kii satta ke sheersh pad par pahunchne vaali chhathin mahila bani. unase pehle pakistan main benazir bhutto, bangladesh main khalida zia, turki kii tansu siller, bangladesh main hi shaikh hasina tathaa senegal main maame maadior boye muslim bahul desh kii satta ke sheersh pad tak pahaunchi thi.  
 unke raashtrapati kaal ko janata ne prarambh main to haathon haath liya magar jald hi unki nitiyon main anischitata, spasht vichaardhaara main kami tathaa mahatvapurn nitigat nirnyon main niskriyata jaisi kamiyaan ujagar hone lagi. 
 2004 main desh main raashtrapati pad ke liye pehali baar pratyaksh chunaav hue. unhen ummid thi kii vah ek muslim bahul desh main ek baar phir se satta sanbhaalengi parantu unhen sucilo bambang yudhoyono ke viruddh dhokhe se dwitiya charan main 61 ke mukaable 39 pratishat mat mili, or vah haar gayin. iske baad unhone naye raashtrapati kaa swaagat bhi nahin kiya, na hi unhen kabhi badhai di.
 megawati kii pehali shaadi 1 june 1968 main sena main pratham leftinent surindo supjarso se hui. vah 22 janvari 1970 ko papua main ek vimaan durghatana main maare gayi. 27 june 1972 ko unhone misr ke kootneetijna hasan gamaal ahamed hasan se shaadi kii. 3 mahine baad hi unki yah shaadi dharmik nyaayalay se amaanya ghoshit ho gayi. iske baad 25 march 1973 ko unhone taific kimas se shaadi kii jinaki mrityu 8 june 2013 main hui. unke teen bachche mohammad rijki pramata, mohammad prananda tathaa puaan maharani hai.
