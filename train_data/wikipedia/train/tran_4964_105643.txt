 buddhacharitam, sanskrit kaa mahakavy he. iske rachayita ashwaghosh hai. isamen gautam buddha kaa jeevancharit varnit he. isaki rachanakaal duusari shatabdi he. 
 san 420 main dharmaraksha ne iskaa chini bhashaa main anuwaad kiya tathaa 7veen evam 8veen shatee main iskaa atyant shuddh tibeti anuwaad kiya gaya. durbhagyavash yah mahakavy mul rup main apurna hi uplabdh he. 28 sargon main virachit is mahakavy ke dwitiya se lekar trayodash sarg tak tathaa pratham evam chaturdash sarg ke kuch ansh hi milte hai. is mahakavy ke shesh sarg sanskrit main uplabdh nahin he. is mahakavy ke puure 28 sargon kaa chini tathaa tibeti anuwaad avashya uplabdh he. 
 buddhacharit ke 28 sargo main bhagavatprasuti, sanvegotpatti, abhinishkaraman, tapovana pravesh, ant:poor vilaap, kumaranveshanam, shrenbhigamanam, buddhatvaprapti, mahashishyana pravrajya pramukh hai.
 is mahakavy kaa armbh buddha ke garbhaadhan se tathaa isaki parinti buddhatv-prapti main hoti he. yah mahakvy bhagwaan buddha ke sanghariumay safal jivan kaa jvalant, ujjwal tathaa moort chitrapat he. iskaa chini bhashaa main anuwaad paanchavin shatabdi ke prarambh main dharmaraksh, dharmakshetra athwa dharmakshar naamak kisi bhartiya vidwaan ne hi kiya tha tathaa tibeti anuwaad naveen shatabdi se purvavarti nahin he. isaki katha kaa rup-vinyas valmikikrut ramayana se miltaa-julata he.
 buddhacharit 28 sargon main tha jisamen 14 sargon tak buddha ke janm se buddhatv-prapti tak kaa varnan he. yah ansh ashwaghosh krit mul sanskrit sampoorn uplabdh he. keval pratham sarg ke prarambh saat shloke or chaturdash sarg ke battis se ek sau baarah tak mul main nahin milte hai. chowkhamba sanskrit siries tathaa chowkhamba vidyabhavan kii prerana se un shlokon kii rachana shri ramachandradaas ne kii he. unheen kii prerana se is ansh kaa anuwaad bhi kiya gaya he.
 15 se 28 sargon kii mul sanskrit prati bharat main bahut dinon se anuplabdh he. uskaa anuwaad tibeti bhashaa main mila tha. uske aadhaar par kisi chini vidwaan ne chini bhashaa main anuwaad kiya tathaa oxford vishvavidhyalay se sanskrit adhyaaapak doctor johnston ne use angreji main likhaa. iskaa anuwaad shrisuryanarayan choudhary ne hindi main kiya he, jisko shri ramachandradaas ne sanskritpadyamay kavy main parinat kiya he.
