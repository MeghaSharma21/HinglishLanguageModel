 vandana kataaria ek bhartiya maidani hockey khilaadi hai. yah maidani hockey ke bhartiya rashtriya team main khelti hai. vandana 2013 main desh main sabase adhik gol karne main safal raheen thi. yah junior mahila vishwa cap main kaansya padak vijeta bani. yah spardhaa germany main hua tha or inhone paanch gol mar kar is spardhaa main teesraa sabase adhik gol karne main safal raheen. yah ab tak 130 spardhaa main 35 gol karne main safal rahi hai. 
 vandana kataaria kaa janm 15 april 1992 ko uttar pradesh main hua tha. uske pita nahar singh bhel main kaam karte hai. yah haridwar jile ke roshnabad gaanv se hai. yah pichley kuch varshon main bharat ke sabase behatar or agrim khilaadiyon main se ek hai. inhone pehali baar junior antarrashtriya spardhaa main 2006 main bhaag liya or senior antarrashtriya spardhaa main uske chaar varshon ke baad bhaag liya.
 kataaria ko bhartiya junior team main 2006 main liya gaya or 2010 main inhen senior rashtriya team liya gaya. yah 2013 main germany main hue junior vishwa cap main kaansya padak jeetne main safal raheen. is dauraan yah is spardhaa main sabase adhik gol karne vaali khilaadi bani. inhone 4 khelon main 5 gol daage the. ek saakshaatkaar main inhone kaha ki kaansya padak paanaa unke pasandida palon main se ek he. mere pita ko media waalon ne bulaayaa or tak unki aankhon main ansu the. to, mere pita ko garv karana mere hockey ke safar main sabase achche pal hai. inhone 100ve topi canada ke khilaaf 2014 rashtramandal khelon main glassgo, scotland main jiti thi.
