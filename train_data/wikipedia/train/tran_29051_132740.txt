 vartmaan main iskaa puraa naam deen dayaal upaadhyaay gorakhpur vishvavidhyalay, gorakhpur he. yah uttar pradesh ke gorakhpur nagar main railway stesan se lagbhag aadhe kimi kii duuri par he. isaki sthapana san 1957 main hui thi.
 yah vishvavidhyalay 1956 main astitv main aayaa or 1 sitambar 1957 se kaary karnaa armbh kar diya. shuru main keval kala, vaanijya, vidhi or shiksha sankaay armbh kiye gayi. 1958 main vigyaan sankaay armbh hua. ejineering, chikitsa or krishi ke sankaay baad ke varshon main aaye. pehle avadh vishvavidhyalay faijabad tathaa jonpur vishvavidyalya ke kuch mahavidhyalya bhi isi vishvavidyalya se sambaddh the.
 is vishvavidyalya kaa praangan lagbhag 300 acre main pasra hua he. shuru main vishvavidhyalay pant block or majithia block main simit tha. baad ke varshon main kendriya pustakalya, kala blaak, prashasnic bhawan, vidhi sankaay, chaatrasangh bhawan, vyaayaamashaala, swasthya kendra, computer kendra, aadi bane. iske alaava vartmaan samay main vaanijya or shiksha sankayon ke liye alag-alag bhawan hai or gruhavigyaan, bhugol, manovigyaan, samajashaastra, hindi, pracheen itihaas, fine arts evam sangeet, raajnneeti vigyaan, vayask shiksha aadi ke bhi alag bhawan hai. diksha bhawan ek vishaal bhawan he jisamen bade-bade kamare evam ek aditorium he. isamen purv snaatak chaatraon kii kakshayen lagatee hai or pariksha haal ke rup main iskaa upyog kiya jaataa he. elahabaad bank kii ek shakha or ek daakghar bhi vishvavidhyalay ke hi bhaag hai. saamaajiki-manviki bhawan bane abhi kuch hi varsh hue hai. vyavsaay prabandhan vibhaag tathaa biotechnology vibhaag ke bhawan nirmit ho chuke hai.
 is vishvavidyalya main adhyaaapak evam anya karmchaariyon ke liye 125 se adhik gharon kii vyavastha he. lagbhag itane hi ghar gaiyr-shaikshik karmchaariyon ke liye hai.
 is vishvavidhyalay main nimnlikhit sankaay hai-
 uttar pradesh rajya vishvavidhyalay adhinium ke antargat 1956 i0 main sthapit gorakhpur vishvavidhyalay main kala sankaay ko pratham sankaay hone kaa gaurav praapt he. iske pratham satr main angreji, shikshashastra, manovigyaan, sansqat evam puraatatv vibhaag main kakshayen prarambh hui. sthapana se lekar ajtak 50 varsho main sankaay ne uttarottar vikaas kiya he. samprati is sankaay main 13 vibhaag hai tathaa iske antargat vishvavidhyalay ke sarvaadhik chaatr adhyayan hetu alag-alag vyavastha he. chaatraon kii diksha bhawan main tathaa chaatro kii kakshayen kala sankaay bhawan main chalati hai. vidhyaarthiyon ko bharat ke gauravshaalee ateet se parichit karaane ke uddeshy se raashtagaurav pathraykram varsh 2002 se prarambh kiya gaya he. snaatak pratham varsh ke chaatr-chaatraon ke liye isaki kakshayen prarambh kar kala sankaay is kshetra main pravartan bhoomika main aa gaya or vishvavidhyalay pradesh kaa raashtragaurav pathyakram lagu karne vaala pratham vishvavidhyalay ban gaya. bhavishya ke liye sankaay kii yojanayen apane gauravshaalee ateet ko banaaye rakhane tathaa samay evam abhinav paristhitiyon or chunautiyon ke anukool apane str ko banaaye rakhane ke liye kala sankaay kii nimna bhavi yojanayen hai -
 gharsh 1957 main gorakhpur vishvavidhyalay kii sthapana ke pashchaat varsh 1958 main vigyaan sankaay main pathan pathan kaa kaary armbh hua. prarambh main is sankaay ke antargat bhautik vigyaan, rasayanashastra, vanaspati vigyaan, praani vigyaan tathaa ganit vibhaag main snaatak tathaa snatakottar kakshaen evam raksha adhyayan vibhaag main keval snaatak str par kakshayen sanchalit hui. vigyaan sankaay ke antargat bhautik vigyaan, electranix vibhaag, computer vibhaag, ganit evam sankhyikee vibhaag majithia bhawan main, praani vigyaan, rasayanashastra evam vanaspati vigyaan vibhaag pant bhawan main sthit he. raksha adhyayan vibhaag, grih vigyaan vibhaag evam jaev praudvogiki vibhaag ke apane bhawan he.

 gorakhpur vishvavidhyalay main shikshan kaary armbh hone ke pratham varsh 1957 main hi vaanijya sankaay ke arntagat em0kaam0 str ke prarambh hone se vaanijya sankaay dwivibhageeya sankaay ho gaya. san 2001 ko akhil bhartiya takneekee shiksha parishad nayi dilli dwaara manyata praapt em0bi0e0 paathyakram ko or adhik sarthak evam gatisheel banane ke liye vyavsaayik prashasan naamak pruthak vibhaag kii sthapana hui. vaanijya vibhaag evam vyavsaayik prashasan vibhaag maharana pratap parisar main sthti hai jab ki arthshaastra vibhaag mukhya parisar main vidyaman he. 
 vaanijya sankaay ke antargat sanchalit vibhaag hai-
 dinadayal upaadhyaay gorakhpur vishvavidhyalay, gorakhpur main shiksha vibhaag kii sthapana san 1953 main 'vishvavidhyalay foundation sosayadi' ke sangrakshan main el0tii0 kalege ke rup main hui. farvari san 1957 main 'vishvavidhyalay foundation sosayadi' dwaara el0tii0 kalege kii sampoorn parisampattiyo evam daayitv ttatha staff vishvavidhyalay ko hastantarit kar diya gaya. is prakaar gorakhpur vishvavidhyalay main sarvapratham 'shiksha vibhaag' kii sthapana hui jo janvari san 1979 main uttar pradesh shashan ke nirnay se ekal vibhageeya 'shiksha sankaay main parivartit ho gaya. san 2003 main 'prauji satat evam prasar shiksha vibhaag' bhi isi sankaay kaa ek ang ban gaya. is prakaar vartmaan samay se is sankaay se do vibhaag - shikshashastra vibhaag tathaa prauji satat evam prasar shiksha vibhaag sambadv hai.vartmaan samay main kul 38 prashikshan mahavidhyalay dinadayal upaadhyaay gorakhpur vishvavidhyalay, gorakhpur ke shiksh sankaay ke paryvekshan evam nirikshan main karyrat he, jinamein 29 prashikshan mahavidhyalay shashan kii swavittaposhit yojana ke antargat karyrat hai.
 is vishvavidhyalay se lagbhag 800 mahavidhyalay sambaddh hai. inaki suchi yahan dekh sakate hai. isse sambaddh kuch pramukh vishvavidhyalay nimnlikhit hai-
