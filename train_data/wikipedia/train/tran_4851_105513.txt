 pithan ek saamaanya kaaryon ke liye upyukt, ucch stareey programing bhashaa, interactive, object oriented, scrypting bhashaa he. is bhashaa ko is tarah se dizin kiya gaya he taaki isamen likhe gaye code aasaani se padhe or samajhe ja saken. 
 anya programing bhashaaon ke vipareet, jinamein code-blox ko darshane ke liye majhole koshthak kaa istemaal kiya jaataa he, pithan main code-blox ko darshane ke liye hwite spes kaa prayog kiya jaataa he. is programing bhashaa ko Guido van Rossum ne 1991 main banaaya tha. yah vastutah ek programing lipi he jisamen program chalaane ke liye code ko compail, yani purv-sanyojit karne kii jarurat nahin he. paythan "vaakya rachana ke saath bahut spasht ullekhneey shakti" kaa daawa karti he. or uske maanak pustkaalay bade or vyapak he.
 is bhashaa kii dizin-darshan main kuta-pathneeyata par jor diya gaya he. pithan kaa daawa he ki iskaa cintax bahut spasht he; isaki maanak librari vishaal or sarvasamahit he. kai linaks systamon ke saath pithan praayah judaa hua aata he.
 anya gatisheel bhashaaon kii tarah, paythan aksar ek scrypting bhashaa ke rup main prayog kiya jaataa he, lekin kabhi kabhi gaiyr screepting sandarbhon kii ek vistrut shrunkhlaa main bhi prayog kiya jaataa he. kuch upkarnon kaa upyog karke, paythan code swasampurna nishpaadan yogya program ke rup main pack kiya ja saktaa he. paythan interpritre kai oparating system ke liye uplabdh hai.
 pithan 1980 ke dashak ke antim varshon dizin kii gayi thi. iske karyanvayan disambar 1989 main shuru hua.
 pithan 3.0 ek lambii avadhi ke baad 3 disambar 2008 ko jari kiya gaya tha. yah pithan 2.x se campatible nahin tha.
