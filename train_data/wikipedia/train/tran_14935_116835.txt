 shrutalekhan kaa arth he 'sune hue ko likhnaa' yaa 'sunakar likhnaa'. 'shrut' kaa arth hota he,'suna hua'. is vidhi main ek vyakti bolta he tathaa doosraa sun kar use likhata he. vidyalayon main shrutalekhan kaa upyog vartanee sudhaarney hetu kiya jaataa he. aajkal bahut si electronic yuktiyon main sunakar likhane kii kshamta he.
 takneekee sandarbh main shrutalekhan se aashay he ki computer ko bolkar likhavana. is vidhi main prayokta microphone main bolta he tathaa computer main maujuud spich tu taxt program use proses kar taxt main badalakar likhata he. is prakaar kaa kaary karne vaale software ko shrutalekhan software kehete hai. angreji hetu is prakaar ke software vindos evam offis ke kuch sanskarnon main anternirmit hai. iske atirikt dragon nachurli naamak prasiddh software bhi he.
 hindi hetu abhi tak aisa keval shrutalekhan-rajbhasha naamak software uplabdh he.
