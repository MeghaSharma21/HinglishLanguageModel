 main purvottar sanyukt rajya ke new england kshetra main uttari rajya he. isaki pashchim main new hampshire se, purv main atlantic mahaasaagar se or uttar main canadai praanton new bransavik or quebec se antarrashtriya seemaa lagati he. main sanyukt rajya main sabase purv main sthit rajya he. 
 main ke ilaake main hajaaron saalon tak adivasee log ekmaatr nivasee the. 1604 main is kshetra main sabase pehle europea basti francisi dwaara basaai gai thi. 1620 ke dashak main angrejon ne main ke tat ke kinare kai bastiyaan basaai. 1812 ke yuddh ke ant main isapar british senaaon dwaara kabja kar liya gaya tha. baad main shanti samajhote ke tahat sanyukt rajya america ko ise lautaa diya gaya. main 1820 tak masachusets kaa hissaa tha. phir usne masachusets se alag rajya banane ke liye matdaan kiya. 15 march, 1820 ko ise 23 wein rajya ke rup main sangh main bhartee kiya gaya.
 2015 ke anumaan anusaar main kii aabaadi 13,29,328 he. jis hisaab se uskaa america ke rajyon main 42waan sthaan hua. kshetra ke hisaab se yah 39waan sabase bada rajya he. angrezi bhashaa 92% janata dwaara boli jaati he par usko koi sarkari darja praapt nahin .
