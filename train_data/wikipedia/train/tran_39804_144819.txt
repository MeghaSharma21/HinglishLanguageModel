 ramashankar yadav 'vidrohi' hindi ke lokapriy janakavi hai. pragatisheel parampara ke is kavi kii rachanaaon kaa ekmaatr prakashit sangrah 'nai kheti' he. iskaa prakaashan inke jivan ke antim daur 2011 i main hua. ve snatakottar chaatr ke rup main jawaharlal nehru vishvavidhyalay se jude. yah judaav aaajeevan bana rahaa. inkaa nidhan 8 disambar 2015 ko 58 varsh kii avasthaa main ho gaya.
 unka janm 3 disambar, 1957 ko uttar pradesh ke sultanpur jile ke antargat ahiri ferozepur gaanv main hua. unki aarambhik shiksha gaanv main hi hui. sultanpur main unhone snaatak kiya. iske baad unhone kamala nehru institute main vakaalat main daakhila liya. ve ise puraa nahin kar sake. unhone 1980 main jawaharlal nehru vishvavidhyalay main snatakottar main pravesh liya. 1983 main chaatr-andolan ke baad unhen jeenyu se nikaal diya gaya. unhone 1980 main jawaharlal nehru vishvavidhyalay main snatakottar main pravesh liya. 1983 main chaatr-andolan ke baad unhen jeenyu se nikaal diya gaya. iske baavjood ve aaajeevan jeenyu main hi rahe.

 vidrohi mukhyatah pragatisheel chetana ke kavi hai. unki kavitaayen lambey samay tak aprakashit or unki smriti main surakshit rahi. ve apni kavita sunaane ke andaaj ke kaaran bahut lokapriya rahe. 2011 i main inaki rachanaaon kaa prakaashan 'nai kheti' sheershak sangrah se hua.kuch pramukh panktiyaan :
 1.mujhe masihaai main koi yakeen he hi nahin 
 main maanata hi nahin ki koi mujhase bada hoga.
 2.main kisaan hoon
 aasmaan main dhan bo rahaa hoon
 kuch log kah rahe hai
 ki pagale! aasmaan main dhan nahin jamaa karata
 main kahata hoon pagale!
 agar zamin par bhagwaan jam saktaa he
 to aasmaan main dhan bhi jam saktaa he
 or ab to donon main se koi ek hokar rahegaa
 yaa to zamin se bhagwaan ukhdega
 yaa aasmaan main dhan jamega.
 3.na to main sabal hum,

 main hi akbar hum,

 kavita or lathi main yahi antar he.
 vidrohi kii kavitaaon main jis tarah ke bimb dekhne ko milte hai anyatra hindi sahitya main durlabh he. jaise vidrohi kii nani mohenjodaro ke talab main nahaati hai, gaay ko everest ke khunte main baandhti hai. vidrohi hindi kaa akelaa kavi he jisane 'nijee sampatti' ke saare maanakon ko tod phodkar phenk diya or jindagi bhar majaluumon kii awaz ko kuwwat di.
