 alphabetum bramhanikum kassiano beligtti dwaara latin bhashaa main likhaa hua hindi vyakaran kaa granth he. iskaa prakaashan san 1771 isvi main hua. isaki visheshta yah he ki isamen nagari ke akshar evam shabd sundar tipon main mudrit hai. grearsan ke anusaar yah hindi varnmaala sambandhi shreshth rachana he. alphabetum unani bhashaa kaa shabd he jisase angrezi kaa alfabet shabd bana he. iskaa arth aksharmala he. is purastak ke adhyay ek main swar, adhyay do main mul vyanjan, adhyay teen main vyanjanon ke uccharan ke vishesh vivaran, adhyay chaar main vyanjanon ke saath swaron kaa sanyog, adhyay paanch main swar-sanyukt vyanjan, adhyay chah main sanyuktakshar or unke naam, adhyay saat main sanyuktakshar kii talika, adhyay aath main kis prakaar hindustani kuch aksharon kii kami puri karte hai, adhyay nau main nagari yaa janata kii varnmaala, adhyay das main hindustani varnmaala kii latin varnmaala ke kram or uccharan ke saath tulana, adhyay gyaarah main arabi ankon ke saath hindustani ankon or aksharon main sankhyaayein tathaa adhyay baarah main adhyetaon ke abhyaas ke liye kuch prarthanaayen bhi hindustani lipi main di gai hai. isamen latin prarthanaaon kaa hindustani lipi main keval lipyantaran miltaa he. ant main hindustani bhashaa main he pita aadi prarthanaayen anudit hai. is rachana kii bhoomika italy kii rajdhani rome main isai dharm ke prachararth sanstha propaganda fide ke adhyaksh yohan christobal amadusi ne likhi thi. is pustak kii bhoomika se yah spasht hota he ki san 1771 isvi main bhi nagari lipi main likhi janabhasha hindi yaa hindustani kaa rashtravyapee prachaar-prasar tha. jo videshi us samay bharat main vyapaar karne ke liye athwa ghoomane phirane ke liye aate the, ve bharat aane ke purv isko sikhte the tathaa bharat ke vibhinn kshetron main iske madhyam se apana kaary sampann karte the.
