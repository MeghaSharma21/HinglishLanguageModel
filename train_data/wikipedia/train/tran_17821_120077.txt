 ethernet, local aria network taiyaar karne kaa ek protocol hota he. yah 1970 ke aarambhik dashak se chali aa rahi vishwasaneey networkig uplabdh karaane vaali seva he. isaki abhiklpana 1973 main bob metcof ne kii thi. baad main digital, intel or xerox ke prayaason se yah local aria network kaa ek maanak praturup ban gaya. ethernet cabalon ke madhyam se vistaar kiya jaataa he. iske cabel kai roopon main uplabdh hote hai. isamen CAT3, CAT5, CAT5i or CAT6 sabase adhik prachlit hai. inaki dizin inke prayog par nirbhar hoti he or inaki keemat gunvattta ke anusaar badhati jaati he. ethernet cabile kaa prayog praayah ucch-gati vaale computer network ke liye kiya jaataa he. saath hi iskaa prayog broadband ke liye bhi hota he. computer ke saath lain/ethernet ko jodne ke liye computer main ethernet card kii aavashyaktaa padati he.
 ek local aria network main kuch aavashyak chijen hoti hai, jaise, do yaa do se adhik computer, jo network se jude hon, har computer main ek network interphes card, computer ko jodne ke liye ek ethernet cabile, network yatayaat ko nirdeshit karne ke liye ek netwarking hub or samarthak software. network interphes card ko har computer main lagaakar ise ek vishesh adress aavantit kiya jaataa he. har interphes card ek ethernet cabile ke madhyam se kendriya hub se judaa hota he. is hub main local aria network ke saare data ko praapt or punarnirdeshit kiya jaataa he. is tarah ethernet, aankadon ke sangrah, unki shering ke saath hi printers, fax mashin or scinar ke puure system kaa samooh taiyaar karata he.
 ethernet pranali kaa vistaar karne hetu taron kaa bada janjaal failaa hota he. in taron ko bade hi vyavasthit dhang se strot se gantavy tak pahunchaana hota he, jisase ki kisi samasya ke samay taron kii pahchaan ho sake saath hi sudhaar bhi sambhav ho. is samasya se nibatane hetu betaar ethernet bhi prachlan main aa gayi hai jinamein wave kaa prayog kiya jaataa he. isamen wireles network interphes card kaa prayog hota he, jisamen ek antina laga hota he. ye network apekshaakrit adhik majboot hota he lekin isamen atirikt suraksha kii aavashyaktaa hoti he, kyonki taron ko to kinhin nishchit computaron se joda ja saktaa he, kintu jab sara data betaar vaataavaran main uplabdh ho to koi bhi computer ise praapt kar saktaa he. atah iske liye paaski aadi kutashabdon kaa prayog kiya jaataa he. ethernet ke vikalp ke rup main ibiem ke taiyaar kiye gaye protocol or atm takanic kaa bhi prayog kiya ja saktaa he.
