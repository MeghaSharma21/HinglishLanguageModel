 sarvangashoth, yaa dehshoth sharir kii ek vishisht sarvangeey shothyukt avasthaa he, jiske antargat sampurna adhstvchiya utak main shoth ke kaaran taral padaarth kaa sanchay ho jaataa he. iske kaaran sharir kaa aakaar bahut bada ho jaataa he tathaa uski ek vishesh prakaar kii akruti ho jaati he.
 dehshoth kaa mukhya kaaran atyadhik shiraagat shirachaap he, jo mukhyat: sthanik shiraagat avrodh se hota he, jaise shiraagat bahya kaaran se dabaav, arbud, ombosis ityadi. kabhi-kabhi yah hridayakapaaton ke vikaron se utpann hota he. hriday ke kaary main shithilataa se bhi yah avasthaa utpann hoti he. hridayakapaat ke is prakaar ke vikaar main dhamnigat rudhirchaap ghat jaataa he or raktasanchaar main shithilataa aa jaati he. ucch shiraagat chaap se shirayen phool jaati hai tathaa unke valve ke kaary main shithilataa aa jaati he. shiraaon main sanchit rudhir gurutvkarshan se sthanik koshikaon par dabaav daalata he or isi ke falswaroop koshikaon se taral padaarth chaanakar adhstvchaa main sanchit ho jaataa he tathaa adhstvchiya utak main shoth utpann kar detaa he. upativr gurdashoth main shoth kaa kaaran sharir se atyadhik maatra main elvumin kaa parityag he. mutr dwaara niklaa albumin plazma se aata he. itnaa elvumin baahar jaataa he ki plazma main uski maatra keval 50% rah jaati he. rakt main do protein rahete hai : albumin or glovulin. iskaa anupaat 3:1 rahata he. albumin ke nikal jane par globulin kii maatra bad jaati he. isse coloid parisaran dabaav kam ho jane se, jal kaa sanchay bad jaataa he or albumin ke mutr dwaara nikal jane se jal kaa sanchay adhik hokar shoth bad jaataa he. sharir ke foole rahane par bhi albumin kii kami se rogi main bal kii kami ho jaati he.
 sharir ke utakon main jal bhar jane se, vishesh karke mahasrotiya bhaag main jal kii maatra badhane se tathaa vrikk main is jal ke nikaalane kii samarthy na rahane se, vaman or atisar prarambh hota he. gurdashoth main albumin kaa anupaat 1:3 ho jaataa he. shoth or raktaalpata ke kaaran rogi ke chehre tathaa sharir kaa aakaar bahut bada tathaa pandu ho jaataa he. isse sharir kii ek vishesh akruti ho jaati he. rakt main haemoglobin kii vishesh kami ho jaati he. mul kii maatra main kami hokar uskaa vishisht ghanatv bad jaataa he. rogi ko vishesh aalasy kaa anubhav hota he tathaa pachan kriya main vikaar utpann ho jaataa he, parantu iske kaaran raktchaap main koi vruddhi nahin hoti.
 jeern satat malaria main dehshoth kamjor, krush rogiyon main dikhaai detaa he. 1934 i. main lanka main malaria ke sankraman kaal main yah dikhaai diya tha. vickramasuriya ne 357 rogiyon main se 40 pratishat main dehshoth dekha tha, jo mukhya rup main prasuta striyon main adhik dikhaai diya tha. is prakaar ke malaria main updrav swaroop dehshoth ke hone kaa kaaran mukhyat: plazma protein kii nunata he. uchit upachaar se aisa shoth lupt bhi ho jaataa he.
 kabhi kabhi chlonorkiasis naamak krimijanya rog main bhi lakshan swaroop dehshoth dekha jaataa he. akunsh krimi janya lakshanon ke antargat raktaalpata, shaareerik krishta ke saath shoth tathaa saamaanya dehshoth kaa hona bahut hi swaabhaavik he. is rog main chehre or pair main, anya sthanon kii apeksha, adhik sujan dikhaai detii he.
