 saboodana ek khaadya padaarth he. yah chote-chote moti kii tarah safed or gol hote hai. yah sago palm naamak ped ke tane ke goode se banataa he. saago, tad kii tarah kaa ek paudha hota he. ye mularoop se purvi africa kaa paudha he. pakane ke baad yah apadarshi se halkaa paardarshee, narm or spanji ho jaataa he.
 bharat main sabudane kaa upyog adhiktar papad, kheer or khichdi banane main hota he. soup or anya chizon ko gadha karne ke liye bhi iskaa upyog hota he.
 bharat main sabudane kaa utpaadan sabase pehle tamilnadu ke selem main hua tha. lagbhag 1943-44 main bharat main iskaa utpaadan ek kutir udyog ke rup main hua tha. isamen pehle tepyaka kii jadon ko masal kar uske doodh ko chaanakar use jamane dete the. phir uski choti choti goliyaan banaakar senk lete the.
 tepyaka ke utpaadan main bharat agrim deshon main he. lagbhag 700 ikaiyaan selem main sthit hai. saboodana main carbohydrate kii pramukhataa hoti he or isamen kuch maatra main calcium va vitamin si bhi hota he.
 saboodana kii kai kismein bazaar main uplabdh hai unke banane kii gunvattta alag hone par unke naam badal or gun badal jate hai anyathaa ye ek hi prakaar kaa hota he, aararot bhi isi kaa ek utpaada he.
 sabudana tepioka-saago ek sansaadhit, pakaane ke liye taiyaar, khaadya utpaada he. sabudana ke nirmaan ke liye ek hi kaccha maal he "tapioka root" jise antarrashtriya str par "kasaava" ke rup main janaa jaataa he. shishuon or biimaar vyaktiyon ke liye yaa upwas ke dauraan, sabudana poshan kaa ek sweekaarya rup mana jaataa he. yah kai tarah se yaa khichdi, vada, bonda aadi yaa desrt ke rup main vyanjanon kii ek kism main prayog kiya jaataa he. sabudana, tapioka-root se taiyaar kiya gaya ek utpaadan he, jiskaa vanaspatik naam "Manihot Esculenta Crantz paryaay Utilissima " he.
 yah sabudana] se atyadhik miltaa-julata he. donon aam taur par chote sookhe, apaardarshee dane ke rup main hote hai. donon safed rang main hote hai. jab bhigoya or pakaaya jaataa he, tab donon naram or spanji, bahut bade paardarshee dane ban jate hai. donon kaa vyapak rup se duniya bhar main aam taur par pudding banane main upyog kiya jaataa he.
 takreeban aam taur par bharat main hindi main sabudana;, bengali main 'Tapioca globule' yaa 'sagu' ট্যাপিওকা গ্লোবিউল yaa সাগু; gujarati main 'sabudana' સાબુદાણા; or marathi main sabudana;', tamil main 'Javvarisi' சாகோவில்;, malayalam main 'Kappa Sagu' കപ്പ സാഗൊ; kannada main 'Sabbakki' ಸಾಬುದಾನ; telegu main 'Saggubeeyam' సగ్గు బియ్యం; urdu main 'sagudan-' ساگودانه; kaha jaataa he. kai jagah ise 'tapioka sabudana' yaa 'tapioka globulls' ke naam se bhi janaa jaataa he. tapioka] or 'tapioka-root ] ' ke alag alag arth hai. "tapioka" kasaava se nikaala jane vaala ek utpaada he. kasaava starch ko tapioka kaha jaataa he. yah "tupi" shabd jise portugali shabd tipi'óka se liya gaya, se niklaa he. jiskaa arth kasaava starch se banaaye gayi khaadya kii prakriya ko darshata he. bharat main, shabd "tapioka-root" kasaava kand ke liye hi upyog kiya jaataa he or shabd 'tapioka' kasaava se nikaali gai ek vishesh aakaar main bhuni hui yaa senki hui starch ke liye pratinidhitiv karata he.
 kasaava yaa manioc paudhe kaa mul armbh dakshin america main hua. amejan nivaasiyon ne chaaval / aalu / makka ke saath yaa iske alaava bhi kasaava kaa istemaal kiya. portugali khojakartaaon ne africa taton or aasapaas ke dwipon ke saath apane vyapaar ke madhyam se africa main kasaava kii shuruyaat kii. tapioka-root sabudana or starch ke liye buniyadi kaccha maal he. tapioka 19 veen sadi ke baad ke hisse ke dauraan bharat main aayaa tha. 1940 ke dashak main mukhya rup se kerala, andhrapradesh, or tamilnadu rajyon main isaki vruddhi hui, jab tapioka se utpaadit starch or sabudane ke tareeke bharat main armbh hue. sabase pehle haath se manual rup se or baad main swadeshi utpaadan ke tareekon se iskaa dikaas hua. yah carbohydrate or calcium or vitamin-si kii paryaapt raashi vaala ek bahut hi paushtik utpaada he. bharat main 1943-44 main sabase pehle sabudana utpaadan, atyant chote paimaane par, tapioka kii jadon se doodh nikaal kar, chaan kar or, dane bana kar ek kutir-udyog ke rup main shuru hua. bharat main, sabudana pehali baar tamilnadu rajya ke selem main taiyaar kiya gaya . bhartiya tapioka-root main aam taur 30% se 35% starch saamagree he. vartmaan main bharat tapioka-rute kii paidaavaar main agrani deshon main se ek he. kareeb 650-700 ikaiyaan tamilnadu rajya ke selem jile main tapioka prasanskaran main lagi hui he.
 tamilnadu ke tapioka sabudana or tapioka starch udyog, dwitiya vishwa yuddh ke dauraan singapore, malaysia, holland, japan or sanyukt rajya america se videshi sabudana or starch ke aayaat kii nishedhagna se upji kami kaa parinaam he. selem ke machhlee vyapaaree shri manikkkam chettiar apane vyapaar ke silsile main bahut baar selem se kerala jate rahete the. unki mulaakaat penang se aakar kerala main based shri popatlal ji shah se hui, jinhe tapioka starch nirmaan kaa gyaan tha. varsh 1943 main, selem se in donon ne bahut chote kutir udyog rup main tapioka starch or sabudana aadim tareekon se nirmaan prarambh kiya. sabudana or starch ke liye dainik badhati maang ko puraa karne ke liye, ek pratibhashali maikenic shri em venkatchalam gaundar kii madad se utpaadan kii mashinari or tareekon main sudhaar hua. parinamswaroop udyog kii utpaadan kshamta prati din 100 kilo ke 2 thilon se batch kar 25 thile ho gai.
 1944 main puure desh main ek gambhir akaal pada. chunki sabudana ek khaadya-padaarth mana gaya, at: selem colector ne selem se baahar bechney par roak laga di. selem sabudana or starch nirmaataaon ne ek sangh kaa gathan kar naagarik aapurti ayukta ke samaksh is maamle kaa pratinidhitiv kiya or jila colector ke nishedhaatmak aadesh ko radd karavaaya. 1945 se sabudana or tapioka starch ke utpaadan main prashansaneey vruddhi hui.
