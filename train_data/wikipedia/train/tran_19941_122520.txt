 supergirl disi comics dwaara prakashit america comic pustakon main dikhne vaali kai kaalpanik charitron kaa naam he. kaara jor-el ko mul tathaa sabase prasidd supergirl mana jaataa he. is charitra ko lekhak oto bindar dwaara banaaya gaya tha, or kalakaar al plastino ne iskaa dizin taiyaar kiya tha. kaara sarvapratham ection comics #252 main prakashit "da supergirl from kryptone" naamak ek kahani main dikhaai di thi. 1985 main kaara kii mrityu ke baad kai alag alag charitron ne supergirl naam apanaaya, jinamein matrix, linda danvars or sir-el ityadi pramukh hai.
 comic book main upasthiti ke apane shuruaati dinon ke baad se yah charitra anemeshan, film, televijan or vyapaar sambandhi anya madhyamon main pradarshit hoti rahi he. may 2011 main supergirl ko ijien ne apni top 100 comic book hiroj of ol time kii suchi main 94wein sthaan par rakhaa. iske baad november 2013 main bhi isien ne is charitra ko disi comics ke sheersh 25 nayakon kii suchi main 17wein sthaan par rakhaa.
 mul supergirl rishte main superman kii chacheri bahan he. pehle kuch varshon tak to use superman kii comic kathaon main ek sahaayak charitra ke rup main darshaya gaya, or phir 1969 main disi kii adventure comics shrunkhlaa main use pehali baar mukhya charitra ke rup main pradarshit kiya gaya. 1972 main supergirl ke naam kii pratham comic shrunkhlaa jari kii gayi, jo 1974 tak prakaashan main rahi. iske baad 1982 main da dering new adventures of supergirl naamak ek masik comic shrunkhlaa bhi jari kii gayi, lekin 1984 main uskaa prakaashan bhi band kar diya gaya. 1985 main chrysis on infinit arths kahani shrunkhlaa main kaara kii mrityu ho gayi.
 chrysis katha shrunkhlaa ke baad sabhi disi paatro ko riboot kiya gaya. is riboot main supergirl ke udgam kii katha bhi puri tarah badal di gayi. naa to vah ab superman kii bahan thi, or naa hi kryptone vaasi. is nae charitra kaa naam matrix rakhaa gaya. matrix ek kritrim jeev thi, jise lekes luther dwaara pocket continam se bane protoplasm se nirmit kiya tha. matrix sabase pehle superman ve2, #16 main dikhaai di. 1996 main lekhak peter david dwaara rachit supergirl naamak comic shrunkhlaa kaa prakaashan shuru hua. shrunkhlaa kii shuruyaat main hi linda danvars naamak ek mahila ko bachaane ke prayaas main matrix kaa uske sharir ke saath vilay ho gaya. manushya sharir praapt karne ke baad kuch varshon tak matrix ne saarvajanik jivan main linda danvars naam se rahanaa shuru kiya.
 supergirl #50 main matrix or linda ke sharir alag ho gaye. haalaanki itane dinon tak matrix ke saath rahane ke kaaran linda ne bhi shaktiyaan praapt kii, or phir usne bhi kuch samay tak supergirl naam apanaaya. 2005 kii shrunkhlaa infinit chrysis main hui ghatnaon main matrix ko itihaas se mitaa diya gaya. supergirl #75 main bhavishya main aayi kaara jor-el kii mulaakaat linda se hui, or phir kaara ke vyaktitv se prabhavit hokar linda uske samay main itihaas main chali gayi. isi samaykaal main usne superman se vivaah kiya, or un donon kii ek putree bhi hui. haalaanki agyaat kaarnon se kuch varshon baad linda superman ko bina kuch bataaye kahin gaayab ho gayi. 2008 main aayi ek laghu shrunkhlaa rain in hell main linda phir pradarshit hui, jisamen darshaya gaya ki linda ab nark main he. superman or linda kii putree, eriela kent ne bhi bhavishya main supergirl naam apanaaya.
 2003 kii comic shrunkhlaa superman: da 10 saint adventure #1 main sir-el naamak ek nayi supergirl prakat hui, jo ki, uske anusaar superman or loise len kii bhavishya se aayi beti thi. maanav rup main vah mia naam se saarvajanik jivan jiti thi. baad kii pustakon main khulaasa hua ki vah koi kryptonian nahin, balki ek manushya thi, jise apni shaktiyaan khalanaayak brinye se praapt hui thi. brinye ke saath hui ek ladaai main hi uski mrityu bhi ho gayi. superman #200 main timeline ne swayan ko punargathit kar diya, or isse sir-el kaa astitv sadaa ke liye mit gaya.
 sir-el kii mrityu ke baad disi comics continuity ke antargat "da supergirl from krypton" naamak superman/bitman comic book shrinkhalaa ke ank #8 main kaara jor-el kaa ek aadhunik sanskaran pun: prastut kiya gaya, jisane kai anya disi comics prakashnon main sahaayak bhoomika nibhaai. aadhunik kaara jor-el par aadhaarit supergirl naamak ek comic pustak shrinkhalaa vartmaan main prakaashan main he.
