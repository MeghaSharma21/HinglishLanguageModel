 kamala sankrutyaayan hindi bhashaa kii ek prasiddh upanyaaskaar thim.
 doctor kamala sankrutyaayan kaa janm 15 agast 1920 ko kalimpong main haa tha. 1949 main hindi-angreji shabdkosh ke nirmaan ke samay unki bhent pandit rahul sankrutyaayan hai. 1950 main donon kaa vivaah haa. isi varsh doctor kamala sankrutyaayan kii pehali laghu katha ‘nayaa samaaj patrika’ prakashit hai. saath hi dehradun se prakashit jaagrath gorkha naamak nepali rachana pehali baar prakaash main aayi. 
 1952 main unhen hindi sahitya sammelan prayag main sahitya ratna se sammanit kiya gaya. dow sankrutyaayan 1956 main agra vishvavidhyalay se hindi main eme va 1959 main piechdi kii. 1964 main unhen hindi samiti kii sampadak chunaa gaya. 1972 main kendriya shiksha mantralay dwaara gaiyr hindi lekhika ke rup main doctor kamala sankrutyaayan kii kriti assam kii lok kathaayen ke liye rashtriya puraskaar se sammanit kiya gaya tha. 1982 main pashchim bengal sarkaar kii ore se vichaar ani vivechna kriti ke liye bhanu bhakt puraskaar diya gaya. rahul sankrutyaayan rashtriya puraskaar se vibhooshit doctor kamala ko 2000 main dehradun main bhanu puraskaar 2008 main varanasi ke vidya dharm prachaarini nepali samiti dwaara madan smarak sammaan pradaan kiya gaya. unki antim kriti divyamani kaa prakaashan 2008 main haa tha.
 88 varshiya doctor kamala sankrutyaayan 25 oktober 2009 ko panchtatv main villin ho gayin. bhartiya-nepali sahitya kii pratishthit samalochak doctor kamala sankrutyaayan kaa nidhan ravivaar ko silligudi ke ek nursig home main ho gaya tha. vah darjeeling ke jiladhikari karyaalay ke upar farren vila main sankrutyaayan bhawan main rahati thim. doctor sankrutyaayan ko darjeeling ke hindu buddha shmashaan ghat par antim vidaai di gayi. antim sanskaar ke samay unke putr jetha sankrutyaayan, jayant sankrutyaayan va putree jaya sankrutyaayan ke saath hajaaron sahitya premi upasthit the.
