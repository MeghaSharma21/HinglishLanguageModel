 khagolashastra main khagoleey madhya rekha prithvi kii bumadhya rekha ke thik upar aasmaan main kaalpanik khagoleey gole par bana hua ek kaalpanik mahavritt he.
 prithvi ke uttari bhaag main rahane vaale agar khagoleey madhya rekha kii taraf dekhnaa chahen to aasmaan main dakshin kii disha main dekhenge. usee tarah prithvi ke dakshini golardh main rahane vaale khagoleey madhya rekha kii taraf dekhne ke liye akaash main uttar kii taraf dekhenge. prithvi ke bumadhya main rahane vaale khagoleey madhya rekha kii ore dekhne ke liye thik apane sir ke upar dekhenge. khagoleey madhya rekha se khagoleey vastuon ke sthanon ke bare main bataanaa aasaan ho jaataa he. udahaaran ke liye ham kah sakate hai ke khargosh taramandal khagoleey madhya rekha ke thik dakshin main he.
 "khagoleey madhya rekha" ko angrezi main "salastial equater" kehete hai. "khagoleey gole" ko angrezi main "salastial sfair", farsi main "kara-e-aasmaan" or bengali main "kha-golok" kehete hai.
