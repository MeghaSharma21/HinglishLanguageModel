 mukundarav aanandarav jayakar ) prakhyat vidhi visharad, samvidhanshastrajna, nyaayadhish, prasiddh vaktaa, shikshashasthri evam samajasevak the. 1917 ke baad hindustan kaa aisa koi bhi andolan nahin jisase aapkaa sambandh na rahaa ho. 1948 se puna ke kulapati ke rup main rahe. aapkaa vyaktitv atyant vyapak rahaa he. aapke saamaajik, raajnneetik, saanskrithik tathaa shiksha sambandhi kaaryon kaa moolyankan kiye bina bharat kaa aadhunik itihaas adhuraa rahegaa. is drushti se aapke bhaashanon, patron tathaa lekhon kaa adhyayan aavashyak he.
 mukundarav aanandarav jayakar kaa janm nasik main hua tha. aapaki shiksha bombai ke elfinston hai skool or kalege tathaa sarkari lau skool main hui thi. 1905 main aapane hoicourt main vakaalat shuru kii. 1937 main fedral court ov india main nyaayadhish ke rup main aapaki niyukti hui. privi counsil kii juditial kamity ke bhi aap sadsy the par 1942 main aapane is pad se tyaagptr de diya. consitatuent essembly ke liye sadsy ke rup main aapkaa nirvachan hua tha par 1947 main is pad se bhi ap ne tyaagptr de diya.
 1907 se 1912 tak lau skool main aap kanoon ke praadhyaapak the. aapke atmasamman kii bhaavana kaa isi samay saakshaatkaar hota he jab apane se nimna str ke europea adhyaaapak kii aapse ucchapad par niyukti par aapane tyaagptr de diya. fergyusan college main "plage ov english litarechar" par aapkaa bhaashan shiksha sambandhi aapke gambhir adhyayan k parichaayak he. mumbai vishvavidhyalay kii riform cameti ke aap 1924-25 main sadsy the. shiksha sudhaar kii yojana aapane isi samay prastut kii thi. sarkaar kii deccan college ko band karne kii niti ke viruddh aapane sangharsh kiya jo bombai vishvavidhyalay ke itihaas main chirsmaraneey he. 1941 main maharashtra uniwarsity ke sambandh main aapaki adhyakshata main ek kamity kaayam hui thi. shiksha or sahitya ke saath sangeet or kala main bhi aapaki ruchi thi. inke utthan ke liye bhi aap chintit the.
 shikshashasthri ke rup main aap sarvatra vikhyath the. nagpur, lakhanu, patna, aadi anek vishvavidhyalayon main hue aapke dikshant bhaashan amar hai. 1917, 1918, 1920 tathaa 1925 ke congress ke adhiveshnon main "swarajya" tathaa duusare raajnneetik vishyon par aapke bhaashan or prastaav bahut hi mahatvapurn rahe hai. bombai kii swaraj party legislative council main aap virodh paksh ke netaa rahe. 1926 main indian legislative essembly ke liye sadsy ke rup main aap nirvaachit kiye gaye. yahan par aap nesanalist party ke upanetaa ke rup main kaary karte rahe. gol mej sammelan main pratinidhi ke rup main aap upasthit the. fedral satrekchar kamity ke bhi aap sadsy rahe. gandhi ervin samjhauta ke liye sar sapru ke saath shantidoot ke rup main aapane kaary kiya. puna pact ke liye bhi aap prayatnsheel rahe.
 aap par sabhi kaa samaan rup se vishwaas hone ke kaaran madhyasth ke rup main aapaki yogyata mahneey thi. sarkaar ne aapako ke. si. es. ai. bananaa chaha par aap mistar jayakar hi bane rahe. 1919 main jaliyanvaala hatyakand se sambandhit aapaki report itihaas main amar he. 1940 main oxford vishvavidhyalay ne di. si. el. padavi se aapako vibhooshit kiya.
