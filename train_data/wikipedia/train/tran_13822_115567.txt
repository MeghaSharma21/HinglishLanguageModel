 sikhom mirabai chanu ek bhartiya bharottolan khilaadi hai. vah 2014 se niyamit rup se 48 kigra shrenee kii antararashtriya pratiyogitaaon main bhaag le raheen hai. chanu ne vishwa championships tathaa rashtramandal khelon main padak jite hai. unhen khel ke kshetra main yogadan ke liye bharat sarkaar se padam shri, rajeev gandhi khelratna puraskaar mil chukaa he.
 chanu ne 2014 rashtramandal khelen main 48 kigra shrenee main rajat padak jeeta tathaa gold coast main hue 2018 sanskaran main vishwa kirtiman ke saath swarna padak jeeta. unki sabase badi uplabdhi 2017 main anahim, sanyukt rajya america main yojit hui vishwa bharottolan championships main swarna padak jeetna tha.
 inhone glassgo main hue 2014 rashtramandal khelon main bharottolan spardhaa ke 48 kilogram varg main rajat padak praapt kiya. unhone kul 170 kilo vajan uthaaya, jisamen 75 sntch main or 95 clin and jerk main tha. inhone brazil ke rio di genero main yojit 2016 greeshmakaleen olympic ke liye kwalifai kiya, kintu clin and jerk main teenon prayaas asafal rahane ke baad vah padak jeetne main asafal raheen. 2017 main unhone mahila mahila 48 kigra shrenee main 194 kigra kaa bhaar uthaakar 2017 vishwa bharottolan championships, anahim, california, sanyukt rajya america main swarna padak jeeta. vah bharat main manipur rajya se hai.
 chanu ne 196 kigra, jisame 86 kg sntch main tathaa 110 kigra clin and jerk main tha, kaa vajan uthaakar bharat ko 2018 rashtramandal khelon kaa pehla swarna padak dilaayaa. iske saath hi unhone 48 kigra shrenee kaa rashtramandal khelon kaa record bhi tod diya.
