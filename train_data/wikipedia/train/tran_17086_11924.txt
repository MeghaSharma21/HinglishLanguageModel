 jab vibhinn prashasnic adhikaar-kshetron main sthit computer-sansadhano kaa prayog kisi ek hi kaary ko karne ke liye kiya jaataa he to use grid abhikalan kaha jaataa he. praaia: iskaa upyog bahut bade vaigyanik, takneekee yaa vanijyik kaaryon ke nishpaadan ke liye kiya jaataa he. is tarah ke kaary ke liye adhikaar kshetra main sthitt computer sansadhano ko samay va din ke anusaar kiraaye kaa bhugtan kiya jaataa he.
 grid computing ek aam lakshya tak pahunchane ke liye vyapak rup se vitarit computer sansadhano kaa upyog he. grid ko gaiyr-interactive varchlod vaale vitarit system ke rup main sochaa ja saktaa he jisamen badi sankhya main filein shaamil hoti hai. grid computing paaramparik ucch-pradarshan computing system se alag he jaise cluster computing main us grid computer main prayek node set ek alag kaary / anuprayog karne ke liye hota he. cluster computer kii tulana main grid computer bhi adhik visham or bhugolic rup se faile hue hote hai . haalanki ek grid kisi vishesh anuprayog ko samarpit kiya ja saktaa he, aamtaur par vibhinn uddeshyon ke liye ek grid kaa upyog kiya jaataa he. grid aksar saamaanya uddeshy grid middleware software pustakaalayon ke saath banaaya jaataa he. grid aakaar kaaphi bade ho sakate hai . 
 grid vitarit computing kaa ek rup he jiske dwaara " super virtual computer " kai network kiye gaye deele yugmit computaron se bana hota he jo bade kaaryon ko karne ke liye milakar kaam karte hai. kuch anuprayogon ke liye, vitarit yaa grid computing ko ek vishesh prakaar ke samaantar computing ke rup main dekha ja saktaa he jo purna computer computer network se judaa hua he ek paaramparik network interfes, jaise ethernet . yah ek superkamputer kii paaramparik dharna ke vipareet he, jisamen sthaneey hai speed computer bas se jude kai procesar hai.
