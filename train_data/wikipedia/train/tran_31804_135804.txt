 sanganak program kisi kaary vishesh ko sanganak dwaara karaane athwa karne ke liye sanganak ko samaj aane vaali bhashaa main diye gaye nirdesho kaa samooh hota he.
 kisi bhi program kii krmanudesan karne ke liye sarvapratham program ke samast nirdishtikaran ko bhalee-bhanti samaj liya jaataa he. program main prayog kii gai sabhi sharto kaa anupaalan sahi prakaar se ho rahaa he yaa nahi, yah bhi jaanch liya jaataa he. ab program ke sabhi nirdishtikaran ko jaanchane-samajhane ke uparaant program ke shuru se vaanchit parinaam praapt hone tak ke sabhi nirdesho ko vidhivat krambadhd kar liya jaataa he arthath programo kii digining kar lee jaati he. program kii dizin ko bhalee-bhaanti jaanchakar, program kii koding kii jaati he evam program ko compail kiya jaataa he. program ko test data input karke program kii jaanch kii jaati he ki vastav main sahi parinaam praapt ho rahaa he yaa nahi. yadi parinaam sahi nahin hote hai to iskaa arth he ki program ke kisi nirdesh kaa kram galat he athwa nirdesh kisi sthaan par galat diya gaya he. yadi parinaam sahi praapt hota he to program main diye gayi nirdesho ke kram ko ekbadhd kar liya jaataa he evam nirdesho ke is kram ko sanganak me sthapit kar diya jaataa he. is prakaar krmanudesan kii sampoorn prakriya sampann hoti he.
 kisi bhi ucch koti ke program main nimnaankit lakshan vaanchhneeya hote hai-
 sanganak ke kaary karne kii prakriya atyant jatil hoti he. sanganak kii apni smriti to hoti he lekin buddhi nahin hoti. sanganak maatr vahi kaary karata he, jiskaa ki use nirdesh diya jaataa he;arthath sanganak ko kaary kii buddhi krambaddh nirdesho athwa program dwaara di jaati he. sanganak me program kunjeepatal par tip karke feed kiya jaataa he, program main sanganak ko kya kya, kis prakraar karnaa he, yah spasht evam krambaddh rup main likhaa jaataa he.kalpana kijiye ki kisi vyakti ko do cap chaay banane kaa kaary diya gaya. ab hame sanganak ko chaay banane se sambandhit sabhi nirdesh nishchit kram main dene honge. yadi nirdesh maukhik rup se dene ho to ye nimnanusaar honge:-1.rasoi ghar main jaaen.2.ek cap len.3.pani kii toti kholen.4.cap main pani bharkar totin ko band kar den.5.cap ko sleb par rakh den.6.chaay kaa bhagona len.7.slab par rakhe cap kaa pani chaay ke bhagone main ulat de.8.chaay kaa bhagona gas burner par rakh de.9.gas liter jalaaye.10.ek haath se gas kii nob ko on karen tathaa saath hi liter bhi jalaat.11 yadi gas nahin jalati he to nob ko of kare or ab kramank 9 vaali kriya ko doharaaye.12.yadi gas jal jaati he to chaay ke bhagone ke pani ko tab tak garam hone de jab tak ki vah ubalane na lage.13. pani garam hone kii avadhi main chaay va chini kaa dibba jin par kramash: chaay or chini likhaa he; or ek chammach apane paas rakh len.14. pani ke ubalane par chaay kaa dibba kholkar aadha chammach chaay or uske baad chini kaa dibba kholkar ek chammach chini usame daal de.15.chaay va chini ke dibbo ko band karke jahaan se uthaaye the vahi rakh de.16.ab doodh ke bhagone main se ek cap doodh le len.17.ab pani ke punah ubalane par usame cap kaa doodh daale.18.is mishran ke ubalane par gas kii nob ko of kar de.19.ek trey le.20.is trey main do plate alag alag rakhiye.21.trey main rakhi gayi plato main ek ek cap rakhiye.22.ab chaay channi va sandasi le.23.sandasi se chaay ke bhagone ko pakadkar chanani ek cap ke upar rakhkar chaay chhaanie or tab tak chhaanie jab tak kii cap 80% na bhar jaye.24.kramank 23 vaali kriya ko duusare cap ke liye doharaaye.25.dono kapo main chaay bhar jane ke pashchaat chanani va chaay ke bhagone ko sinc main rakh dijie. aapane dekha ki chaay banane ke liye spasht evam ek nishcchit kram main nirdesh diye gaye. ye nirdesh kisi aanse vyakti se bhi chaay banavaane ke liye paryaapt he jisane kabhi chaay na banaai ho.isi prakaar sanganak se koi kaary karaane ke liye use spasht evam nishchit kram main nirdesh diye jate hai;or yahi prakriya sanganak krmanudesan he.
 sanganak keval masini bhashaa samajhataa he. vibhinn krmanudesan bhashaa main likhe gaye program main nirdesho ko Assembler, compiler athwa Interpreter kii sahaayta se masini bhashaa main parivartit karke sanganak ke micoprocesar main bheja jaataa he. tabhi sanganak in nirdesho kaa paalan kar upyukt parinaam prastut karata he. masini bhashaa maatr byanary anko arthath 0 se 1 ke samooho se bani hoti he jise sanganak kaa micoprocesar sidhe samaj saktaa he.jab ham sanganak ko koi bhi nirdesh kisi input ke madhyam se dete hai to sanganak swatah in nirdesho ko ASCII Code main parivartit kar saktaa he. nirdesh dene ke liye hame saamanyat: aksharo, sankhyao evam sanketo ke "kies" ko kii-board par dabaanaa hota he or sanganak swatah hi inhe apni bhashaa main badal letaa he. example-TYPEmaSInI bhashaa main -T Y PE
 sanganak ko nirdesh program tareeke se, atyant spasht bhashaa main evam vistaar se dena atyant aavashyak hota he. sanganak ko kaary vishesh karne ke liye ek program banaakar dena hota he. diya gaya program jitna spasht, vistrut or sateek hoga, sanganak utane hi sucharu rup se kaary karegaa, utani hi kam galtiyaa karegaa or utane hi sahi uttar degaa. yadi program aspasht hoga or usame samuchit vivaran evam spasht nirdesh nahin honge to yah sambhav he ki sanganak bina parinaam nikaale hi gananaa karata rahe athwa usase praapt parinaam aspasht or nirarthak ho. atah program atyant saavdhaanee or ekagrachit hokar taiyaar karnaa chaiye. sanganak kii sampoorn kaaryakshmata yojanevalverthaat program banane vaale vyakti kii kshamta par nirbhar hoti he.
 sanganak me apni koi budhdi nahin hoti. yah ek mastishk rahit evam atyant aagyaakaari mashin he. yadi use koi nirdesh nahin diya jaataa athwa aspasht nirdesh diya jaataa he tab bhi vah koi aapatti nahin karata or die gaye nirdeshaanusaar hi kaary karata he. atah program banaate samay atyant saavdhaanee baratani padati he.
 sanganak par program banaate samay nimna batoo ko dhyaan main rakhana aavadhyak he:-
 program main diye jane vaale nirdesho ko ek pravaah talika ke rup main prastut karnaa uchit hota he. isame yah spasht hona chaiye ki sanganak ko kab or kya karnaa he evam use vibhinn kriyaaye kis rup main kerani he. program ko upar se neeche kii ore pravaah chitra ke rup main darshaya jaataa he evam jahaan tark aadi karnaa hota he vahaan yah do bhaago main vibhakt kar diya jaataa he.
 program main nirdesho ko sanganak kii samaj main aane vaali bhashaa main likhnaa aavashyak hota he; taaki sanganak pradutt nirdesho ko samaj sake or unke anusaar kaary karke vaanchit parinaam prastut kar sake. ek baar program ko sanganak bhashaa main likhane ke baad ise sanganak kii smriti main arthath flopy, chumbakeey feete, chhidrit card aadi nivesh yuktiyo par ankit kar diya jaataa he. saath hi yah samasya ko hal karne ke liye aavashyak data bhi sanganak kii input unit ko pradaan kiya jaataa he. ab sanganak us program ke anusaar kaary karke input data kaa vishleshan pradarshit karke uchit parinaam prastut karata he. yadi program main praapt parinamo ko monitor screen par pradarshit karne athwa flopy, harddisk yaa chumbakeey feete par ankit karne ke nirdesh diye gaye hai tab sanganak praapt parinamo ko vahiin ankit kar detaa he.
