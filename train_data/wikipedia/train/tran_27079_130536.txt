 'bhartiya tarkshastra' se simit arth main 'nyaay darshan' kaa bodh hota he. kintu adhik vyapak arth main isamen buddha nyaay or jain nyaay bhi samavisht kiye jate hai. sabase vyapak rup main bhartiya tarkshastra se abhipraay sabhi bhartiya vidwaanon dwaara pratipaadit sabhi taarkik siddhanton ke sammucchy se he. 
 bhartiya tarkshastra, vishwa ke teen mul tarkshastron main se ek he; anya do hai, unani or chini tarkshastra. bhartiya tarkshastra kii yah parampara navyanyay ke rup main aadhunik kaal ke aarmabhik dinon tak jari rahi.
 bharat main tarkshastra kii parampara bahut puraani he. yah kam se kam 6thi shatabdi isha purv main medhatithi gautam ke anvikshiki se armbh hoti he. isi parampara main panini ke sanskrit vyakaran ke niyam ; ishapurv duusari shatee kaa vaisheshik sampradaay; duusari shatabdi ishapurv main nyaay darshan ke praneta gautam dwaara pramaanon kaa vishad vishleshan; tathaa duusari shatabdi ishapurv nagarjun kii chatushkoti ullekhneey hai. 
 bhartiya tarkshastra kaa itihaas 23 shatabdiyon main pasra hua he. bhartiya nyaayikon kii prakashit evam aprakashit kritiyon kii sankhya bhi vishaal he. pashchimi bhashaaon main yaa achche haalat main praapt rachanaaon kii sankhya kul rachanaaon kii sankhya kaa ek chota bhaag hi he. 
 bhartiya tarkshastra ke itihaas ke liye jise paanch bhaagon main baant sakate hai -
 1) vyakaran - panini aadi ke dwaara viksit ; is vyakaran ke atyant parishkrut tarkpurna niyamon ne baad ke adhikansh vidvatapurna kaaryon par apni chhap chodi.
 2) mimasaa
 3) vaisheshik evam purana nyaay
 4) buddha nyaay
 5) navyanyay
