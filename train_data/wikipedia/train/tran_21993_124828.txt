 kivaanch 10-12 feet lambi ekvarshiya shaakeey lata he. pattiyaan triparnak va parnak andakar tathaa romil chote he. pushp begni rang ke tathaa fali 5-10 se.me. lambi tathaa 1.2-1.8 se.me. chaudi ore 4-6 beej yukt hoti he. beej andakar tathaa safed yaa kaale rang ke hote he. kivaanch, bharat ke samast maidani kshetron main junglee lata ke rup main paaya jaataa he. yah himalay ke nichle kshetron main jangalon main paayi jaati he. 
 vaigyanik naam : mukuna prurins
 kul : pepileonasy
 anya naam : ise kau haij, kaunch, kapikachhu aadi anya naamon se bhi janaa jaataa he.
 fasal main alkelide paye jate he. jiske kaaran adhik maatra main yah vish he kintu kam maatra main dawa he. iskaa mukhya prayog dama evam saans ke rogon kii dawa nirmaan main hota he. iske atirikt iskaa prayog behoshee, shishu janm tathaa aankh kii dawaayein banane main hota he. kinwach ek upyogi fasal he.
 isamen el-dopa rasayan paaya jaataa he. iske beej, pattee, jad, rome sabhi kaa prayog aushadhi main hota he. beej main 'lecivin' glucoseida tathaa 0.5 pratishat alkelide bhi paye jate hai. beej main gahare bhure rang kaa tel paaya jaataa he. iskaa prayog kamottejak evam mutr rog tathaa durbalata ke liye kiya jaataa he. faliyaan sabji ke rup main tathaa jad evam rome-pet ke krimi tathaa pitt rog dur karte hai. yah uttam laabhadaayak fasal he.
