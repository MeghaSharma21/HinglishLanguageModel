 iskaa puraa naam Metropolitan Area Network hai yah ek aisa ucch gati vaala network he jo aawaaj, data or image ko 200 megabite prati second yaa isse adhik gati se data ko 75 ki.me. kii duuri tak le ja saktaa he. yah len se bada tathaa wain se chota network hota he. is network ke dwaara ek shahar ko duusare shahar se joda jaataa he.
 iske antargat do yaa do se adhik local aria network ek saath jude hote hai. yah ek shahar ke simaao ke bheetar kaa sthit computer network hota hai. router, switch or habs milakar ek metropolitan aria network kaa nirmaan karata hai. metropolitan aria network vrihad stareey network he jo kai corporate local aria network ko ek saath jodate hai. metropolitan aria network kisi ek sanghathan ke dwaara kharida nahin hota he balki inke sanchaar sanyantr tathaa upkaran kisi ek samooh yaa ek network pradaata dwaara mentin kiya jaataa he jo apane corporate graahakon ko isaki simaaen bechate hai. metropolitan aria network praayah ucch gati network ke rup main kaary karata he tathaa kshetreey sansadhano ke saajheedaari sahaayta karte hai.
