 dow mohan avasthi ko pipargaanv farrukhabad main hua. dow avasthi kii kavy kritiyaan ‘agnigandh’ or ‘abhishpt maharathi’ ullekhneey hai. abhishpt maharathi ucchkoti kaa kavy granth he, jisamen unhone bhartiya samskruti kaa chitra prastut kiya he or yah aashaa prakat kii he ki yah granth ‘desh kii aadhunik samasyaon kaa samaadhaan khoj nikaalane main sahaayak hoga.’ samprati elahabaad vishwa vidhyalay hindi vibhaag main professor ke pad se sevanivrutt hokar sahitya srijan main rat hai. praarambhik shiksha gaanv main hi hui. aapane 1942 main varnacular final pariksha uteern kii. avasthi ji batalaate hai ki ‘meri praarambhik shiksha or vyaktitv nirmaan main pan0 triyugi dayaal dikshit ji va abodh mishra jaise kaviyon kaa atyadhik yogadan he, jinka sanidhya praapt hone se mere kavi-rup ko praudhata praapt hui.’ hai skool karne ke baad aapane ek amareeki company main ‘assistant manager’ ke rup main tathaa distric board ke account vibhaag main ‘assistant accountint’ ke rup main kaary kiya. tadnantar intermediat vyaktigat parikshaarthi ke rup main bi0en0es0di0 kalege kanpur se pratham shrenee main uteern kiya. 1951 main agra vishwa vidhyalay se sambaddh es0di0 kalege kanpur se snaatak pariksha dwitiya shrenee main uteern kii. 1951 se june 1953 tak kanhaiyalaal ramsharan rastogi intermediat kalege, farrukhabad main adhyaapan kaary kiya. rastogi inter kalege main avasthi ji ko shri suresh chandra shukla se bade bhai kaa sneh mila. shukla ji ke atirikt anya sahyogiyon main shri atmanand shrivastav evam shri manohar singh chohaan pramukh the.vikasman vyaktitv main sthirataa kahan? atah avasthi ji ne adhyaapan kaary chhodkar 1954 main elahabaad vishwa vidhyalay se pehle angreji se em0e0 pratham varsh kiya’ kintu hindi prem ke kaaran angreji chhodkar 1956 main hindi se pratham shrenee or vishwa vidhyalay main pratham sthaan praapt kar apane gantavya kii pratham manzil praapt kii. 1958 main di0phil0 kii oopaadhi praapt kar elahabaad vishwa vidhyalay main hi praadhyaapak niyukt hue. adhyaapan kaary karte hue 1974 main di0lit0 kii oopaadhi hasil kii. pravaktaa, reader evam professor ke padon par kaary karte hue 1989 main avkaash praapt kiya.dow avasthi ke vyaktitv kaa yadi vishleshan kiya jaay, to kaha ja saktaa he ki unke paas na to chandronnat lalaat he na keshari vakshasthal or na hi ve ajaanubahu hai. unhone soundarya kii pratispadrdha main kaamdev ko vijit bhi nahin kiya he. ve ek saadhaaran or saamaanya se dikhne vaale insaan hai. unka sharir lambaa, charhara or ksheen he. unki aankhon main taakane or nihaarne kii vritti, haath main karm kii kalam, mastishk main lokhit kaa chintan, man main ishwar kaa dhyaan evam asaadhaaran santoshvritti tathaa hriday main aseem sahrudayata, bhavukta, samvednaa or paradukh kaatarata kaa bhav he. ve ek teevr pragna, nir-ksheer-viveki, gambheer prakruti or antermukhi vyaktitv ke dhrak hai.
 1. maharathi- is granth main karn katha varnit he. may 1946 main iske 18 chand ‘sukavi’ masik patrika main ‘karn-vadh’ naam se prakashit hue the. baad main is katha ko vistrut kar ‘maharathi’ kaa rup diya gaya. 1947 main yah granth purna hua or 1948 main pan0 shrinarayan chaturvedi ji ko samarpit kar diya gaya. prakaashan 1953 main indian press elahabaad se hua.
 2. bal kavita- isamen paanch rachanaayen hai- gandh ji ke bal mukunda, sanyam raay, muni or mooshak, parishram kaa fal or eklavya. iskaa prakaashan indian press elahabaad se san 1954 main hua.
 3. dekhbhaal kar chalo- is kavy pustika main parv main ritu sambandh geet hai. iske adhkansh geet elahabaad se nikalne vaale masik ‘bal sakha’ main samay-samay par prakashit hote rahe hai. pustakaakaar main iskaa prakaashan saroj prakaashan elahabaad se 1959 main hua. rachanaayen bal-vyaktitv nirmaanopayogi hai-
 is mahaan raashtra ko savaanrate huelok kii kureetiyaan sudhaarte huenvin surya vishwa main nikaalkar chalochalo, parantu yah dekhbhaal kar chalo !
 4. abhishpt maharathi- is khandakavy main ‘maharathi’ ke kuch chand samahit hai, lekin yah ‘maharathi’ se bhinn prakaar kii rachana he. ‘maharathi’ itivritmak he; kintu isamen desh kii vartmaan samasyaon kii chintaa he. khaskar dalit chetana ko abhivyakti dene vaala yah pratham khand kavy he, jo sohanlal dvivedi dwaara prashansit evam da0 ramamurti treepathi dwaara utkrisht ghoshit he. rashtrakavi sohanlal dvivedi kaa vichaar he- “charitra nirmaan karne vaale khand-kaavyon kaa aaj nitant abhaav he. aisi sthiti main uganuroop prerana pradaan karne vaala yah charitra kavy isaki sucharu purti karegaa.” yah kavy jivaji vishvavidhyalay gwalior, kanpur vishvavidhyalay kanpur va da0 raamamanohar lohia vishvavidhyalay faijabad ke paathyakram main sammilit rahaa he. yah saraswati press elahabaad se 1975 main prakashit hua.
 5. agni gandha-	yah granth vibhinn patr-patrikaon- saraswati, vishaal, veena, sahityakar, madhyapradesh sandesh evam nai dhra adimein prakashit geeton tathaa anugiton kaa sangrah he. 72 prishthon kii is pustika kaa prakaashan sahitya sangam, elahabaad se 1982 main hua.
 6. chaabidaar khilone- is pustak ke adhikansh chand ‘bal sakha’ main samay-samay par prakashit hote rahe hai. kavi ne is bal kavy ganrath main sahaj, saral or prabhvishnu bhashaa kaa prayog kiya he. prakaashan sahitya sangam, elahabaad se 1984 main hua.
 7. ek hamaara desh- isamen ek hamaara desh, vir aaryon kii ham santan, tum vir bano balavaan bano, nav varsh, holi ne sara rang badla aadi 13 geet hai. prakaashan sahitya sangam elahabaad se 1985 main hua.
 8. hua ujaalaa- aravind prakaashan agra se 1993 main prakashit is granth main baalagit sangrahit hai.
 9. halchal ke pankh-	is sangrah main 1963 se 1984 tak ke rachit kul 118 anugit hai. anugit vidh ko hindi sahitya main prasthaapit karne vaala yah pratham kavy granth he. rachanaakaar ne bhoomika main anugit kii avadhrana ko spasht karte hue jivan kii vividh anubhutiyon ko anugiton ke madhyam se dhvanit kiya he. is kriti ko padhakar prasiddh aalochak pro ramvilas sharma ne apni mangal kaamana preshit kii- “kavi rup main aapaki khyaati ore unnati ke liye meri mangal kaamana he.” prakaashan elahabaad se 1995 main hua.
 1. hindi sahitya kaa adhyatan itihaas-yah itihaas granth da0 avasthi kii moulik pratibha kaa parichay detaa he. da0 avasthi ne acharya shukla ke namakaran or kaal vibhaajan kii tark sammat samiksha karte hue hindi sahitya ke itihaas kaa jo vibhaajan evam namakaran kiya he, vah ati vaigyanik evam auchitypurna he. yah granth ucch shiksha ke shikshaarthiyon ke liye bahut upyogi he. prakaashan- saraswati press elahabaad se 1970 main hua.
 2. aadhunik hindi kavy-shilp-	is shodh-prabandh main 1900-1940 tak kii kavita kaa shilpagat vivechan he. mukhya visheshta yah he ki isamen prakashit pustakon se adhk un rachanaaon ko aadhra banaaya gaya jo vibhinn patr-patrikaon main chhapi hai. prakaashan- hindi parishad prakaashan elahabaad se 1962 main hua.
 3. niraala or ‘tulsidas’ kavy- is granth main sarvapratham niraala ke tulsidas kavy kii vidvatapurna vyakhya kii gai he. jisaki prashansa da0 ramvilas sharma ne bhi kii he. yah granth saroj prakaashan elahabaad se 1964 main prakashit hua.
 4. hindi reeti kavita or samakaleen urdu kavy-1943 se 1850 tak ke hindi-urdu kavy kaa shailpic evam saanskrithik-tulanaatmak or talasparshi vivechan he. is ganrath kii pan0 vishwanath prasad mishra, da0 rasaal evam da0 nagendra ne bhuri-bhuri prashansa kii he. da0 ramshankar shukla rasaal kaa vichaar he- “da0 avasthi kaa yah granth nitant moulik evam anoothaa he. aisa kaam abhi tak hindi yaa urdu bhashaa main nahin hua.” or acharya vishwanath prasad mishra ji ne to yahan tak kaha- “da0 mohan avasthi ke is krititv main jaisaa shram he or arthon kii paraten jis sookshmata se udghatit hui hai waisi mahneey shodh saadhna vaala reeti dhra par prabandh pehle koi nahin dikhaa.” prakaashan- saraswati press elahabaad se 1978 main hua.
 5. samikshan or pareekshan-	is granth main keshav ke prati samalochakon kaa ekangi drushtikon, bekan or pan0 ramachandra shukla, maithili sharan gupt kaa kavy shilp, kurukshetra, urvashi aadi 9 sameekshak niabandh sangrahit hai. ye sabhi nibandh lekhak ke moulik chintan ke dyotak hai. prakaashan sahitya sangam elahabaad se 1985 main hua.
 6. desh ke gaurav -in donon khandon main desh ke mahapurushon- nanak, gandhi aadi kaa jivan charitra varnit he. prakaashan saraswati press elahabaad se 1980 main hua.
 1. hindi niabandh kii vibhinn sheliyaan-raja shiv prasad sitare hind se lekar pan0 vidyanivas mishra tak ke sabhi nibandhkaron ke vibhinn shaileyon ke nibandh sangrahit hai. prakaashan saraswati press elahabaad se 1969 main hua.
 2. shreshth hindi ekanki-is sangrah main dow ramkumar verma aadi ke shreshth ekankiyon kaa sangrah he. prakaashan saraswati press elahabaad se 1970 main hua.
 3. urdu kavy sangrah-is sangrah main bali, sauda, mir, mir hasan, nazir akbarwaadi, jock, gaalib, anis or haali aadi 12 urdu kaviyon kii gazalon va anya kaviyon kii kavitaaon kaa sangrah he. prakaashan narayan publicing house elahabaad se 1969 main hua.
 4. sur sangrah- isamen surdaas ke 151 pad sangrahit hai. is sampaadan main kathakram ko dhyaan main rakhaa gaya he. is prakaar isamen sursagar kii puri katha krambaddh evam sankshipt rup main milti he. prakaashan hindi parishad prakaashan elahabaad se 1972 main hua.
 5. sursagar saar sangrah- is sampadan main dow dhirendra verma dwaara sampadit ‘sursagar saar sangrah’ kii vidvatapurna vyakhya he. prakaashan gyaan bharti elahabaad 1972. 
 6. niabandh sangrah - is sangrah kii bhoomika main niabandh ke udbhav or vikaas kaa tathypurna vivechan hua he or isamen bhi vividh prakaar ke niabandh sangrahit hai. prakaashan- hindi parishad prakaashan elahabaad se 1980. uparyukt vivechan se spasht he ki dow avasthi utkrisht koti ke kavi, sameekshak or sampadak hai. dow avasthi kaa kaary-kshetra bada vyapak he. ve akashwani elahabaad kii kshetreey paraamarsh datri samiti ke sadsy, akashwani kii chayan samitiyon ke visheshgy sadsy, kendriya vidyalayon kii chayan samitiyon ke visheshgy sadsy, anek vishwa vidyalayon kii chayan samitiyon ke rajyapal dwaara namit visheshgy sadsy tathaa jivaji vishwa vidhyalay gwalior, bundelkhand vishwa vidhyalay jhansi, kumaun vishwa vidhyalay nainital, avadh vishwa vidhyalay faijabad, vishwa bharti vishwa vidhyalay shaanti niketan kii hindi paathyakram nirdhaaran samitiyon evam shodh-samitiyon ke sadsy rahe hai.
