 unics main ek kami thi - isko samajhnaa tathaa chalaanaa mushkil he. andrew tenenbam, aimusterdam main computer vigyaan ke professor hai. unhone isaki sahaayta ke liye minix naam kaa program likhaa. isamen bhi kuch kamiyaan thim. linus torwald finland ke helsinki vishvavidhyalay main kamputer vigyaan ke chaatr the. unhone minix kii kami ko dur karne ke liye ek program likhaa jo ki baad main ‘linus kaa unics’ yaa chote main linaks kahalaaya. iskaa sabase pehla kor yaa karanal unhone 1991 main internet main post kiya. tab tak richard stallmen kaa 'gnu' project shuru ho chukaa tha. linus torwald ne isse bahut saare program apane linaks main liye. isliye richard stallmen kaa kahana he. ki ise 'gnew-linaks' kahana chaiye. par yah naam, shaayad lambaa rahane ke kaaran chal nahin paaya. par iskaa arth yah nahin hai ki linaks kii safalataa main gnew project kaa haath nahin he. gnew project ke bina linaks sambhav nahin tha.
 unics oparating system kii kalpana 1969 main sanyukt rajya america main anti and tii ke bel prayogshaalaon main ken thompsan, denis richi, duglas mexloyay or jo ossan dwaara kii gai thi or lagu kii gai thi. unics pehali baar 1971 main relies hua, or puri tarah se assembly bhashaa main likhaa gaya tha, jaisaa ki us samay aam abhyaas tha. baad main, 1973 main ek pramukh agrani drushtikon main, ise denis richi dwaara kuch programing bhashaa main likhaa gaya tha . unics ke ucch stareey bhashaa karyanvayan kii uplabdhta ne vibhinn computer platformon ko apane porting ko aasaan bana diya.
