 google krom ek web brouzar he jise google dwaara mukt strot code dwaara nirmit kiya gaya he. iskaa naam graphical yuzar interfes ke frem yani krom par rakhaa gaya he. is prakalp kaa naam chromium he tathaa ise biesadi license ke tahat jari kiya gaya he. 2 sitambar, 2008 ko google krom kaa 43 bhashaaon main microsoft windos prachaalan tantra hetu beta sanskaran jari kiya gaya. yah nayaa brouzar mukt strot lines code par aadhaarit hoga, jisamen tritiy party vikasakarta ko bhi uske anukool anuprayog banane kii suvidha mil sakegi.
 google krom ko behatar suraksha, behatar gati evam sthayitv ko dhyaan main rakhkar banaaya gaya tha. krom kaa sabase pramukh lakshan isaki gati or anuprayog nishpaadan hai. iske beta sanskaran ko march 2009 main launch kiya gaya tha. is sanskaran main jo nai suvidhaayen jodi gai thim unamem prapatr swatahpurna, sampurna prishth zum, auto scrol or nae prakaar kaa draig teb pramukh he. is broujar kii website ke anusaar, dekhne main ye paramparaagat google mukhaprishth  kii tarah he or tej tathaa spasht he. google krom kaa prayog karne par anya brouzaron kii bhaanti sidhe khaali prishth nahin khulata balki broujar upayokta dwaara sabase zyada prayog kiye gayi antim kuch webprushthon kaa thambanel drishy dikhaata he, jise clic karne par vaanchit prishth khul jaataa he. is kaaran se upayokta apane manavaanchit prishthon par sheeghra hi nevigate kar paata he. isamen uplabdh omanybox kaa laabh ye he ki bina google khole hi, google main search kar sakate hai. udhaaharan ke liye adress baar main maatr olympic daalate hi usase sambandhit website ke pate bataa detaa he, saath hi adhure or galat paton ko ricover karne kii suvidha bhi isamen he.
 is broujar main upasthit task manager iken se is bare main jaankaree mil sakati he, ki kis prakriya main kitni smriti kaa prayog ho rahaa he. iske saath hi yadi koi website nahin chal rahi to usase duusari site par fark nahin padataa he. kresh recovery ke dwaara computer system ke achaanak band ho jane par or phir kholane par yah upayokta se poochta bhi he, ki vah usee prishth par pun: aana chaahate hai yaa phir nayaa prishth kholana chaahate hai. incognito ke kaaran upayokta ypi adress li nahin hota jisase suraksha bad jaati he. kuch site aisi hai, jahaan pehali baar kisi cheez ko lod karte hue samay kam lagegaa, phir jitani baar aaenge, samay badhata jayega. prayek site ko usko surf karne vaale ke bare main jaankaree uske ypi adress se milti he.
 krom main opera web brouzar kii bhaanti hi teb pranali kaa upyog kiya gaya he. is teb pranali main zyada prayog kii gayi websiton kaa yah apane aap itihaas banaakar naye teb main jodta chalaa jaataa he. jaise hi naye teb par clic karte hai yah apane aap saheje gayi prishthon ko baks main pradarshit karata he. isse purv pasandida saiton ko naye teb main sahejkar rakhane kii yah suvidha keval opera ke broujar main milti thi. google dwaara abhi tak samarthit firefaks sabase badi kami yah thi ki diphalt search ingine google hi hota tha jisamen sidhe hompej se jimail aadi kii suvidhaon kii kami rahati thi. baad main ai.i-7 main ekasaath kai saare hompej banaakar rakhane kii suvidha mili thi. kintu isaki kami isaki manthar gati he. bharat main 128 capibies speed ko broadband speed kaha jaataa he, jabaki pashchim ke deshon main 1 embipies kii speed broadband kii shrenee main aati he. ausat internet upabhokta isi speed par kaam karte he. chote shaharon or gramin ilaakon main to yah speed 76 capibies maatr hi hoti he. aise main ai i-7 atyadhik dhimaa ho jaataa he. yahaan opera, safari or firefaks is lihaaj se kuch behatar hai lekin itani kam speed par koi bhi broujar thik se kaam nahin kar saktaa. isi kaaran se ai.i-6 hi adhik prayog hota aayaa he. 
 krom ke prayog karte hue broujar ke upar koi pattee nahin dikhaai detii he jis par fil, edit or vikalp ke batan hote the. ise hataane kaa sahi kaaran to gyaat nahin he, kintu isse vindo kaa aakaar kaaphi bad jaataa he. 14-15 inch kaa monitor prayog karte hue bhi behatar vijabilti milti he. haa sidhe clic kar kuch vikalp chune ja sakate the, jinke liye isamen kuch shortkat kunjiyon kaa sahaara lenaa padataa he. krom main ek kami he ki isamen mouse ke daayen clic par rifresh kaa vikalp nahin miltaa he. is kami ke sang hi ek achchhai bhi he, vah he gupt pej. yadi bina record kii sarfing kerani ho to google gupt vindo kaa prayog kar sakate hai.
 google ne haal hi main krom broujar ke tisare sanskaran kaa beta varjan release kiya he. is krom main ekstansion saport pehle se hi chaalu hote he. is sanskaran main thiming suvidhaayen bhi sammilit hai, kintu inke liye customise and control main opshans main personal staf main janaa hota he. daayen diye chitra main dekhen jisamen thims vaale anubhag main get thims naamak batan milegaa jo ki google kii theme gailaree main le jaataa he. ise clic karne par neeche vaale chitra jaisaa prishth khulegaa, jahaan se theme chune ja sakate hai.
