 4.10.209.0
 microsoft security essentials ek antivirus software he, jo kuch khaas tarah ke softwaron se suraksha pradaan karata he. inamen computor virus, spyware, rootkit or trogen horse aadi aate hai. iske license kaa upyog ghar main upyog karne vaale or kuch chote vyapaar karne vaale log bina kisi kharch ke kar sakate hai. 
 ise microsoft ke duusare antivirus software ke samaan virus paribhaasha or scaning ingine se banaaya gaya he. yah real-time suraksha pradaan karata he or lagaataar computor main hone vaali gatividhiyon par najar rakhata he or jaise hi koi nayaa fil downlod hota he yaa banataa he to vaise hi ye us nae fil ko scan karata he. yadi koi samasya milti he to use niskriy kar detaa he. 
 is utpaada kaa apane yujar interphes, kam sansaadhan ke upyog or nickshulk license ke kaaran saraahna ke saath-saath sakaaratmak samiksha milti aa rahi he. oktober 2009 ke antivirus pareekshan main sabhi vyapak malware ko hataa kar pareekshan main safalataa pai thi, lekin uske kuch saalon baad oktober 2012 main hi lagaataar suraksha or sudhaar main giravat ke kaaran ye pareekshan main vifal ho rahaa. june 2013 main isane suraksha main sabase kam ank "shunya" hasil kiya. march 2012 main opsavat ke anusaar ye uttar america kaa sabase lokapriy or vishwa kaa doosraa sabase lokapriy utpaada tha. is kaaran iske jaise kai nakli antivirus kaa nirmaan bhi hua, jo isi kii tarah apane utpaada ko dikhaane kaa prayaas karte hai. 
 microsoft security essentials ek antivirus software he, jo hanikaarak software jaise computor virus, spyware, rootkit or trogen horse aadi se system ko bachaata he. isse pehle microsoft do alag alag prakaar ke antivirus software kaa vikaas kar rahaa tha, jisamen se ek windos live vankere naam kaa sadsyta shulk vaala antivirus tha or doosraa windos dephender naam kaa ek nickshulk antivirus tha. windos dephender keval windos 8 tak hi simit tha or keval adware or spyware se hi suraksha pradaan karata tha. in donon software kaa sthaan isane le liya.
 yah virus kii paribhaasha downlod karne ke liye microsoft apadet par nirbhar rahata he or din main teen apadet hota he. upyogkarta chahen to use microsoft ke website se sidhe bhi apadet kar sakate hai. 30 sitambar 2011 ko ek galat paribhaasha vaala apadet aayaa tha, jo google krom ko malware ke rup main chinhit karne laga. is samasya ko maatr teen ghanton ke bheetar hi thik kar diya gaya. yah software mul rup se windos ekspe, vista or 7 main chalata tha, lekin sanskaran 4.5 ke baad se ye ekspe main chalanaa band ho gaya or microsoft ne 14 july 2015 se iske liye apadet dena hi band kar diya.
 iske aane ke lagbhag saal bhar baad, microsoft ne iskaa doosraa sanskaran bhi bazaar main utaar diya. ye takneekee jhalak vaali sthiti main 19 july 2010 ko tha or antim nirmaan 16 disambar 2010 ko main laayaa gaya. isamen network inspection system bhi he, jo network se ghuspaith par najar rakhti he. ye windos vista or windos 7 ke saath kaam karti he. iske saath saath isamen ek nae tarah kaa malware hataane kaa ingine daalaa gaya he. iske baad se internet explorer se jud kar ye web se jude samasyaon kaa samaadhaan karata he.
 sanskaran 2.0 ke aane ke solah maheene baad 3.0 sanskaran ko chhod kar sidhe hi 4.0 ko laayaa gaya. ise saarvajanik beta ke rup main 18 navambar 2011 ko laayaa gaya, is samay microsoft ne ise chalaa paane main sakasham logon ko bina iske sanskaran ko bataae hi ise upyog karne hetu aamantrit kiya tha. iskaa pehla beta sanskaran 29 navambar 2011 ko laayaa gaya. 24 april 2012 ko iskaa antim nirmaan vaala sanskaran laayaa gaya. microsoft ne isi ke saath saath ek kaarykram kii shuruyaat bhi kii, jisamen jin logon ko nae beta sanskaran pradaan kar unase pratikriya le rahe the.
 21 farvari 2014 ko beta 4.5 ko laayaa gaya. isi din microsoft ne ghoshnaa kiya ki ab iske baad se windos ekspe main ye kaam nahin karegaa. puraane sanskaran 14 july 2015 tak swatah hi apadet lete rahenge. uske baad logon ko microsoft ke website se unhen apadet karnaa padegaa.
 4.10 kaa nayaa sanskaran 29 navambar 2016 ko nikaala gaya. ye windos vista or windos 7 ke liye 4.10.209.0 tha. is apadet se isse pehle vaale sanskaran main aae truti ko thik kar liya gaya tha. isse pehle kaa 4.10.205.0 vaale sanskaran main kisi fil yaa phoulder ko chun kar mouse se daayen clic karne se jo scan karen vaala vikalp dikhataa tha, wo hat gaya tha. jise agle apadet se waapas laayaa gaya.
 microsoft ne aadhikarik rup se windos vista or ekspe ke liye apadet dena band kar diya. puraane sanskaaran ab bhi unamem kaam karte hai or virus kii jaankaree dene vaala apadet ab bhi miltaa rahata he. ye 14 janvari 2020 tak windos 7 ke liye bhi uplabdh rahegaa. 
 ye software windos 8 yaa uske baad vaale sanskaran main kaam nahin karata he, kyonki unamem unki apni suraksha pranali, jise windos dephender kehete hai, uplabdh rahati he. 13 sitambar 2011 ko california main ek sammelan main microsoft ne windos 8 ke develper privyuu ko dikhaaya, jisamen suraksha ke aise intjaam kiye gaye hai, jisase virus se prabhavit yuasabi flash memory se boot karte samay system ko bachaayegaa. 15 sitambar ko windos 8 ke develper blog ne is baat kii pushti kiya ki ab windos 8 main virus se suraksha karne kaa kaary windos dephender karegaa. isi ke saath vaale video main, jesan garms ne bataaya ki ye software "ection centre" main antivirus or spyware tul ke rup main judega or ye dikhaaya ki ye kis tarah malware ko roktaa he. 3 march 2012 main softpedia ne windos 8 ke privyuu kii samiksha kii, or bataaya ki kis tarah windos dephender or microsoft security essentials 4.0 main samanta he. softpedia ke anusaar windos 7 main apgrade karne ke liye microsoft security essentials ko hataanaa hi padataa he.
 18 navambar 2008 ko microsoft ne ek muft suraksha software banane kii ghoshnaa kii, jiskaa naam code naam morron rakhaa gaya tha. is software ke vikaas ne microsoft kii antivirus ke vyapaarik ranneeti ko hi badal diya. shulk aadhaarit suraksha software ke sthaan par morro, puri tarah muft suraksha pradaan karne vaala or bahut kam system risource lene vaala software he. 
 23 june 2009 ko microsoft ne iskaa saarvajanik beta sanskaran sanyukt raashtra, israel, china or brazil ke 75,000 upyogkartaaon ke liye jari kiya. 
 iske bazaar main aane ke ek saal baad, 29 sitambar 2010 main microsoft ne ghoshnaa kiya ki iske lagbhag 3 karod se adhik upyogkarta ho gaye hai. opsavat ne june 2011 main apane report main bataaya ki ye vishwa ke sabase adhik prasiddh antivirus main se ek he. jiskaa puure vishwa main 10.66 pratishat or uttar america main 15.68 pratishat hissedari he. isi report ke anusaar ye uttar america kaa sabase adhik upyog kiya jane vaala antivirus tha. vishwa main iskaa sthaan chauthaa tha.
