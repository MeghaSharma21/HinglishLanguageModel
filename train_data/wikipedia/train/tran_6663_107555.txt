 bharat kala bhawan, varanasi main kaashi hindu vishvavidhyalay ke praangan main sthit ek chitrashaala he. yah asia kaa sabase bada vishvavidhyalay sangrahalay he. yahaan kii chitra vithika main 12veen se 20veen shatee tak ke bhartiya laghu chitra pradarshit hai. inke chithran main tad patr, kaagaj, kapadaa, kaath, hathi daant aadi kaa upyog kiya gaya he. pradarshit chitra bhartiya sabhyataa or samskruti ke vibhinn pahaluon ko prakashit karte hai. vithika main pradarshit chitron kaa kram govind paal ke shashan ke chaturth varsh main chitrit buddha granth 'pragnaparmita' se prarambh hota he. laghuchitron kii vikaas gaatha purvi bharat main chitrit pothee chitron se aarambh hoti he, jinamein ajanta-bhitti chitron kii utkrisht parampara tathaa madhyakaalin kala vishistataaon kaa adbhut samanvay he.
 bhartiya chitrakala ke vishay main yadi koi bhi vidwaan, shodhakarta yaa kalavid gahan adhyayan karnaa chaahe to yah baat nishchit rup se kahi ja sakati he ki use varanasi main sthit 'bharat kala bhawan' ke chitra sangrah kaa avlokan karnaa hi hoga. bharat main prachlit lagbhag samast shaileyon ke chitron kaa vishaal sangrah is sangrahalay main he. yahan kaa chitra sangrah, visheshkar laghuchitron kaa vishwa main apana ek vishisht sthaan rakhata he.
 bharat kala bhawan kii sthapana san 1920 main hui thi. vikhyath kala marmajna tathaa kalavid padmavibhushan raay krishnadas 'bharat kala bhawan' sangrahalay ke sansthaapak the. unhone apana sampurna jivan 'bharat kala bhawan' ke liye sangrah hetu samarpit kar diya. unke jivan kaa yahi samarpan or aatmavishvaas aaj 'bharat kala bhawan' ke rup main kaashi hindu vishvavidhyalay ko gauravanvit kar rahaa he. vibhinn kala kritiyon ke sangyojan main to unki abhiruchi thi hi, kintu bhartiya chitron ke sankalan ke prati unki athmiya aasthaa thi. yahi kaaran he ki 'bharat kala bhawan' na keval rashtriya str par apitu antarrashtriya str par laghu chitron ke sangrah main apana ek nijasav rakhata he.
 is sangrahalay main lagbhag baahar hajaar vibhinn shaileyon ke chitra sankalit hai. in sabhi chitron kii apni pruthak tathaa romaanchak kahaniyaan hai. yahan keval unheen chitron kii charchaa kii ja rahi he jo chitra vishwa main apana vishisht sthaan rakhte hai.
 bhartiya itihaas main mughal chitrakala ek sukhad sanyog ke rup main darbari sabhyataa or bhoga-vilaas kaa ullas apane main sanchit kiye he. mughal chitrakala kaa janm anek videshon or swadeshi chitrakaaron ke parishram or shilpsaadhana kaa parinaam he. babar bharat aate samay jin pustakon ko apane saath laayaa, unamem shahnamaa kii sachitr prati bhi thi. yah prati 200 varsh tak mughal qutubakhaane main rahi jo baad main angrejon ke haath chali gai, sanyog se is shahnamaa ke chah mahatvapurn chitron kaa sankalan is sangrahalay main surakshit he.
 kabul par adhikaar karne ke pashchaat, humayun do videshi chitrakaar khwaja abdussamad tathaa mir saiyad ali ko apane sangrakshan main iran se kabul laayaa. in chitrakaaron ne humayun kii ajna se daastaan-e-mir hamja ke chitra bananaa prarambh kar diye jo purnataya farsi style ke the. is chitrawali ke lagbhag 1400 chitra baad main akbar ke samay main banaae gaye. chitron ke chehre kii banaavat, prakruti tathaa pahanava adhikansh irani-farsi dhang kaa he. hamjanama ke 1400 chitron main se aaj praayah 150-160 chitra vishwa main praapt hai. bharatvarsh main maatr paanch chitra avshisht hai, jinamein se do 'bharat kala bhawan' sangrah main hai. bade aakaar ke ye chitra sooti kapadey par astr lagaakar banaae gaye hai.
 pahari chitron ke antargat jayadev krit gitgovind kavy par bhi sundar chitra banaae gaye jo yahan sangrahit hai. isamen basohali style kii samast vishistataaen camel-jaisi lubhaavni ankhen, samahit hai. kavi bhanudutt krit 'ras manjari' basohali ke raja kripal paal kaa priy kavy granth tha. jiskaa sundar chithran kiya gaya. 'ras manjari' main chitrit vagvidagdha nayika chitra bhavabhivyanjana, rang samaayojan evam utkrisht kala pradarshan hetu vishwa prasiddh he. is chitramala kaa parichay prishth bhi kala bhawan main vidyaman he, jo iske chitrakaar, daata tathaa kaal kii jaankaree pradaan karata he. parivarti chitrakaaron ne baramasa chitrawaliyon ke nirmaan main vishesh ruchi pradarshit kii. in vishyon ke atirikt basohali style main 'ragamala' par aadhaarit chitra bhi praapt hote hai, jinke udhaaharan bhi 'bharat kala bhawan' main surakshit hai.
 aadhunik bengal style ke unnayak avanindra nath tagore va unke shishy nandlaal bose, e.ke haldar, es.en. de, o.si. ganguli, kshitindra nath majumdaar, yamini raay tathaa gaganendra nath tagore ke chitron kaa sangrah is sangrahalay kii amuly nidhi he. es.en.de kii meghdoot chitrawali, e.en. tagore kii abhisarika tathaa umr khaiyam, nandlaal bose kii shiv-parvati, pyau, hazrat daraab khan evam chitrit postcard 'bengal chitra style' kii shreshth kritiyaan hai. hebbaar, akbar madamasi, bendre, sultan ali, dinakar kowshik, em.epa, hossain, je.em. ahivaasee, ke.es. kulkarni, vasudev smart aadi aadhunik chitrakaaron ke chitra bhi is sangrahalay kii shobhaa he.
 bhartiya chitron ke atirikt 'bharat kala bhawan' main nepal or tibet main chitrit patra, pothee chitra or chitrit thankan kaa bhi sangrah he, jisamen jai prakaash mall kaaleen san 1765 i. tithi yukt chitrit tantric pothee evam praayah 13-14veen i. shatee kaa ratnasambhav thankan ullekhneey he.
 vividh madhyamon- kaagaj, kapada, kashtha, sheesha, hathi daant, tad patr, abhrak tathaa chamade par chitrit ukt sabhi shaileyon ke chitra is sangrahalay kii dharohar hai jo arakshit sangrah ke atirikt chhavi, nicholas, rorikh, elice boner tathaa banaras vithikaaon main pradarshit he.
