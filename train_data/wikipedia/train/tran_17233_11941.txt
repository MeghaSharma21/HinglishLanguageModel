 jaal aplikeshan yaa web aplikeshan yaa web app aisa aplikeshan software hota he jo kisi web brouzar main chal sake. iskaa prayog karne ke liye prayogkarta ko apane computer par web brouzar ke alaava kisi vishesh software ko daalane kii aavashyaktaa nahin hoti. i-mail jaise kai aplikeshan, jinhen prayog karne ke liye 1990 ke dashak ke madhya tak prayogkartaaon ko pehle apni mashiinon par i-mail aplikeshan karidakar daalana zaruri tha, dhire-dhire web appon main parivartit ho gayi, jis se unhen prayog karnaa ati-saral ho gaya. aam-taur par web app apana adhiktar kaary web app nirmaankarta ke sarvaron par karti he jo ki prayogkartaaon se chupa hua hota he.
