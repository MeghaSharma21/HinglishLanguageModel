 bhartiya sthapaty main hindu mandir kaa vishesh sthaan he. hindu mandir main andar ek garbhagrih hota he jisamen mukhya devtaa kii murti sthapit hoti he. garbhagrih ke upar tower-numa rachana hoti he jise shikhar kehete hai. mandir ke garbhagrih ke chaaron ore parikrama ke liye sthaan hota he. iske alaava mandir main sabha ke liye kaksh ho saktaa he.
 mandir shabd sanskrit vaanmay main adhik pracheen nahin he. mahakavy or sutaragranthon main mandir kii apeksha devalayas, devaytan, devkul, devgrih aadi shabdon kaa prayog hua he. mandir kaa sarvapratham ullekh shatapath brahman main miltaa he. shakhanyan strot sutara main prasaad ko deevaaron, chat, tathaa khidakiyon se yukt kaha gaya he. vaidic yug main prakruti devon kii puja kaa vidhaan tha. isamen darshanik vichaaron ke saath rudra tathaa vishnu kaa ullekh miltaa he. rugveda main rudra prakruti, vanaspati, pashuchaaran ke devtaa tathaa vishnu yajna ke devtaa maane gayi hai. baad main, uttarvarti vaidic sahitya main vishnu devataaon main shreshthatam maane gaye. 
 bharat kii pracheen sthapaty kala main mandiron kaa vishisht sthaan he. bhartiya samskruti main mandir nirmaan ke peeche yah satya chupa tha ki aisa dharm sthapit ho jo janata ko sahajata va vyavhaarikta se praapt ho sake. isaki purti ke liye mandir sthapaty kaa prardubhav hua. isse purv bharat main buddha evam jain dharm dwaara guha, stupon evam chaityon kaa nirmaan kiya jane laga tha. kushankaal ke baad gupt kaal main devataaon kii puja ke saath hi devalayon kaa nirmaan bhi prarambh hua. 
 praarambhik mandiron kaa vaastu vinyas buddha biharon se prabhavit tha. inaki chat chapati tathaa inamen garbhagrih hota tha. mandiron main rup vidhaan kii kalpana kii gai or kalakaaron ne mandiron ko saakaar rup pradaan karne ke saath hi deharup main sthapit kiya. chauthi sadi main bhagwat dharm ke abhyuday ke pashchaat bhagwaan kii pratimaa sthapit karne kii aavashyaktaa prathit hui. atev vaishnav matanuai mandir nirmaan kii yojana karne lage. sanchi kaa do stambhayukt kamare vaala mandir guptmandir ke pratham charan kaa mana jaataa he. baad main gupt kaal main vrihadastar par mandiron kaa nimarn kiya gaya jinamein vaishnav tathaa shaiva donon dharmo ke mandir hai. prarambh main ye mandir sada the or inamen stambh alankrita nahi the. shikharon ke sthaan par chat sapaat hoti thi tathaa garbhagrih main bhagwaan kii pratimaa, unchi jagati aadi hote the. garbhagrih ke samaksh stambhon par ashrith ek chota athwa bada baramada bhi milane laga. yahi parampara baad ke kaalon main praapt hoti he.
 bhartiya upmahadweep tathaa vishwa ke anya bhaagon main sthit mandir vibhinn shaileyon main nirmit hue hai. mandiron kii kuch sheliyaan nimanlikhit hai-
 nagar style kaa prasar himalay se lekar vindhyas parvat maala tak dekha ja saktaa he. vaastushastra ke anusaar nagar style ke mandiron kii pahchaan aadhaar se lekar sarvocch ansh tak iskaa chatushkon hona he. viksit nagar mandir main garbhagrih, uske samaksh kramshah interval, mandap tathaa ardhmandap praapt hote hai. ek hi aksh par ek duusare se sanlagn in bhaagon kaa nirmaan kiya jaataa he.
 yah style dakshin bharat main viksit hone ke kaaran dravin style kehalaati he. isamen mandir kaa aadhaar bhaag vargakar hota he tathaa garbhagrih ke upar kaa bhaag pyramidnuma sidhaa hota he, jisamen anek manzilen hoti hai. is style ke mandiron kii pramukh visheshta yah he ki ye kaaphi oonche tathaa vishaal praangan se ghire hote hai. praangan main chote-bade anek mandir, kaksh tathaa jalkund hote hai. pragann kaa mukhya pravesh dwaar 'gopuram' kehalaata he. 
 nagar or dravid style ke mishrit rup ko besal style kii sangya di gai he. yah vinyas main dravid style kaa tathaa rup main nagar jaisaa hota he. is style ke mandir vindhya parvatamala se krishna nadi ke bich nirmit hai.
 yah teen style se bhinn anya style main bhi hindu mandir nirmit hue he, jinamein se kuch is prakaar he 
 pagoda style nepal or indonesia kaa bali tapu main prachlit hindu mandir sthapaty he. yah style main chhaton kaa shrunkhlaa anulambit rup main ek ke upar dusraa rahata he. adhikansh garbhagrih bhootal str main rahata he. parantu kuch mandir main garbhagrih dusraa manzil main sthapit he. kuch mandir kaa garbhagrih sammuchit sthal main nirmit hote he, jo bhusthal se kariban 3-4 manzil ke uchaai par sthit hote he. is style main nirmit prasiddh mandir main nepal kaa pashupatinath, bali kaa pura besaaki aadi pramukh he.
