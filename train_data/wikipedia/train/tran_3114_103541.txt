 anterrashtriya nagar vimaanan sanghathan yaa ICAO vimaanashetra code yaa sthiti parichaayak ek chaar akshar kaa alphanumeric code hota he. yah harek vimaanashetra ke liye abhinn hota he. yah code anterrashtriya nagar vimaanan sanghathan dwaara abhikalpit or paribhaashit hota he. ise ICAO pralekh 7910: lokation indicators arthath sthaan parichaayak, ke naam se prashit kiya jaataa he.
