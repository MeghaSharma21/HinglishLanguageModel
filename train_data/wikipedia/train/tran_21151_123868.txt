 waal nadi, jise africans bhashaa main faal nadi bhi uccharit kiya jaataa he, dakshin africa kii orange nadi kii sabase badi upanadi he. iskaa strot ampumalanga praant ke bratten naamak nagar ke paas he jo mahanagari johanesberg se zaraa purv main sthit he. yahan se waal pashchimi disha main bahati hui uttari kep praant main kimbrali se dakshinpaschim main orange nadi se sangam karti he. kul milakar waal nadi 1,120 kimi lambi he. aitihaasik rup se yah parmparik transawaal praant kii dakshini seemaa thi. aadhunik dakshin africa main iske uttar main ampumalanga, khouteng or pashchimottar praant hai jinako yah apane se dakshin main sthit free state praant se alag karti he.
