 duniya kii number ek khilaadi dinaaraa safina ne denmark kii kerolin vojhaniyaki ko sidhe seton main 6-2, 6-4 se haraakar madrid masters tennis spardhaa ke khitaab par kabja jamaaya. 
 23 varshiya roosi khilaadi safina ne yah khitaab jitkar dikhaa diya ki ve is varsh french open khitaab kii prabal daavedaar hai. unhen pichley varsh paris main final main ana evanovich se haar jhelni padi thi. 
 safina pichley teen saptahon main lagaataar teesri baar dablutie toornament ke final main pahaunchi hai. ve stutgart main hamawatan swetlana kujnetsova se parrast hui thim, lekin agle saptaah unhone rome main kujnetsova ko haraakar badla chukaa liya tha. 
 is paraajay ke baavjood 19 varshiya vojhaniyaki agle saptaah kii dablutie ranking main sheersh 10 main sthaan bana lengi. vojhaniyaki kaa yah is varsh kaa chauthaa final tha. 
 duniya ke purv number ek purush khilaadi maraat safin kii choti bahan dinaaraa pichley kuch samay main mahila tennis main ek takat banakar ubhari hai. 20 april ko duniya kii number ek khilaadi banane ke baad se unka jeet-haar kaa record 14-1 ho gaya he. madrid masters ke final main safina ne pehle set ke tisare tathaa paanchaven game main vojhaniyaki kii sarvice bhang karte hue pehla set aasaani se jeet liya. 
 vaise vojhaniyaki ne chaar set point bachaae, lekin ve pehla set 44 minuton main haar gayi. vojhaniyaki ne duusare set main behatar pradarshan kiya, lekin safina ne apane anubhav kaa laabh uthaate hue is set ko bhi 6-4 se jitkar khitaab apni jholii main daal liya.
