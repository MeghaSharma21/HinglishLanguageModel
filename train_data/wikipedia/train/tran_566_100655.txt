 gangabai ballab sampradaay kii kaviyon main gangabai kaa sthaan pramukh he.
 gangabai kaa janm vi 0 san 0 1628 main mathura ke paas mahaavan naamak sthaan main hua tha. gangabai jati se kshatraani thim.do sau baawan vaishnavon kii vaarta main bhi gangabai kshatraani kaa ullekh kiya gaya he. gangabai goswami vitthalanath kii shishyaa thim. atah apane kirthan-padon main inhone apane naam ke sthaan par shri vitthal giridharan shabd ko prayukt kiya he kiya he. 'do sau baawan vaishnavon kii vaarta ' main di gai inaki vaarta se pataa chalata he ki vi 0 san 0 1736 main shrinath ji ne apni lila main sadeh angikaar kar liya.
 gangabai ke kuch pad 'shri vitthal giradharan laal ' naam pushti-maarg ke varshotsav granthon main sankalit hai. namen janm se lekar krishna kii vibhinn lilaon kaa varnan kiya gaya he. mahila hone kaaran inkaa madhur-lilaon kaa varnan atyadhik maryaadit he.
 gangabai ke aradhy krishna hai. krishna ke swarup kii vishesh charchaa inke padon main nahin milti. keval itnaa pataa chalata he ki shrikrishna sundar hai or rasik hai. gopiyaan inaki rup maadhuri se mugdh ho unki prapti ke liye vrath-upwas aadi kaa aacharan karti hai. gopiyon ke purvanurag ke varnan main bhi krishna kii sundarataa kaa varnan karne kii apeksha uske prabhaav kii ore vishesh dhyaan diya gaya he:-
 kintu kuch padon main haas - parihaas kii jhalak avashya mil jaati he:-
 krishna kii madhur lilaon main keval sanyog paksh kii raas,jhulan ,hori,dhyut-krida aadi lilaon kaa varnan gangabai ne apane padon kiya he. surat ke prasang ko roopaatmak style main is prakaar prastut kiya he ki vah spasht rup se rati kaa varnan nahin prathit hota :-
