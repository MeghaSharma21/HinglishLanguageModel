 abhiklitr kaarykram vah nirdesh yaa kai nirdeshon kaa samooh hota he jinka prayong abhiklitr se kisi nishchit samay par koi kaary sampann karaane ke liye kiya jaataa he. ise software program yaa keval program yaa phir strot code bhi kehete hai. program kaa prayog microprocesar main kisi soochna cokriyanvit karne ke liye kiya jaataa he. taaki procesar upayokta kii aavashyaktaa ke anusaar kaary karasake. iskaa nirmaan kaarykram bhashaa jaise kii si++ aadi main kiya jaataa he. 
 jis prakaar se kisi bhashaa kaa prayog karke kisi anya vyakti ko kisi tarah kaa nirdesh diya ja saktaa he usee tarah kaarykram bhashaa kaa prayog karke abhiklitr ko kisi tarah ke kaary karne kaa nirdesh diya ja saktaa he. sabhi kaarykram bhashaaon kaa ek nishchit praaroop hota he, jiske anusaar hi usamen strot sanket kaa nirmaan kiya jaataa he. is praaroop main kisi bhi prakaar kaa parivartan karne par vah strot sanket nishchit parinaam nahin de saktaa he.
 kaarykram bhashaaon kaa vargeekaran mukhyatah do samoohon main kiya gaya he - 1 nimna stariy bhashaa or 2. ucch stareey bhashaa.
 is samooh main un kaarykram bhashaaon ko shaamil kiya jaataa he, jo procesar tak nideshon ko preshit karne ke liye yaa to kisi bhi anuvadak kaa prayog nahin karti hai yaa phir keval assembler kaa prayog karti hai.
 nim stareey bhashaaon main likhaa gaya code mul rup main hi kriyanit hone ke liye purnat: upyukt hota he. saath hi saath kam se kam smriti grahan karata he or kriyanvayan main nunatam samay letaa he. inkaa kriyanyayan jitna saral he. inhen prayog karnaa utana hi kathin he kyonki nimna stareey bhashaaon kaa prayog karne vaale ke liye abhiklitr upkarnon se sambandhit kaaryon main purnatah daksh hona aavashyak he.saath hi saath is bhashaa main strot sanket kaa nirmaan karne main samay bhi adhik lagata he. isliye nim stareey bhashaayen chote kaarykram ke liye to upyukt ho sakati hai, parantu bade kaarykram kaa nirmaan karne ke liye inkaa prayog karnaa atyant kathin hota he. nimna stareey bhashaaon ko do virgon main vibhajit kiya gaya he: mashin stareey bhashaa tathaa asembli bhashaayen.
 abhiklitr kaarykram banane kii prakriya kai charnon main sampann hoti he.
