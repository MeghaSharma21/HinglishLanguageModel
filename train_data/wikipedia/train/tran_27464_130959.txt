  chandiprasad bhatta bharat ke gandhivadi paryaavaranvaadi or samaajik kaaryakarta hai. unhone san 1964 main gopeshwar main 'dasholi gram swarajya sangh' kii sthapana kii jo kaalaantar main chipko andolan kii matra-sanstha bani. ve is kaary ke liye ve 1982 main reman magsese puraskaar se sammanit hue tathaa varsh 2005 main unhen bharat sarkaar dwaara padmabhushan puraskaar diya gaya.bharat sarkaar dwaara saal 2013 main unhen gandhi shanti puraskaar se sammanit kiya gaya.
 adbhut jeevat ko samarpit chandi prasad bhatta gandhi ke vichaar ko vyaavhaarik rup main aage badhaane main ek safal jan netaa ke rup main ubhare hai. ‘chipko andolan’ ke rup main saumyatam ahimsak pratikaar ke dwaara vrikshon evam paryaavaran ke antarsambandhon ko shashaktata se ubhaar kar unhone sampurna vishwa ko jahaan ek ore paryaavaran ke prati sachet evam samvedansheel banane kaa abhinav prayog kiya, vahiin pratikaar kii saumyatam paddati ko safalataa purvak vyavhaar main utaar kar dikhaaya bhi he. ‘parvat parvat, basti basti’ chandi prasad bhatta kii behtareen yatraaon kaa sangrah he.
 23 june 1934 ko nirjala ekadshi ke din gopeshwar gaanv uttarakhand ke ek garib parivaar main janme shri chandi prasad bhatta saatven dashak ke prarambh main sarvodayi vichaar-dhara ke sampark main aae or jaiprakash narayan tathaa vinoba bhaave ko adarsh banaakar apane kshetra main shram kii pratishtha saamaajik samarasata, nashaabandi or mahilaon-daliton ko shashaktikaun ke dwaara aage badhaane ke kaam main jute gaye. wanon kaa vinaash rokane ke liye gramvaasiyon ko sangathit kar 1973 se chipko andolan aarambh kar wanon kaa kataan rukvaaya.
 rus, america, germany, japan, nepal, pakistan, bangladesh, france, mexico, thailand, spain, china aadi deshon kii yatraaon, saikadon rashtriya or antarrashtriya sammelnon main bhagidaari ke saath hi shri bhatta rashtriya str kii anek samitiyon evam aayogon main apane vyaavhaarik gyaan evam anubhav kaa laabh-pradaan kar rahe hai. 1982 main raman magsse puraskaar, 1983 main arkansus arkansus travelers sammaan, 1983 main little rock ke meyar dwaara sammaanik naagarik sammaan, 1986 main bharat ke mananeey raashtrapati mahoday dwaara padmashri sammaan, 1987 main sanyukt raashtra paryaavaran kaarykram dwaara global 500 sammaan, 1997 main california main pravasi bharatiyon dwaara indian for colective ection sammaan, 2005 main padam bhushan sammaan, 2008 main doctor of sains oopaadhi, govind vallabh pant krishi evam praudyogiki vishvavidhyalay pantnagar, 2010 real heroj lifetim echievement award si.en.en. ai.bi.en, -18 network tathaa rilians industries dwaara sammaan evam puraskaar praapt hue hai. 77 varsh kii umr ke baavjood bhi unamem utsaah kisi noujvaan se kam nahin he.
