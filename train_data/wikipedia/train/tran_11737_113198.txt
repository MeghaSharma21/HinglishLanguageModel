 bharat main ‘bhartiya janaganana 1961’ ke anusaar 1652 maatrubhashaayen chaar bhashaa pariwaron main vargeekrut kii gai hai. neeche boliyon ke vivaran main aadhaar dow grearsan kaa ‘bharat bhashaa surveykshan’ he, kintu nivanatam shodh adhyayan sarvopari maane gaye hai. tulana ke liye jan0 kaa ullekh kiya gaya he. boliyon ke milansaalon par matbhed asambhav nahin he. koshthkon main boliyon ke sthaan varnit he.
 sansar ke bhashapariwaron main kai drushtiyon se mahatvapurn bhaaropee bhashapariwaar kii darad shakha ke darad varg kii shina bhashaa kii boli gilgiti or kashmiri bhashaa kii kishtawari, ponguli, bhujwali, siraaji visheshat: ullekhya hai.
 yadhyapi mul lehanga tathaa sindhi kshetra pakistan main chalaa gaya he, phir bhi visthapiton ke rup main lahanda kii 14 boliyon main multani tathaa pucchhi evam sindhi kii saat boliyon main kacchi pramukhat: pai gai. 
 jan0 main marathi kii 65 boliyaan hai. dakshin maharashtra, karnataka, kerala, main boli janevaali konkani vastut: swatantra bhashaa he, marathi kii boli, jaisaa gearsan ne kaha tha, nahin he. konkani kii 16 boliyon main chetti bhashaa konkani guanij evam kudabi pramukh he. marathi ke antargat halbi, kamari, katiyaa katakaari koshtee marathi, kshatriya marathi, chhindwara shioni thakri boliyaan jan0 main ullikhit hai. shesh karhandi, mirgani, bhandari prabhruti ullekhya hai.
 urisa kii 24 boliyon main bhami, bhuiya, relli paidi pramukh hai. katak kii katki, andhra seemaa par ganjami, sanbhalpur main sanbhalpuri bhi ullekh hai.
 bengali ke antargat jan0 main ullikhit 15 boliyon main chakma kisangajia, rajwanshi pramukh he. grearsan ne saraki, khadiyathar, coach aadi bhi ginai he. jan0 main assam kii koi boli nahin varnit he, kintu grearsan ne kachaar ke hinduon kii bishunpuria kaa ullekh kiya tha.
 hindi kshetra main bihari varg main 35 matra boliya hai jinamein 
 maithili kii upboli siraajapuri purniya main boli jaati he. purvi hindi kii avadhi evam chhattisgadhi pramukh boliyaan hai. avadhi lakhimpur, sitapur, lakhanu, unnav, fatehpur baharich, barabanki, raibareli, gonda, faijabad, sultanpur, pratapgarh jonpur, mirzapur jilon kii boli he. isamen baandaa bhi gina ja saktaa he. badheli reva satna shadol ke atirikt gri0 ke anusaar damoh, jabalpur, mandla, balaghat tak faili he avadhi kii marari poali tathaa paradeshi maharashtra bhi boliyaan hai. chhattisgadhi chhattisgarh, raipur, raigarh durg bilaspur, sarguja, bastar main kaker, kabardha chaanda uttar purv main bhi boli jaati he. sargujiyaa sarguja main grearsan ke anusaar korea udaipur main bhi, guaro upboli assam main, tathaa lariya orisa main boli jaati he.
 pashchimi hindi
 puure rajasthan main rajasthani boli 71 matra boliyon sahit faili he. jan0 ke anusaar inamen bagri rajasthani banjaari, dhughaari, lamoni, gojri, hadoti khesri, malwa, maaraadi, mewari shekhavati, pramukh boliyaan hai. nimadi dhar tathaa nimad kii boli hai. gri0 main bhili tathaa khandeshi mishrit boliyaan bhashaa ke rup main pruthak varnit hai. jan0 ke anusaar bhilo kii 36 upboliyon main barel , gamati gavit, kokena, bagdi, pavari pramukh he khan desh kii ahirani khandesh main prayukt he. 
 86 boliyon ke samooh vaale pahari varg kii pashchimi pahari ke antargat bhadravahee sirmauri, bharmour, mandeali, chamealo, churhi jonsari, kului upboliyaan hai. purvi pahari me nepali tathaa madhya pahari main kumauni, gadhwali pramukh he. vastut: inamen prakritik duuri he. jan0 main gujarati kaa 27 boliyon main dhisaadi kolchi, parsi, saurashtri pramukh varnit hai. iske atirikt gamadiya, charotari, kathiyanwadi bhi ullekh he.
 punjabi kii 29 boliyon main jan0 ke anusaar bilaspuri, dogri, kangri rathi jalandhri, ferozepuri, pattiyani maanjhi pramukh boliyaan hai.
 bharat main sankhya kii drushti se duusare bhashapariwaar dravid main jan0 main 161 maatrubhashaayen ginai gai he jinamein 104 ko samvidhangat tamil, telegu, kannada malayalam chaar bhashaaon ke sandarbh main vivechit kiya ja saktaa he. tamil kii 22 boliyon main jerukul andhra main, kaikadi maharashtra main korwa pehle madras main, pattapu bhashaa andhra main pramukh boliyon ke rup main boli jaati he. gri0 ke anusaar saalwary, beradi bhi pradhan boliyaan hai. kannada kii 32 matra boliyon main pramukhatam badga he. holiya pramukhat- maharashtra main, gataar kannada m0 pra0 main, montadentyi madras main pai gai hai. korcha boli korwa kii paryaay nahin he, jaisaa gri0 main varnit he, apit yah mysore main boli janevaali, kannada kii pramukh boli he. malayalam kii 14 boliyon main yerav jati kii yerav boli mysore main, paniya madras tathaa kerala main boli jaati he. nagari malayalam trichur jile ke sanskritgya brahmanon kii malayalam he. shesh boliyaan gaun hai. tulu korgi, toda, kota, chaar bhashaaon kii bhi kai boliyaan hai. korga tulu kii pradhan boli he. ye madras, mysore, maharashtra main bikhri hai.
 iske atirikt uttar dravid samooh kii kurukh bhashaa main vaangari nagesiyaan tathaa maalti bhashaa kii sauria pramukh he. nai shodhon ke aadhaar par kaha ja saktaa he ki godi, kui, khond, koya, parji, colami konda bhashaayen siddh hui he. gri0 main ye boliyon ke rup main varnit hai. gondi kii dotli, bhariya kui kii pengu, colami kii maane evam naiki boliyaan ullekh hai.
 tibet-chini parivaar kii bhashaayen ladakh se lekar assam se purv tak uttung himani shikharon, behad jangalo, ghaatiyon main dur tak faili hui he. jan0 main 226 maatrubhashaayen ginai gai hai. tibet bhotiya varg main 33 matra boliyaan hai jinamein bhotiya, balti, bhutani, lahuli, spiti, kageti pramukh hai. kai ek kaa namakaran sthanvishesh ke aadhaar par hua he. himalay varg kii 24 matra boliyon main malani pramukh boliyut kanci, kanori rai taamang, locha pramukh bhashaa himachal pradesh main boli jaati he. assam shakha ke nefa varg main 24 maatrubhashaayen hai jinamein aakaa, haso, dafka abor, miri, tathaa mishmi pramukh maatrubhashaayen hai. assam-burmi shakha ke bodo varg main 40 maatrubhashaayen hai, jinamein bodo sahit chaar boliyaan kachari, dimasa, gaaro, tripuri mikir, rabbha, ullekhneey hai or jo assam main bikhri hui hai. naanaa varg kii kul 47 maatrubhaashaon main kinyak, aao, angaami, sema, taangkhul aadi nagaland tathaa nefa main boli jaati hai. kuki-chinavarg kii 61 maatrubhaashaon main pramukh bhashaa manipuri kii vishunapariya boli tripura tathaa kachaar main boli jaati he. iske atirikt vaisen, khongajai, halam kuki anishchit bhashaayen assam tathaa naga pahadiyon main he. burma varg kii arkanij bhashaa k modh pramukh boli tripura main boli jaati he.
 austric bhashaa parivaar kii mon khmer shakha main saat tathaa munda shakha main 58, kul milakar 65 maatrubhashaayen jan0 main varnit hai. khaasi bhashaa kii jayantiya tathaa pnar boliyaan khaasi tathaa jayantiya pahadiyon main boli jaati he. khervari bhashaa ke antargat santhali, mundaari, ho, kurukh, corku, bhumij, gadba boliyaan ginai gai hai. vastut: inhen bhashaayen kahana, jaisaa jan0 main he, adhik upyukt hoga, kyonki bodhgamyata ke siddhant ke aadhaar par unamem atyadhik duuri aa gai he. munda shakha kii shesh boliyaan hai - makaari, koda sambandhit miradha, bairi, lodhajo kharia se sambandhit hai,, nicobari, tathaa kol bhashaayen.
