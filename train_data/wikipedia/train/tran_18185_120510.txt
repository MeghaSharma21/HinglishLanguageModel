 yah hindi pustakon ke ek pramukh prakaashak hai. isaki sthapana aazaadee se pehle lahore main hui thi, baad main 1947 main yah prakaashan dilli aayaa. iske sansthaapak jansangh ke sansthaapak sadsy rahe vaidya gurudutt the. is santhaan ne rashtravaadi lekhakon kii hi pustaken prakashit kii hai. is prakaashan ke lekhakon main vir saavarkar, gurudutt, pien ok, balraj madhok, tejpal singh dhama, farhaana taj or sivakumar goyal mukhya hai.
 isaki sthapana aazaadee se pehle 1936 main lahore main hui thi, baad main 1947 main yah prakaashan dilli aayaa or knot place main bharti sahitya sadan ke naam se sthapit hua, lekin baad main iskaa naam hindi sahitya sadan kar diya gaya.
