 swami sahjanand saraswati bharat ke rashtravaadi netaa evam swatantraaa sangram senaanee the. ve bharat main kisaan aandolan ke janak the. ve aadi shankaracharya sampradaay ke dasnami sanyasi akhaade ke dandi sanyasi the. ve ek buddhijivi, lekhak, samaaj-sudhaarak, krantikari, itihaaskaar evam kisaan-netaa the. unhone 'hunkaar' naamak ek patr bhi prakashit kiya.
 swami sahjanand saraswati kaa janm uttar pradesh ke gajipur janapad main ek brahman parivaar main hua tha. unki aarmabhik rajnatik gatividhiyaan adhiktar bihar main kendrit thim or akhil bhartiya kisaan sabha ke nirmaan ke baad puure bharat main failin. unhone patna ke nikat bihata main ek ashram banaaya tha jahaan se apane jivan ke uttarardh ke saare kaam sanchalit karte the.
 swami sahjanand kii punya smriti main unke grih janapad gajipur main swami sahjanand snatakottar mahavidhyalay sthapit he.
 1889 : i. mahashivaratri ke din gajipur jile ke deva gram main janm.
 1892 : i. mata kaa dehant.
 1898 : i. jalalabad madrasa main akshararambh.
 1901 : i. loar tathaa apper primary kii 6 varsh kii shiksha 3 varshon main samaapt.
 1904 : i. middle pariksha main sampoorn uttar pradesh main chatha sthaan praapt kar chaatravrutti prapti.
 1905 : vivaah 
 1906 : patni kaa swargvaas.
 1907 : pun: vivaah kii baat jaanakar mahashivaratri ko ghar se nishkraman tathaa kaashi pahunchakar dasnami sanyasi swami achyuthanand se pratham diksha praapt kar sanyasi bane.
 1908 : praaia: varshaparyant guru kii khoj main bharat ke tirthon kaa bhraman.
 1909 : pun: kaashi pahunchakar dashashvamedha ghat sthit shri dandi swami advaitanand saraswati se diksha grahan kar dand praapt kiya or dandi swami sahjanand saraswati bane.
 1910 : se 1912 tak-kaashi tathaa darbhanga main sanskrit sahitya vyakaran, nyaay tathaa mimasaa kaa gahan adhyayan.
 1913 : swami purnanand saraswati ke prayaas se 28 disambar ko baliya main hathua-naresh kii adhyakshata main sampann a. bha. bhoomihaar brahman mahasabha main pratham baar upasthit tathaa brahman samaaj kii sthiti par bhaashan.
 1914 : kaashi se 'bhoomihaar brahman patr' nikaalkar 1916 tak uskaa sampaadan tathaa prakaashan.
 1914-15 : bharat ke vibhinn bhaagon main bhraman kar bhoomihaar brahmanon tathaa anya brahmanon kaa vivaran ekatra karnaa.
 1916 : 'bhoomihaar brahman parichay' kaa prakaashan. 
 1917 : pratham baar bhojpur ke dumri main aagaman tathaa kanyakubj or shakadwipi brahmanon ke vivaad main pan. devraj chaturvedi dwaara.
 1919 : simari aakar pun: vishwambharpur waapas.
 1920 : 5 disambar ko patna main shri majaharul haq ke nivas par tehre mahatma gandhi se raajnneetik vaarta tathaa raajnneeti main pravesh kaa nishchay kar congress main shaamil.
 1921 : gajipur jila congress kaa adhyaksh chunaa janaa tathaa ahmedabad congress main shaamil hona.
 1922 : 2 janvari ko giraftaar hokar 1 varsh kaa karawas.
 1923 : jail se chhootane par simari main nivas.
 1924 : simari tathaa aasapaas praytn se khaadi vastrotpaadan ke liye 500 charkhe tathaa 4 karghe kaa chalavaanaa prarambh. saamaajik sabhaaon main brahmanon kii ekta tathaa sanskrit shiksha prachararth bhaashan. simari main chaar maheene main 'karmakalap' .
 1925 : kaashi main yojit sanyukt prantiya bhu. bra. sabha main bhu. bra. dwaara purohiti karne se sambandhit bhaashan. disambar main khalilabad main raja chandradeshwar pra. singh kii adhyakshata main sampann a. bha. bhoomihaar brahman mahasabha main purohiti vaala prastaav pesh kar usamen mahatvapurn bhaashan.
 1925 : bhoomihaar brahman parichay kaa parivardhdan kar 'brahmarshi vansh vistaar' naam se prakaashan.
 1926 : 'karmakalap' kaa kaashi se prakaashan.
 1926 : amila, ghoshi azamgarh main yojit bhu. bra. mahasabha main bhaashan tathaa sanskrit shiksha prachaar kaa sandesh. samastipur main nivas tathaa patna main yojit akhil bhartiya bhu. bra. mahasabha main tyaagi brahman chau. raghuveer singh ko adhyaksh nirvaachit karana tathaa purohiti ke prastaav ko paarit karana.
 1927 : samastipur se aakar patna jile ke bihata main shri seetaram dasji dwaara pradutt bhoomi main shri seetaramashram banaakar sthaayee nivas tathaa pashchimi patna kisaan sabha kii sthapana.
 1928 : sonpur main 17 navambar ko sampann bihar prantiya kisaan sammelan kaa adhyaksh chunaa janaa.
 1929 : antim baar munger kii a. bha. bhoomihaar brahman mahasabha main sammilit, matbhed ke kaaran waapas.
 1930 : amaharaa main 26 janvari ko namak kanoon bhang ke kaaran 6 maah kaa karawas. hazaribagh jail main 'gita rahasy' kii rachana. jail se lautakar congress tathaa kisaan sabha ke kaaryon main sanlagn hona.
 1934 : bhucamp peediton kii seva hetu bihar main samiti gathit kar seva kaary kaa sanchaalan.
 1935 : patna jila congress ke adhyaksh pradeshik congress kaarysmiti ke sadsy tathaa a. bha. congress samiti kaa sadsy chunaa janaa.
 1936 : a. bha. kisaan sabha kaa sanghathan tathaa pratham adhiveshan kaa adhyaksh pad grahan. 
 1937 : a. bha. kisaan sabha kaa mahamantri chunaa janaa.
 1938 : 13 may se 15 may tak sampann a. bha. kisaan sammelan ke adhyaksh.
 1939 : a. bha. kisaan sabha ke mahamantri nirvaachit tathaa 1943 tak ukt pad par.
 1940 : ko ramgarh main shri subhaashachandr bose kii adhyakshata main yojit a. bha. samjhauta virdohi sammelan ke swagatadhyaksh. uparyukt sammelan main bhaashan ke liye 3 varsh kaa karawas.
 1941: se 43 : jail main kai pustakon kii rachana.
 1944 : 14,15 march ko bejabaada main a. bha. kisaan sammelan ke adhyaksh.
 1948 : 6 disambar ko congress kii praathmik evam a. bha. congress kii sadsyta kaa tyaag tathaa samyavaadee sahyog se kisaan morcha kaa sanchaalan.
 1949 : mahashivaratri ko bihata main hirak jainti samaaroh samiti dwaara saath lakh rupee kii thili bhent tathaa uskaa tadarth daan.
 1949 : 9 april ko ayodhya main sampann a. bha. virakt mahamandal ke pratham adhiveshan main shankaracharya ke baad adhyakshiya bhaashan.
 1950 : april main raktchaap se vishesh peedit hokar prakritik chikitsarth dow shankar nair se chikitsa prarambh. may-june se apane anany anuyaayi kisaan netaa pan. yamuna karyi ke nivas par nivas.
 1950 : 26 june ko pun: dow nair ko dikhaane aane par mujffarpur main hi pakshaaghaat kaa akraman tathaa 26 june kii raatri 2 baje prasiddh vakil pan. muchkund sharma ke nivas par dehant.27/6/50 ko shav kaa patna gandhi maidan main lakhon logon dwaara antim darshanadow mahmud kii adhyakshata main shoksabhaa, netaaon dwaara shradhdhanjali.
 28/6/50 : dow shrikrishna singh mukhyamantri tathaa anya netaaon kaa arthi ke saath shri seetaramashram, bihata patna main pahunchnaa tathaa vahiin antim samadhi.
 brahmarshi vansh vistaar, brahman samaaj kii sthiti, jhootha bhay mithyaa-abhimaan, karm-kalap, gita-hriday kranti or sanyukt morcha, kisaan sabha ke sansmaran, kisaan kaise ladate hai, jharkhand ke kisaan, kisaan kya karen, meraa jivan sangharsh aadi.
 swamiji ne kramash: bhoomihaar brahman, kaashi or lok sangrah, patna naamak patron kaa sampaadan 1911-16 or 1922-24 tak safaltaapurvak kiya. unke lekh 'hunkaar' patna, janata, vishaal bharat tathaa kalyaan ke ishwarank tathaa yogank main bhi prakashit hue the.
