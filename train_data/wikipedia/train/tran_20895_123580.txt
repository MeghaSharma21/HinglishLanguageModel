 krishi main hari khaad us sahaayak fasal ko kehete hai jisaki kheti mukhyat: bhoomi main poshak tattvon ko badhaane tathaa usamen jaivik padaathon kii purti karne ke uddeshy se kii jaati he. praaia: is tarah kii fasal ko iske hari sthiti main hi hal chalaakar mitti main mila diya jaataa he. hari khaad se bhoomi kii upjau shakti badhati he or bhoomi kii raksha hoti he.
 mrudaa ke lagaataar dohan se usamen upasthit paudhe kii badhwaar ke liye aavashyak tatva nasht hote ja rahe hai. inaki kshatipurti hetu va mitti kii upjau shakti ko banaaye rakhane ke liye hari khaad ek uttam vikalp he. bina gale-sade hare paudhe ko jab mrudaa kii natrajan yaa jeevansh kii maatra badhaane ke liye khet main dabaaya jaataa he to is kriya ko hari khaad dena kehete hai. 
 hari khaad ke upyog se na sirf natrajan bhoomi main uplabdh hota he balki mrudaa kii bhautik, rasinic evam jaivik dasha main bhi sudhaar hota he. vaataavaran tathaa bhoomi pradooshan kii samasya ko samaapt kiya ja saktaa he laagat ghatane se kisaanon kii aarthik sthiti behatar hoti he, bhoomi main sookshm tatwon kii aapurti hoti he saath hi mrudaa kii urvara shakti bhi behatar ho jaati he.
 1. hari khaad keval natrajan va carbonic padaarthon kaa hi saadhan nahin he balki isse mitti main kai anya aavashyak poshak tatva bhi prachur maatra main uplabdh hote hai. 
 2. hari khaad ke prayog main mrudaa bhurabhuri, vaayu sanchaar main acchhee, jaldhran kshamta main vruddhi, amleeyta/kshariyata main sudhaar evam mrudaa ksharan main bhi kami hoti he. 
 3. hari khaad ke prayog se mrudaa main sookshmjivon kii sankhya evam kriyasheeltaa badhati he tathaa mrudaa kii urvara shakti evam utpaadan kshamta bhi badhati he. 
 4. hari khaad main mrudaajanit rogon main bhi kami aati he. 
 5. iske prayog se rasaynic urvarkon main kami karke bhi tikaoo kheti kar sakate hai. 
 hari khaad ke liye dalhani fasalon main sanai, dhaincha, lobia, urad, mung, guar aadi fasalon kaa upyog kiya ja saktaa he. in fasalon kii vruddhi sheeghra, kam samay main ho jaati he, pattiyaan badi vajandaar evam bahut sankhya main rahati he, evam inaki urvarak tathaa jal kii aavashyaktaa kam hoti he, jisase kam laagat main adhik carbonic padaarth praapt ho jaataa he. dalhani fasalon main jadon main nitrogen ko vaataavaran se mrudaa main sthir karne vaale jeevanu paye jate hai.
 adhik varsha vaale sthanon main jahaan jal paryaapt maatra main uplabdh ho sanai kaa upyog karen, dhaincha ko sookhe kii dasha vaale sthanon main tathaa samasyagrast bhoomi main jaise kshareey dasha main upyog karen. guar ko kam varsha vaale sthanon main retili, kam upjau bhoomi main lagaayen. lobia ko achche jal nikaas vaali kshareey mrudaa main tathaa mung, urad ko kharif yaa greeshm kaal main aise bhoomi main le jahaan jal bharaav na hota ho. isse inaki faliyon kii acchhee upaj praapt ho jaati he tathaa shesh paudha hari khaad ke rup main upyog main laayaa ja saktaa he. 
 hari khaad ke mukhya chaar gun hai-
 kam upjau samasyagrast mrudaa main in fasalon ko kam samay main acchhee badhwaar ke liye dalhani fasalon main 20 se 25 kigra. natrajan prati hectare tathaa gaiyr-dalhani fasalon main 40 se 50 kigra. natrajan prati hectare kii dar se buaai ke samay daalane se kaaphi laabh hota he. in fasalon ko parati avasthaa main uchit nami ke samay beej ko chhidakar bo diya jaataa he evam ek jutai mitti palatney vaale hal se karke pata chalaa diya jaataa he. mrudaa main thodi maatra main 40 se 50 kigra. phosphorus prati hectare tathaa 20-25 kigra. prati hectare potash kii maatra daalane se sadane galane kii kriya shighrata se purna hone main sahaayta milti he. 
 hari khaad kii fasal ko buaai ke din se 40 se 60 din kii avasthaa main pata lagaakar mitti palatney vaale hal se 15 se 20 semi kii gaharaai par palat dena chahie. samay se pehle palat dene se paryaapt carbonic padaarth praapt nahin hote tathaa der se palatney se resha adhik hone ke kaaran sadane-galane main adhik samay lagata he. atah achit samay par palatnaa laabhadaayak hota he. adhik varsha tathaa adhik tapakram kii dasha main sadane-galane kii prakriya sheeghra-prarambh ho jaati he. dhaincha yaa sanai kii fasal ko hari khaad ke rup main palatney ke baad pani bharkar dhan kii ropai kii ja sakati he.
