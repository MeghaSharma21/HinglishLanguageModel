 java motorcycle swatantra bharat main bani 250 sisi kshamta vaali tu stroke racing motorcycle thi. san 1960 main mysore shahar main faruk irani dwaara sthapit idial java limited ne ise checkoslovakia kii company java motors se license lekar banaaya tha. us samay 250 sisi model main eendhan kii khapat ke lihaaj se yah sarvottam motorcycle thi jo 3 litre petrol main 100 kilometre kii duuri tay karti thi. 1970 se lekar 1990 ke dashak tak puure 30 saal racing motorcykilon main iskaa koi muqabla nahin tha. do stroke or do silencer vaali yah anuthi motorcycle thi jiske sabhi kalpurje kaward hua karte the. isaki sabase badi visheshta yah thi ki iske kik starter se hi giyar badalane kaa kaam ho jaataa tha. iske alaava iskaa rakrakhaav bhi bahut hi sasta or aasaan tha. amuman ise kisi motor makenic ke paas le jane kii jarurat hi nahin padati thi. 1960 se 1974 tak puure pandraa saal java brand se yah bharat main banaayi gayi baad main brand or model badal kar yezadi ho gaya. parantu bhartiya bazaar main globaligeshan evam vishwa vyapaar sanghathan ke kathor maapdandon ke kaaran san 1996 main isaki nirmaataa company ko iskaa utpaadan band karne ke liye baadhya hona padaa.
 yadhyapi java motorcycle banane vaali company ko band hue barason ho chuke hai phir bhi mysorevasiyon ke liye aaj bhi java motor cycle sirf ek bike hi nahin balki bahut kuch he. java shabd iske checkoslovakian sansthaapak janik frantisek or wanderr ke naam ke pehle do-do aksharon ja or vaa ko milakar banaaya gaya tha. aaj bhi karnataka praant main rahane waalon ko is motor cycle ke naam main mysore ke ek pratapi raja jaaychamarajendra wadiyaar kii smriti jhalkati he. mysore rajya main aaj bhi prati varsh anterrashtriya java yezadi diwas badi dhoomdhaam se manaaya jaataa he jisamen san 1945 ke check java model se lekar yezadi 350 tak kii saikadon vintage motorcycilen sadakon par daudti nazar aati hai. 
 bharat main sthapit is company dwaara shuruyaat main java ke keval do hi model bazaar main laaye gayi jinka license java motors checkoslovakia se liya gaya tha:
 mul rup se checkoslovakia kii ek company java motors se license praapt kar yah motorcycle bharat main banaayi gayi thi. "forever bike-forever vailu" ke aakarshak vigyaapan ke saath bhartiya bazaaron main utaari gayi yah motor cycle yuvaaon kii pehali pasand thi. iske purze checkoslovakia se hi aayaat kiye jate the. baad main isaki nirmaataa company idial java limited ne apana brand java se badal kar yezadi kar diya. 
 bharat kii is company ne ye motorcycilen na keval desh main bechin apitu turki, nigeria, srilanka, misr or guatemala jaise 61 deshon ko niryaat bhi kin. safed rang kii yezadi rodking motorcycle to guatemala main keval police ke prayog hetu hi niryaat kii jaati thi.
 kaalaantar main iskaa ek or model yezadi 250 MT naam se banaaya gaya. yah model vishesh rup se parvatarohiyon kii jaruraton ko dhyaan main rakhkar banaaya gaya tha. ise venezuela ko niryaat kiya jaataa tha.
 bharat main isaki nirmaataa company ne sholavaram main ek racing team bhi banaayi hui thi jo is motorcycle ko neshnal motorcycle raily championships main bheja karti thi. sabase ullekhneey baat yah he ki kai varshon tak lagaataar sabhi muqablon main java motorcycle hi jitati rahi. "java ko keval java hi haraa sakati he, koi duusari motorcycle nahin" yah jumala un dinon ek muhaavara ban gaya tha. 
 1996 main jab idial java limited company bharat main band hui us samay vahaan 175 sisi, monark, dealks, rodking or clasik II jaise kai modelon kaa utpaadan jari tha. itani aakarshak motorcycle banane vaali company ke band ho jane kaa kaaran bharat main pradooshan maanakon par kathor niyantran va utpaadan factory main lebar problem kaa hona bataaya jaataa he. jabaki vastusthiti yah he ki bhartiya bazaar main globaligeshan ke chalate vishwa vyapaar sanghathan ke kathor maapdandon ke kaaran aisa hua.
 bharat main java motorcycle banane kaa license dene se purv san 1952 se 1958 tak java motors company ne 500 sisi OHC tip 15 motorcycle bhi banaayi thi. yah four stroke motorcycle thi parantu atyadhik manhagi hone va avishwasaneey gunvattta ke kaaran kaamyaab nahin hui. isaki asafalataa se sikh lekar baad main java ne tu stroke model main 250 sisi kii kai bikes bharat main banaayin. 3 litre petrol main 100 kilometre kii duuri tay karne vaali java va yezadi motorcycilen apane behatar maapdandon va aakarshak modelon ke kaaran kai dashakon tak na keval deshi apitu videshi bazaar main bhi chhaayi raheen.
 bharat main bhale hi java motorcycle kaa utpaadan band ho gaya ho parantu united kingdom main aaj bhi iskaa ek samunnat model java 350 sisi tu stroke clasik va sports 2012 jaise do naamon se launch kiya gaya he. puri tarah se electronic ignishan, electric starter va automatic pump oil injection jaisi atyanudhik suvidhaon se yukt java kaa yah model ireland dwaara apane yahan aaayaatit karke europe ke adhikansh deshon main bechaa jaataa he.
 isaki nirmaataa company idial java limited dwaara neshnal notercycle raily championships ke liye vishesh rup se banaayi gayi factory-team ke alaava mysore main iske shaukeenon ne java friends club bhi banaaya hua he. is club ke dwaara prati varsh july ke duusare ravivaar ko anterrashtriya java diwas dhoomdhaam se manaaya jaataa he jisame keval java or yezadi motorcycilen hi shaamil kii jaati hai.
