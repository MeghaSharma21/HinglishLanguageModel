 lord ervin bharat main 1926-1931 i. tak governor general tathaa samrat ke pratinidhi ke rup main vysarai the.
 bharat main bad rahi swarajya tathaa sanvaidhanik sudhaaron kii maang ke sambandh main inaki sanstuti se 1927 i. main lard simon kii adhyakshata main british sarkaar ne simon kamishan kii niyukti kii, jisamen sabhi sadsy angrej the. falswaroop saare desh main kamishan kaa bahishkaar hua, 'simon, waapas jaao' ke naare lagaae gaye, ore kaale jhandon ke pradarshan ke saath andolan hua. sanders ke netritv main police kii laathiyon kii chot se lala lajpatraay kii mrityu ho gai. bhagat singh ke dal ne ek varsh ke bheetar hi badale ke liye sanders kii bhi hatya kar di.
 prarambh main bharat oupaniveshik swarajya kii hi maang karata rahaa, kintu 26 janvari, 1929 ko akhil bhartiya rashtriya congress ke lahore adhiveshan main jawaharlal nehru ke netritv main 'purna swarajya' kii ghoshnaa kii gai tathaa shapath lee gai ki prayek varsh 26 janvari ganatantr ke rup main manaai jaaegi.
 simon kamishan kii report ke anusaar 1930 i. main lard irvin kii sanstuti se sanvaidhanik sudhaaron kii samasya ke samaadhaan ke liye landan main ek golmes sammelan kaa aayojan kiya gaya, jiskaa gandhi ji ne virodh kiya. saath hi gandhi ji ne sarkaar par dabaav daalane ke liye 6 april, 1930 se namak satyagrah ched diya. saare desh main namak kanoon todaa gaya. gandhi ji ke saath hajaaron vyakti giraftaar hue. sar tejbahadur sapru kii madhyasthata se gandhi-irvin-samjhauta hua. yah samjhauta bhartiya itihaas kaa ek pramukh mode he. isamen 21 dharaayen thim jinke anusaar golmes kanfrarence main bhaag lene ke liye gandhi ji taiyaar hue tathaa yah tay hua ki kanoon todane kii karrwai band hogi, british saamaanon kaa bahishkaar band hoga, police ke karnaamon kii jaanch nahin hogi, andolan ke samay bane adhyadesh waapas honge, sabhi raajnneetik kaidi chhod die jaayenge, jurmaane vasuul nahin honge, jabt achal sampatti waapas ho jaaegi, anyaaypurna vasuuli kii kshatipurti hogi, asahyog karanevaale sarkari karmchaariyon ke saath udaartaa barti jaaegi, namak kanoon main dheel di jaaegi, ityadi. is samjhaute ke falswaroop 1931 i. kii dwitiya golmes sammelan main gandhi ji ne madanamohan malviya evam shrimati sarojini naidu ke saath bhaag liya.
 yadhyapi lard irvin ne ek samrajyavaadee shaasha ke rup main swadeshi andolan kaa puraa daman kiya, tathaapi vaiyaktik manushya ke rup main ve udaar vichaaron ke the. yahi kaaran he ki rashtravaadi netaaon ko inhone kaaphi mahatv pradaan kiya. inke jivit smarak ke rup main nai dilli main vishaal 'irvin aspataal' kaa nirmaan karaayaa gaya.
