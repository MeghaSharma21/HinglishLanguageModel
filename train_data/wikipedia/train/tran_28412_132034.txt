 shatbhisha, nakshatron main 24vaan nakshatra he. is nakshatra kaa swami rahu he, isaki dasha main 18 varsh he va kumbh raashi ke antargat aata he. jahaan nakshatra swami rahu he vahiin raashi swami shani he. rahu kaa prabhaav lagbhag shani vrut hi padataa he. kuch jyotishiyon ne isaki drushti maani he, lekin jab akaash mandal main iskaa astitv hi nahin he to drushti kisi? yadi rahu mesh lagn main ucch kaa ho to iske parinaam bhi shubh milte hai.
 mesh kaa rahu ho to aise jataka prabal rup se shatruhanta hota he. gupt vidya main safalataa milti he. santan ke maamalon main baadha bhi aati he. rahu mesh lagn main shashthbhaav main ho to shatruhanta hoga.
 vrishabh lagn main rahu ho to aise jataka par stri gami bhi hote hai. dwitiya bhav main hota vaani main chaturata hoti he. singh, kanya, makar, kumbh main ho to uttam parinaam detaa he. ashtum main ho to gupt vidya kaa jaankaar bana detaa he. vahi aise jataka bhi sexi hote hai. yadi shani kaa ucch kaa swarashi kaa yaa mitra raashi kaa ho to aise jataka nishchit apni vidya ke bal par ucch safalataa pate hai.
 mithun lagn main nakshatra swami rahu lagn main aise ho to raajnneeti main uttam safalataa pate hai. vakaalat main bhi safal hote hai. chaturth bhav main ho to sthaneey raajnneeti main uttam safalataa milti he. tritiy bhav main ho to shatruhanta hoga. shani kii sthiti main shani main lagn, chaturth, navam, pancham main ho to uttam safalataa paane vaala hoga.
 kark lagn main nakshatra swami dasham main ho to aise raajnneetik guru vidya main safal hote hai. rahu kii sthiti tritiy, ekadash, saptam main thik rahegi. raashi swami bhi yadi mitra swarashi kaa ho to parinaam bhi shubh milenge.
 singh lagn main rahu lagn main ho to chaturth ekadash main ho to dasham, dwitiya, shashth main shubh rahegaa.
 kanya lagn main rahu dasham bhav main safal raajnneetijna bana detaa he. lagn, pancham, navam main bhi shubh fal degaa. swami shani pancham, navam, ekadash, dasham lagn main hoto uttam rahegaa. shani chaturth, pancham, navam main shubh rahegaa.
 vrishchik lagn main rahu dasham, ekadash, chaturth, tritiy bhav main shubhafaladaayi rahegaa. vahiin shani kii sthiti saptam, ekadash, tritiy bhav main navam main ho to thik rahegaa vahiin shani dasham main saptam, dwitiya, tritiy, ekadash main shubhafaladaayi rahegaa.
 makar lagn main rahu shashth, pancham, lagn, navam main shani dasham lagn navam shashth main shubhafaladaayi rahegaa.
 kumbh lagn main rahu chaturth, pancham, lagn, navam, shashth main shubhafaladaayi rahegaa. kumbh lagn main rahu chaturth, pancham, saptam main shubhafaladaayi rahegaa. shani lagn, chaturth, pancham, navam main thik fal degaa.
