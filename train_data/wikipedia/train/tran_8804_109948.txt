 i-mail hindi main likhi kaise jaye ?
 in samasyaon kaa samaadhaan unicode hindi aadhaarit i-mail ke jariye sambhav he. parantu yahan aapako duusari tarah kii samasyaon se do-chaar hona padataa he. aapke paas yadi purana oparating system he, to vah unicode ko samarthan nahin karegaa or aap unicode hindi main kaary karne main asamarth honge. yadi aapkaa oparating system unicode kaa samarthan karata bhi he to aapkaa i-mail claayent or aapkaa brouzar bhi unicode hindi samarthan pradaan karne vaala hona chahie. kuch halaaton main, jaise ki windos98 main oparating system str par unicode hindi samarthan uplabdh nahin he, parantu is par internet explorer 6 kaa sampurna package santhaapit kar ise unicode hindi ke aanshik upyog laayak banaaya ja saktaa he. windos 2000 tathaa iske baad ke sanskarnon main tathaa varsh 2002 ke baad ke jari linaks ke praaia: sabhi lokapriy sanskarnon main difolt rup se hindi unicode ke samarthan kii suvidha upayoktaaon ko oparatins system str par hi uplabdh hoti he. or inamen i-mail claayent tathaa brouzar bhi shaamil hote hai. aajkal java aadhaarit kuch aise auzaar bhi jari kiye ja chuke hai jinhen chalaane ke liye aapke computer par kisi prakaar ke unicode samarthan kii aavashyaktaa nahin he, parantu java-virtual mashin santhaapit hona aavashyak he.

 hindi main i-mail aadan-pradaan ke anubhavon main yah paaya gaya he ki haal philhal, web aadhaarit jimail seva jo ki aapako aapke i-mail claayent par esasail 3 surakshit pop3 or esaimatipi mail seva bhi pradaan karti he, sarvottam he. pramukhat: web aadhaarit jimail outluk express 6 par bhi utani hi aasaani se kaam karata he jitna ki linaks ke ke-mail par. i-mail client ko ji-mail samarthit banane ke liye bas aapako i-mail claayent ke pop/esaimatipi sarwar ko kramash: pop.gmail.com tathaa smtp.gmail.com par set karnaa hoga or esasail suraksha sahit port kramash: 465 tathaa 995 paribhaashit karnaa hoga. agar ye setting aapako kathin prathit hote hai to iske liye vistrut dishanirdesh to hai hi, ek sansthapana yogya fil bhi he jise chalaakar in vinyas ko aap swachaalit rup se set kar sakate hai. ji-mail kii anya suvidhaayen ye hai jo ise kisi bhi anya i-mail seva se behatar banaate hai:
 1. pramukhat: web aadhaarit yah i-mail seva aapako pop3 tathaa esaimatipi suvidha bhi detii he. yani aap kisi brouzar se bhi i-mail kar sakate hai tathaa i-mail claayent jaise ki outluk express yaa ke-mail se bhi. jimail aam upyog ke liye muft he. haalaaki abhi yah kuch simit upayoktaaon ke liye hi jari kiya gaya he.
 2. isamen aapako 2 jibi bhandaaran suvidha milti he, saath hi 10 embi fil atechament kii suvidha bhi.
 3. unicode hindi i-mail ke vishay field bhi hindi main die ja sakate hai tathaa i-mail ke aadan-pradaan main yahuu kii tarah bhashaa encoding khandit nahin hota or i-mail barkaraar rahata he.
 4. isamen oski aadhaarit hindi jaise ki shusha yaa web-duniya font main bhi kaam kar sakate hai.
 5. suraksha ke liye ji-mail esasail kaa upyog karata he jise aap jeepiji ke jariye or badha sakate hai. isamen antarnirmit spom filter bhi he jo aapako avaanchit mail se dur rakhata he.
 6. i-mail ke bheetar hi google khoj kii suvidha milti he, jo jaahir he google khoj kii tarah twarit or samuchit hota he.
 7. isamen ek bilkul nai vichaardhaara yukt i-mail ko flag karne kii suvidha di gai he jisase i-mail ko mitaayaa na jaakar hameshaa ke liye apane dak-dibbe main rakhe rah kar badhiya upyog main liya ja saktaa he.
 8. yah atom tathaa aresas feeds kaa bhi samarthan karata he jo aapke i-mail main internet ke tazaatreen khabaron tathaa anya aankadon ko swachaalit ekatra karata he.
 9. ji-mail ke jariye kisi bhi computer se kahin se bhi hindi unicode main kaam kiya ja saktaa he. udhaaharan ke liye broujar aadhaarit chahaari jaise online unicode sampadak ke jariye kaat/chipka kar hindi unicode main kaam kiya ja saktaa he.
 10. abhi ji-mail kaa beta sanskaran jari kiya gaya he, jo nity prati nikhaar par he. sambhav he iske purna sanskaran main hamen or bhi achche fichar milen.
 ji-mail kii ek khami he ki yah javascript par chalata he at: iske liye aapke mashin main java santhaapit hona aavashyak to he hi aapkaa brouzar java script chalaane laayak hona bhi aavashyak he. udhaaharan ke liye web aadhaarit i-mail ke liye linaks system main mojila or firfox se to aap ise chalaa sakate hai, parantu kankarar broujar se nahin, jisamen java kaa kuch simit samarthan he. parantu yah brouzar kii khami he na ki ji-mail kii. windos system main aisi koi samasya nahin he chunki praaia: sabhi nae broujar javascript kaa samarthan karte hai. javascript par aadhaarit hone ke kaaran upayokta ko ji-mail ke nae sanskarnon ko alag se downlod karne kii aavashyaktaa nahin hoti. jab bhi aap ji-mail kaa upyog karte hai, yah naveenatam sanskaran hi rahata he. tathaa ek baar prarambh main script ke chalane main thodaa samay lene ke pashchaat baad ke upyog main yah atyant teevr hota he chunki phir yah sirf paath ko hi downlod karata he, jo maatr kuch biton ke hote hai.
