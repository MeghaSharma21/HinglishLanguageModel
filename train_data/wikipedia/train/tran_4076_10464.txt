 pracheen samay se hi sansar ke anek bhaagon main vividh prakaar kii shilaayein bhavannirmaan ke kaaryon main aati raheen hai. bharat bhi un katipay deshon main he jo is kaary main sahasron varshon se nipun rahe hai or aaj bhi vibhinn prakaar ke pattharon se nirmitt anek bhawan hamaari sabhyataa or samskruti kaa sandesh de rahe hai. madhyakaal evam aadhunik kaal main bhi anek bhawan in pattharon se banaae gaye, jo sundarataa ke utkrisht udhaaharan hai.
 bhavannirmaan main prayukt shilaon main kuch visheshtaayen honi aavashyak hai, udaahranarth rituksharan rokane kii kshamta, khanan main sugamawa, varna evam sundarataa aadi. nirmanshilaon main nimnaankit mukhya he :
 yah pathar mukhya rup se vindhyas parvatamala main praapt hota he. iske itane vishaal strot hai tathaa yah itani adhik suvidha se mil jaataa he ki anek kaaryon main iskaa upyog sahaj hi sambhav he. vindhyas parvatamala kaa balua pathar samaan aakaar ke sookshm kanon se nirmit he tathaa isaki banaavat bhi shilaon main lagbhag niyamit tathaa samaan rahati hai. yah laal, pile tathaa bhure aadi anek varnon main prapya he. anek sthalon par iske khananakentre hai, jahaan se yah desh ke vibhinn bhaagon main bhavannirmaan kaaryon main upyog ke liye vitarit kiya jaataa he. madhyakaalin tathaa arvachin anek utkrisht bhawanon kaa nirmaan is pathar se kiya gaya he. bharat main utkhanit sabhi pattharon main iskaa sthaan sarvocch he.
 bharat main chuunaa pathar bhi desh ke anek bhaagon main prachur maatra main praapt hota he. kuch samay purv tak isaki maang adhik nahin thi, kintu gat kuch hi varshon main khapat itani adhik badi he ki anek kaaryon main vibhinn prakaar ke chuunaa pattharon kaa vishesh upyog hone ke kaaran is kshetra main paryaapt anveshan kii aavashyaktaa hui. uttam varg kaa chuunaa pathar aajkal loh tathaa ispaat udyog main dravak ke rup main, anek rasinic kaaryon main, viranjan churna, soda kshar tathaa calcium carbide aadi ke nirmaan main evam shakkar or vastrodyog main, prayog kiya jaataa he. iske atirikt cement varg ke chuunaa pattharon ko vishaal band yojanaon ke sameep bhi sthit karnaa padataa he. in anek upyogon ke kaaran chuunaa pathar ke vishaal nikshep desh ke anek bhaagon main utkhanit hue hai tathaa anek par khanan kaary kiya ja rahaa he. jin pradeshon main chuunaa pathar kaa khanan mukhyat: hota he unamem urisa, bihar, uttar pradesh, purvi punjab, himachal pradesh, madhya pradesh, madras, maharashtra tathaa saurashtra sammilit hai. bhartiya bhutatvik sarvakshan ke anveshanon ke aadhaar par cement ke anek kaarkaane sthapit kiye gaye hai tathaa anek kaa nirmaan kaary chal rahaa he.
 rasinic varg kaa chuunaa pathar, jisaki khapat aajkal adhikadhik vruddhi par he, uttarapradesh ke dehradun tathaa mirzapur, madhya pradesh ke jabalpur, madras ke tineveli jile, saurashtra ke porbandar tathaa junagarh evam assam kii khaasi pahadiyon main prachur maatra main praapt hota he.
 vartmaan tathaa bhavishya main aavashyak cement ke liye bharat main mul kaccha maal paryaapt maatra main paae jate hai. aajkal cement chuunaa pathar tathaa mrittika, athwa shell ke atyant sookshm pise hue mishran dwaara, jisamen donon avyavon kaa anupaat sadaiv samaan rakhaa jaataa he, nirmit hota he. mishrit padaarth, jo pankak ke rup main hota he, praarambhik dravan tak, ispaat ke 8-10 foot vyas kii lambii, rambh ke aakaar kii bhatthiyon main tapt kiya jaataa he. in bhatthiyon main bheetar kii ore ushmarodhi padaarthon kii iten lagi hoti he tathaa bhatthiyaan swayan hi ek ore ko thodi jhuki rahati hai, jisase mishran dhire dhire sarvaadhik santapt bhaag kii ore bade. is santapt bhaag ko niyamit rup se tapt karne ke liye churnit koyala tathaa nistapt padaarthon se nikalne vaali gaison ke jalane se utpann ushma kaa upyog kiya jaataa he. is prakaar tapt mishran ko jipsom ke churna kii upyukt maatra ke saath milakar ‘bal evam t yub chakki’ main avakrut kiya jaataa he. sookshm churna ke rup main isi ko ‘cement’ kehete hai.
 bharat main cement ke kaarkaane apane kaary main deshi chuunaa pathar kaa prayog karte hai. parivahan suvidha ko drushti main rakhte hue desh main cement ke kuch naveen kaarkaane sthapit karne ke praytn ho rahe hai.
 gat kai varshon se bharat main jipsom kii khapat din par din bad rahi he. sindri khaad sanyantr sthapit hone ke purv jipsom ke utpaadan kaa ardhansh keval cement udyog main prayukt hota tha. san 1948 se travankor main ek khaad sanyantr ne ammonium sulphate kaa utpaadan madhyam str par prarambh kar diya. isi prakaar 31 oktober san 1951 se sindri ke khaad sanyantr main bhi ammonium sulphate kaa utpaadan prarambh hua, jisamen prati din 1500-2000 ton jipsom kii aavashyaktaa hoti he.
 bharat vibhaajan se purv pashchimi punjab main lavanshrinkhalaa hi jipsom kaa mukhya strot samajhi jaati thi, kintu iske pashchaat yah bhaag pakistan main chalaa gaya tathaa bharat ko anya sroton kii aavashyaktaa kaa anubhav hua, jo badhati hui jipsom kii khapat ko puraa kar saken. parinamswaroop bhartiya bhuvigyaan surveykshan ne rajasthan, madras tathaa saurashtra main jipsom ke nikshepon kaa vistrut adhyayan prarambh kar diya.
 rajasthan ke jipsom nikshep jipsite ke rup main bikaner tathaa jodhpur main paae jate hai. in nikshepon main jamsar kaa nikshep sarvaadhik vishaal tathaa mahatvapurn he or yah strot aajkal sindri ke khaad sanyantr kii aavashyaktaapurti karata he. madras ke tiruchirpalli jile ke antargat critacius shilaon kii aniyamit patticoon main jipsom mila he.
 saurashtra main mrittikaon tathaa avamrid main tritiyak yug ke ‘gaj’ chuunaapattharon ke aadhaar ke sameep jipsom ke prapthisthaan hai. vishaltam nikshepon main, jo halar jile main khadi ke sameep mili hai, 25 lakh ton jipsom kii maatra hone kaa anumaan he.
 kuch vishaal nikshep himalay ke katipay bhaagon, jaise kashmir, sikkim, tathaa bhutan evam bharat ke anya bhaagon main praapt hue hai. kashmir ke nikshep to atyant aashajanak hai, yadhyapi parivahan ke abhaav ke kaaran inkaa upyog abhi nikat bhavishya main sambhav nahin he, tathaapi suniyojit kaarykram ke pashchaat ye nikshep atyant laabhprad honge, isamen andeh nahin.
 pracheen, madhyakaalin evam arvachin kaal main bharat main anek bhavy bhawan sangemarmar se nirmit hue. yadhyapi bharat main anek sthanon par sangemarmar praapt hota he, tathaapi uttari pashchimi bharat kaa sangemarmar apane gunon main adwitiya he. nagaur jile main makraana naamak sthaan kaa sangemarmar atyant sundar hota he, jo agra sthit tajmahal ke nirmaan main prayukt hua tha. rajasthan ke anek bhaagon main vibhinn varnon evam vibhinn aakaar ke kanon dwaara nirmitt sangemarmar praapt hota he. iske prapthisthaan jodhpur, jaipur, udaipur, tathaa alvar aadi hai.
 madras ke coyambootor tathaa mysore ke chitaldrug or mysore jilon main bhure shwet varna kaa sangemarmar miltaa he. kuch nikshep selem, madurai thirunelaveli jilon main bhi hai. madhya pradesh ke jabalpur, betul or chhidwara tathaa maharashtra ke nagpur tathaa siwani jilon main bhi sangemarmar ke upyogi nikshep praapt hue he. baroda se sundar haraa, gulabi tathaa shwet, rivakanth se kaala, andhra main kurnul jile se pit-haraa tathaa pilaa evam krishna tathaa guntur jilon kii naarangi virachanaaon se lekar pit-samudrihara aadi varnon tak ke sangemarmar mili hai.
 bahya himalay ke kumaun, gadhwal, kongda tathaa chamba aadi main slate pathar kaa khanan kiya jaataa he tathaa isko gharon kii chhaton ke nirmaankarya main prayog kiya jaataa he. dholadhar shrinkhalaa kaa slate pathar atyant upyogi he tathaa vividh aakaaron main praapt hota he. yahan slate ke pathar purnataya shuddh silicy shila ke rup main miltaa he, jiskaa varna pilaa bhura he. kongda ghati sthit punjab ke dharmashala jile main kunpaara naamak sthaan par slate kaa khanan hota he. himachal pradesh ke mandi jile kii chichot tehsil main tathaa pandoh ke sameep bhi slate pathar nikaala jaataa he. purvi punjab ke gudgaanv jile main rewadi ke paas bhi khanan kaary kiya jaataa he. kuch slate pathar bihar ke singhbhum or munger jilon main bhi hai, kintu inkaa adhik upyog nahin hota. andhra ke kurnul jile main markapur naamak sthaan par skool slate kaa khanan kiya jaataa he. nellore jile main bhi slate ke kuch nikshep hai.
 vibhinn virgon kaa slate pathar maharashtra, kashmir, madhya pradesh, mysore tathaa assam aadi pradeshon main praapt hota he.
 granite, besalt, charanokite tathaa anya anek shilaayein bharat ke vibhinn bhaagon se milti hai tathaa suvidhanusaar khanan sthanon ke sameep ke kshetron main hi inkaa prayog bhawan nirmaan ke kaaryon main kiya jaataa he.
