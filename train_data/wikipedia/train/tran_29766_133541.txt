 uganda ganraajya purvi africa main sthit ek landla desh he. isaki seemaa purv main kenya, uttar main sudan, pashchim main congou loktantrik ganraajya, dakshin pashchim main rwanda or dakshin main tanzania se milti he. desh ke dakshini hisse main victoria jhil kaa ek bada bhaag shaamil he, jisase kenya or tanzania se seemaa nirdhaarit hoti he. uganda naam buganda rajshahi se liya gaya he, jisamen desh kaa dakshini hssi, rajdhani kampala ko shaamil kar, aata tha. desh kii ek tihaai jansankhya antarrashtriya garibi rekha se neeche jeevanyaapan karti he. 
 uganda purvi madhya-africa kaa ek vishuvatrekhiy desh he, jo purnat: antarvarti he. iske uttar main sudan, pashicham main congo purv main kenya, dakshin-pashchim main ruanda, evam dakshin main tanganyika desh tathaa victoria jhil sthit he. iskaa kul kshetrafal 93,981 varg mile he, jiskaa 13,689 varg mile bhaag jalagrast evam daldali he.
 desh kaa adhikansh bhubhag pathari he, jo samudratal se lagbhag 4,000 foot unchaa he. pashchimi seemaa par rubanjori parvat sthit he, jiskaa ucchtam shikhar samudratal se 16,791 foot unchaa he, jakaki purvi seemaa par sthit algan parvat kii adhiktam oonchaai 14,178 foot he. desh ke madhya main kyoga evam dakshin main victoria jhilein sthit hai.
 samudratal se adhik unchaai par sthit is desh kaa taap anya vishuvatrekhiy pradeshon kii tulana main nune he. ausat vaarshik taap uttar main 15° sein0 evam dakshin main 22° sein0 he. vaarshik tapantar saadhaaran hai. ausat vaarshik varsha kii maatra uttar main 35 inch evam dakshin main 59 inch tak he.
 pashchim ke ucch pradesh main lambii ghaas tathaa wanon kii prachurtaa he. uttar ke shushktar kshetra main choti ghaas hi adhik milti he. desh ke dakshini bhaag main prakritik vanaspati saaf karke bhoomi ko krishiyogy bana liya gaya he, hisamen kele kii upaj mukhya he. kahin kahin hathi ghaas ugti he, jisaki oonchaai 10 foot tak ho jaati he.
 pashuon main hathi, dariyaan ghoda, bhainsa, bandar ityadi adhik hai. kuch bhaagon main sher, jiraph tathaa gande bhi milte hai.
 krishi main kapaas, kahavaa, ganna tambaakoo tathaa chaay kii upaj mahatvapurn he. anya fasalon main kelaa, makka or bajra ullekhneey hai.
 khanij tanba utkhanan tathaa kapaas evam kahavaa sambandhi udyog pramukh hai. anya udyog dhandhon ke antargat vastranirmaan, cement, madiraa, chini, lakdi chirane tathaa sabun nirmaan kaa kaam hota he.
