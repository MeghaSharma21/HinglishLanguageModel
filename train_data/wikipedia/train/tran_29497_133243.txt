  bellur krishnamcharee sundarraj ayangaar bharat ke agrani yoga guru the. unhone ayangaryog kii sthapana kii tathaa ise sampoorn vishwa main mashhoor banaaya. san 2002 main bharat sarkaar dwaara unhen sahitya evam shiksha ke kshetra main padam bhushan se tathaa 2014 main padam vibhushan se sammanit kiya gaya. 'time' patrika ne 2004 main duniya ke sabase prabhaavshaalee 100 logon kii suchi main unka naam shaamil kiya tha. anyagaar ne jin logon ko yoga sikhaaya, unamem ziddu krishnamurthi, jaiprakash narayan, yehudi menuin jaise naam sammilit hai. aadhunik rishi ke rup main vikhyath ayangaar ne vibhinn deshon main apane santhaan kii 100 se adhik shakhaaen sthapit kii. europe main yoga failaane main ve sabase aage the. 
 karnataka ke bellur main 1918 main paidaa hue ayangaar 1937 main maharashtra ke pune chale aae or yoga kaa prasar karne ke baad 1975 main yogavidya naam se apana santhaan shuru kiya jisaki baad main desh videsh main kai shakhaaen kholi gayin.
 'lite of yoga' kaa 18 bhashaaon main anuwaad ho chukaa he.
