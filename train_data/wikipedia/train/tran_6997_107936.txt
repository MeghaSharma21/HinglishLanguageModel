 soochna ke samooh or unke naamon ko prabandhit karne ke liye upyog kii jane vaali sanrachana or tark niyamon ko "file system" kaha jaataa he. data ko tukdon main vibhajit karke or prayek tukade ko ek naam dekar, jaankaree aasaani se alag or pahchaan kii jaati he. paper-aadhaarit soochna pranali ke naam se lete hue, data ke prayek samooh ko "fil" kaha jaataa he. computing main, ek fil ko file system niyantrit karata he iskaa nirmaan is liye hua kii yah bataane kaa koi tarika nahin he ki jaankaree kaa ek tukdaa kahaan band ho jaataa he or aglaa shuru hota he. data kaise sangrahit or punarprapt kiya ja saktaa he? ek file system ke bina, storage madhyam main di gai jaankaree data kaa ek bada hissaa kaha hoga ? 
 file system ke kai alag-alag prakaar hai. prayek main alag sanrachana or tark, gati, lacheelaapan, suraksha, aakaar or adhik gun hote hai. kuch file system ko vishisht anuprayogon ke liye istemaal karne ke liye dizine kiya gaya he. udhaaharan ke liye, iesao 9660 file system vishesh rup se optical discs ke liye dizine kiya gaya he.
 file system kaa istemaal vibhinn prakaar ke storage upkarnon par kiya ja saktaa he jo vibhinn prakaar ke media kaa upyog karte hai. upyog main sabase aam storage divaais aaj ek hard discs drive he. upyog kiye jane vaale anya prakaar ke media main flash memory, chumbakeey tepe or optical discs shaamil hai. kuch maamalon main, jaise ki tmpfs ke saath, computer kii mukhya memory, ram) kaa upyog alpavadhi upyog ke liye asthaayee fil system banane ke liye kiya jaataa he.
