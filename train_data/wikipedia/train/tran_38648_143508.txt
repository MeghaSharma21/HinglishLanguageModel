 Google Indic Keyboard se aap apane Android phone par apni maatrubhaasha main sandesh likh sakate hai, saamaajik network apadet kar sakate hai yaa email likh sakate hai. abhi isamen nimna keyboard shaamil hai:
 - angrezi keyboard- asamiya keyboard - bengali keyboard - gujarati keyboard - hindi keyboard - kannada keyboard - malayalam keyboard - marathi keyboard - oriya keyboard - punjabi keyboard - tamil keyboard - telugu keyboard 
 apane phone par, yadi aap apni bhashaa ko upar uski mul lipi main pard sakate hai, to aap apni bhashaa input karne ke liye Google Indic Keyboard ko install or upyog kar sakate hai; anyathaa sambhawatah aapkaa phone aapaki bhashaa kaa samarthan nahin karata he.
 Google Indic Keyboard input ke vibhinn mod kaa samarthan karata he:
 - lipyantaran mod - uccharan ko angrezi aksharon main likhakar apni maatrubhaasha main output praapt karen 
 - mul keyboard mod - sidhe mul lipi main likhen.
 - hastlekhan mod - sidhe apane phone kii screen par likhen.
 - hinglish mod - yadi aap “hindi” ko ek input bhashaa ke rup main chunate hai, to angrezi keyboard angrezi or hinglish donon tarah ke shabdon kaa suzav degaa.
 ise sakasham kaise kiya ja saktaa he or difolt keyboard ke rup main set kaise kiya ja saktaa he?- Android 5.x or nae varshan par:setting -> bhashaa or input kholen, “keyboard or input paddatiyaan” anubhag main, vartmaan keyboard -> keyboard chune par jaaen -> “Google Indic Keyboard” ko check karen -> “bhashaa or input” par waapas aaen -> vartmaan keyboard -> “angrezi or bhartiya bhashaaen ” chune. kisi input box main likhte samay, aap screen main neeche daaen kone main keyboard iken ko clic karke bhi difolt input paddati badal sakate hai. - Android 4.x par:setting -> bhashaa or input kholen, “keyboard or input paddatiyaan” anubhag ke antargat, Google Indic Keyboard ko check karen, phir difolt clic karen or “input paddati chune” samvaad main “Google Indic Keyboard” chune. kisi input box main likhte samay, aap notification kshetra main “input paddati chune” ko chunakar bhi difolt input paddati badal sakate hai.
