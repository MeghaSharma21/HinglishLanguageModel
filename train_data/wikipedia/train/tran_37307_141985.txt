 rog ke nimnlikhit rup maane jate hai :
 1. anthracosis - yah koyale kii dhul se utpann hota he. isko koyale kii khaanon main kaam karanevaalon kii thysis bhi kaha jaataa he.
 2. siderosis - isko silico-siderosis bhi kaha jaataa he. tin, tanba, jasta or lohe kii khaanon main tathaa ispaat ke kaarkaanon main kaam karanevaalon ko in dhaatuon ke kanon ke shwaas ke saath baraabar jate rahane se yah hota he.
 3. silicosis - yah bhayankar rog dhul main sookshm kanon ke atyadhik maatra main rahane se utpann hota he. sphatik ke kaarkaanon main kaam karanevaalon main yah rog adhik hota he. alumina, carbon, jipsom tathaa hamatite ke kan yadi dhul main silica ke saath mili rahete hai, to pratikriya main anishtakar prabhaav kam hote hai.
 4. bissinosis - yah rooyee or namade ke kaarkaanon main kaam karanevaalon ko hota he.
 5. asbestosis - yah rog asbestus kii saamagree banaanewaalon main hota he.
 6. bagaosis - yah rog chini ke kaarkaanon main khoi kii dhul ke kanon ke phufphuson main pravesh se hota he. khoi main chah pratishat silica hota he.
