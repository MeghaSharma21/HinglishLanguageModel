 saahityacharya dow. pandit pannalaal jain ek vishisht jain vidwaan the.dow kastoorchand kasliwaal ne 20 veen shatabdi ke jain vidwaanon ke 20 sabase sammanit vyakti ke rup main unhen mana he.unka shikshan satr 1931 se 2001 tak 70 varshon tak rahaa.
 pannalaal ji kaa janm san 1911 main sagar ke ek chote se gaanv paraguwan main hua. inke pita evam mata kaa naam gallilaal va janakibai tha. 1919 main unke pita kii mrityu ke baad vah sagar chale gaye. pannalaal ne sagar main ganeshprasad varni dwaara sthapit prasiddh santhaan "sattaraka sudhaa tarangini sanskrit paathshaala" main adhyayan kiya, jo ki vartmaan main ganesh varni sanskrit vidhyalay ke naam se prachlit hai. lambey samay tak pannalaji ne bhi vahaan padhaayaa. unka vivaah 1931 main hua tha. unki patni kaa naam sundarbai tha. usee varsh unhen vidhyalay main shikshak ke rup main niyukt kiya gaya tha. baad main unhone varanasi main siyadwaad mahavidhyalay main adhyayan kiya. unhone 1936 main saahityacharya kii padavi arjeet kii. unhone 1933 se 1983 tak sagar ke moraaji main sattaraka sudhaa tarangini sanskrit paathshaala main maardarshak ke rup main apana adhikansh jivan vyateet kiya, jise vartmaan main ganesh varni digambar jain sanskrit mahavidhyalay ke rup main janaa jaataa he. sevanivrutt hone ke baad unhone 8 janvari 2001 tak jabalpur main varni digambar jain gurukul me adhyaapan kaa kaary kiya.
 unhone kshullak ganeshprasad varni dwaara prasiddh aatmakatha "meri jivan gaatha" ke donon khandon ko sampadit kiya jo 1949 or 1960 main prakashit hue the. unhone 1974 main shri ganeshprasad varni smriti granth bhi sampadit kiya.
 unhone pramukh bhikshuon or tapaswinon ke liye ek maardarshak ke rup main kaary kiya tha. unhone 1980 main sagar main dhawala granthon par vyaakhyaan shuru karne main acharya vidyasagar kii sahaayta kii. unhone aksar sagar main mahilashram kii adhyaksh aryika vishuddhmati ko sanchaalan hetu salaah di. vah dronagiri or bada malhara main jain sansthaanon se bhi jude the. 1955 main unhone aitihaasik gajrath ko prachaar mantri ke rup main vyavasthit karne main madad kii. unhone 1981 main khajuraho main gajrath tyohaar main bhi sahaayta kii. vah 1946 se 1985 ke dauraan vidwaanon kii ek agrani parishad digambar jain vidyaparishad se jude the.
 ek saadhaaran, sabhy or nirvivaad paaramparik vidwaan, unhen mahakavi harichand par unke kaam ke liye 1973 main sagar vishvavidhyalay dwaara piechdi se sammanit kiya gaya tha.
 unhen 1969 main sanskrit ke shikshan main unke shaikshanik yogadan ke liye bharat ke raashtrapati dow ve ve giri dwaara sammanit kiya gaya tha.
 pannalaal jain mahapurana, uttarpurana or padmapurana samet sanskrit puranic sahitya ke anuwaad or tippaniyon ke liye sabase zyada jane jate hai.unhone shastreey sanskrit sroton ke aadhaar par kai vrutodapan granth bhi banaaye hai.unhen 1976 main sagar shravakas dwaara vidya-varidhi sheershak diya gaya tha.vah ek abhootpurv lekhak the. unke dwaara virachit moulik kritiyaan-
 8 march 2001 ko kundalpur tirth main, bhagwaan adinath bade baba kii chaaya main, pannalaal ji ne dhyanamgn hokar apana sharir ko tyaag diya. sagar main, namak mandi ke madhya se guzarane vaale maarg ko saahityacharya pandit pannalaal jain maarg naam diya gaya he, jisamen sadak ke bich main ek ardh-pratimaa he. unki murti namak mandi tirahaa par ek tarah se rakhi gai he.
 achal jain, jinhone sanskrit sahitya main unke yogadan par unke shodh patron ko likhaa tha, ko indore vishvavidhyalay dwaara piechdi se sammanit kiya gaya tha.
 pannalaal jain smarak samiti naamak sanghathan sagar main sanskrit vidwaanon kaa sammaan karti he..jain muni pratishkshgar unke pote the.
