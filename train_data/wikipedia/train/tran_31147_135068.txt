 computer, mobile evam anya compyating divison par hindi main tip karne ke liye vividh tareeke prayog kiye jate hai. inamen vividh prakaar kii typing, vividh prakaar ke keyboard evam software shaamil hai.
 maikenical tiperiter par hindi main tip karne ke liye remington keyboard leout kaa prayog kiya jaataa he. yah atyant kathin leout he, kyonki har chihn ke liye alag alag kunjiyon ko yaad rakhana padataa he, computer kii tarah isamen sanyuktakshar aadi ekaadhik kunjiyon ke sangyojan se nahin banaaye ja sakate. parantu maikenical yukti hone ke kaaran tiperiter par ekmaatr yahi leout sambhav he.
 vartmaan main tiperiter dwaara hindi typing kaa sthaan computer par hindi typing ne le liya he tathaa iskaa prayog bahut hi kam dekhne ko miltaa he.
 computer par typing do prakaar kii hoti he-
 yah vidhi computer par unicode pranali ke aane se pehle prayog kii jaati thi. isamen puraane samay ke hindi font prayog kiye jate the. is typing kaa upyog sirf chapai aadi ke kaamon main hi hota he. kisi verde prosesar main hindi kaa non-unicode font chunakar tip kiya jaataa he tathaa uskaa print liya ja saktaa he. kisi anya computer par vah taxt dikhne ke liye vah vishesh font install hona chaiye anyathaa hindi taxt kii jagah sirf kachra dikhataa he.
 unicode hindi typing kii nai vidhi he. unicode kii visheshta he ki yah font evam keyboard leouton par nirbhar nahin karti. aap kisi bhi unicode font evam kisi bhi keyboard leout kaa prayog karke hindi tip kar sakate hai. unicode font main likhi hindi dekhne ke liye us font vishesh kaa computer main hona jaruri nahin he. kisi bhi unicode hindi font ke hone par hindi dekhi ja sakati he. adhiktar naye oparating systamon main unicode hindi font bana-banaaya aata he.
 yadi phone main hindi pradarshan hetu samarthan he to input kaa vikalp ho bhi saktaa he or nahin bhi. yadi phone main hindi input kaa vikalp ho to hindi main sandesh bheje ja sakate hai tathaa vaib par kahin bhi taxt box main hindi likhi ja sakati he. is vikalp ke hone par mobile se hindi main email bhejne, chittha likhane, tippanee karne samet internet par tamaam kaary hindi main kiye ja sakate hai.adhiktar phonon main hindi tankan ke liye tii-9 system hota he. sabhi devanagari varnon ko keepad ke nau batano par samyojit kiya jaataa he tathaa baar-baar dabaakar sahi varna tip kiye jate hai. iske atirikt kai phonon menti-9 purvanumaan bhi hota he jisase ki purvanumani input dwaara keval kuch batan dabaakar shabdon ko tip kiya ja saktaa he.tach screen vaale jin kuch phonon main hindi samarthan uplabdh he unamem iske liye inascript onscreen virtual keyboard hota he. yah computer ke ikhmai kii tarah hota he jisase ki phone main kahin bhi hindi likhi ja sakati he.
 vaise to hindi ke liye bahut se keyboard leout prachlit hai parantu mukhya teen hai:-
 tach typing se aashay hota he bina keyboard ko dekhe keval chhukar tip karnaa. hindi main mul rup se inascript evam remington tach typing pranaliyaan hai. phonetic mul rup se tach typing pranali nahin he lekin yadi angreji kii kvarty tach typing kaa abhyaas ho to phonetic ko bhi bina dekhe tip kiya ja saktaa he.
 site typing se aashay he ki kunjiyon ko dekh-dekhakar tip karnaa. hindi main site typing ke liye devanagari varna ankit inascript leout ke keyboard uplabdh hai. iske atirikt inascript leout ke steekar bhi milte hai jinhen maujuuda keyboard par chipkaaya ja saktaa he. inascript leout main varna vishesh kram main hote hai isaliye ye tach or site donon prakaar kii typing ke liye upyukt hota he.
 duusari vidhi phonetic main yaddapi dekh-dekhakar hi tip kiya jaataa he lekin vah sahi rup main site typing bhi nahin he kyonki devanagari ke bajaay angreji ke varnon ko tip karke hindi likhi jaati he. remington site typing ke liye katai upyukt nahin kyonki iskaa leout kaaphi kathin hota he or varnon ko dhundhna bahut mushkil hota he.
