 sutluj uttari bharat main bahanevaali ek sadanira nadi he. iskaa puranic naam shatadru he. jisaki lambaai punjab main bahane vaali paanchon nadiyon main sabase adhik he. yah pakistan main hokar bahati he.
 dakshin-pashchim tibet main samudr tal se 4,600 meter kii oonchai par iskaa udgam maansarovar ke nikat rakshas taal se he, jahaan iskaa sthaneey naam logachen khambaav he. 
 udgam sthal se himachal pradesh main pravesh karne se pehle yah pashchim kii ore mudakar kailash parvat ke dhaal ke paas bahati he. yahan se yah nadi gahare khaddon se hokar bahati he or parvat shreniyon kii kramik oonchai sutluj ghati main chabutaron main parivartit ho jaati he. himachal pradesh ke pahaado se apana rasta tay karaate huye yah nadi punjab ke nangal main pravesh karti he. nangal se kuch kilometre upar himachal pradesh ke bhakhda main sutluj par baandh banaaya gaya he. baandh ke peeche ek vishaal jalashay kaa nirmaan kiya gaya he, jo govind sagar jalashay kehalaata he. bhakhda nangal pariyojna se panbizli kaa utpaadan hota he, jisaki aapurti punjab or aasapaas ke rajyon ko kii jaati he. punjab main pravesh ke baad yah nadi dakshin-purv ke ropar jile main shivalik pahadiyon ke bich bahati he. ropar main hi yah pahaad se maidan main utarati he, yahan se yah pashchim kii ore teji se mudakar punjab ke madhya main bahati he, jahaan yah best doaab or malwa ko vibhajit karti he. harike main beas nadi sutluj main milti he, jahaan se yah dakshin-pashchim kii ore mudakar bharat or pakistan ke bich seemaa rekha nirdhaarit karti he. iske baad yah bharat ko chhodkar kuch duuri ke liye pakistan main fazilka ke pashchim main bahati he. bahawalpur ke nikat pashchim kii ore yah chanaab nadi se milti he. donon nadiyaan milakar panchanad kaa nirmaan karti he. 
 rugveda ke nadisukt main ise shutudri kaha gaya he.vaidic kaal main saraswati nadi 'shutudri' main hi milti thi. parvarti sahitya main iskaa prachlit naam 'shatadru yaa shatadru' he. valmiki ramayana main keky se ayodhya aate samay bharat dwaara shatadru ke paar karne kaa varnan he. mahabharat main punjab kii anya nadiyon ke saath hi shatadru kaa bhi ullekh he.shrimadbhagwat main iskaa chandrabhaga tathaa marudavridha aadi ke saath ullekh he-'sushoma shatadrushchandrabhagamarudavridha vitasta'.vishnu puraana main shatadru ko himavan parvat se nissruth kaha gaya he- 'shatadruchandrabhagadya himavatpadanirgtah'. vastav main sutluj kaa strot ravananad naamak jhil he jo maansarovar ke pashchim main he. vartmaan samay main sutluj 'bias' main milti he. kintu 'di mihraan of sindh and itraj trivuteries' ke lekhak reberty kaa mat he ki '1790 i. ke pehle sutluj, bias main nahin milti thi. is varsh bias or satluj donon ke maarg badal gaye or ve sannikat aakar mil gai.' shatadru vaidic shutudri kaa roopaantar he tathaa iskaa arth shat dhaaraon vaali nadi kiya ja saktaa he. jisase isaki anek upanadiyon kaa astitv ingit hota he. greek lekhakon ne satluj ko hejidras kaha he. kintu inke granthon main is nadi kaa ullekh bahut kam aayaa he. kyonki alakshendra kii senaayen beas nadi se hi waapas chali gai thi or unhen beas ke purv main sthit desh kii jaankaree bahut thodi ho saki thi.
 punjab kii samruddhi ke peeche sutluj kaa bhi yogadan he. sutluj par bhakhda par bane baandh se na sirf bijli kii aapurti hoti he, balki isse rajya kaa bada hissaa baadh se bhi bachaa rahata he. nagal baandh kii nahar, sarhind or best doaab kii nahar, jo ropar se nikalati he, sarhind jaisi sahaayak nahar, rajasthan nahar or bikaner nahar, jo husainivaala se nikalati he, sabhi sutluj se hi pani praapt karti hai.
