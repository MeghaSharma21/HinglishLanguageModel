 anju bobi jorge malayalam: അഞ്ജു ബോബി ജോര്ജ്ജ് ek bhartiya athlete he. anju bobi jorge ne 2003 main paris main yojit vishwa athletics championships main lambii kud main kaansya padak jeet kar itihaas rachaa tha. is uplabdhi ke saath vah pehali aisi bhartiya athlete baniin, jisane vishwa athletics championships main 6.70 meter kii chalaang lagaate hue padak jeeta. 2005 main ieeef vishwa athletics final main unhone rajat padak jeeta, is pradarshan ko vah apana sarvshereshth prardashan maanati hai. 
 anju kaa janm kerala main changnashshery ke kochuparambill parivaar main ke.tii.marcos ke ghar hua. shuruuaat main unke pita ne unhen athletics sikhaaya, aage chalakar koruthod skool main unke prashikshak ne atheletics main unki ruchi viksit kii. unhone apni schuli shiksha sikeem koruthod skool se puri kii or vimala college se snaatak kiya. san 1991-92 main skool athlettic sammelan main unhone 100 meter baadha va riley daud main jeet hasil kii or lambii kud va unchi kud pratiyogitaaon main vah duusare sthaan par raheen, is prakaar ve mahilaon kii champion baniin. anju kii pratibha ko rashtriya skool khelon main sabane dekha jahaan unhone 100 meter baadha daud or 4x100 meter riley main teesraa sthaan praapt kiya. ve kalikat vishvavidhyalay main thim.
 haalanki unhone apni shuruuaat heptathalan ke saath kii thi, lekin baad main unhone kud kii pratiyogitaaon par dhyaan dena shuru kiya or 1996 dilli junior asian championships main lambii kud kaa khitaab jeeta. 1999 main anju ne bangalore federation cap main tripal jump kaa rashtriya record banaaya or nepal main hue dakshin asian federation khelon main rajat padak praapt kiya. san 2001 main anju ne tiruanantpuram main yojit neshnal circuit meet main lambii kud ke apane record ko or behatar banaakar 6.74 meter kar diya. isi varsh unhone ludhiana main hue rashtriya khelon main tripal jump or lambii kud main swarna padak jeeta. anju ne hyderabad rashtriya khelon main bhi apni pratiyogitaaon main sarvocch sthaan banaae rakhaa. unhone 2002 main manchester main hue rashtramandal khelon main 6.49 meter kii kud lagaakar kaansya padak jeeta. unhone busaan main hue asian khelon main bhi swarna padak jeeta. unhone paris main 2003 main hui vishwa athletics championships main 6.70 meter lambii kud lagaate hue kaansya padak jitkar itihas rach diya, iske saath hi vishwa athletics championships main padak jeetne vaali pehali bhartiya athlete ban gayi. unhone 2003 effro asian khelon main ek swarna padak jeeta. unhone 2004 main athens main hue olympic khelon main vyaktigat rup se apana sarvshereshth pradarshan karte hue 6.83 meter kii chalaang lagaai or ve chathe sthaan par raheen. sitambar 2005 main, dakshin korea ke inchaan shahar main yojit 16 veen asian atheletics championships main unhone mahilaon kii lambii kud main 6.65 meter kii chalaang lagaakar swarna padak jeeta. unhone 2005 main hue ieeef vishwa athletics final main 6.75 meter kii chalaang lagaakar rajat padak jeeta, is pradarshan ko ve apana sabase acha pradarshan maanati hai. unhone 2006 main doha main yojit 15 wein asian khelon main mahila varg kii lambii kud main rajat padak jeeta. 2007 main amman main hui 17 veen asian athletics championships main anju ne 6.65 meter kii chalaang lagaakar rajat padak jeeta or agast 2007 main osaka hone vaali vishwa championships ke liye kwalifai kiya jahaan ve nauven sthaan par raheen. anju ne 2008 main apane satr kii shuruuaat, doha main yojit teesri asian indor championships main 6.38 meter kii chalaang lagaakar rajat padak ke saath kii or uske baad unhone apni kud main sudhaar karte hue kochi main yojit teesri dakshin asian athletics championships main 6.50 meter kii chalaang lagaakar swarna padak jeeta.
 2002-2003 main unhen bharat sarkaar dwaara khel ke prakhyat vyaktiyon ko die jane vaale pratishthit arjuna puraskaar se nawaaza gaya tathaa 2002-2004 main vishwa athlettic meet main unki safalataa ke baad unhen khel ke sarvocch puraskaar rajeev gandhi khel ratna se nawaja gaya. 2004 main unhen bharat ke chauthe sabase bade naagarik sammaan padmashri se sammanit kiya gaya. 12 farvari 2007 ko, athletics federation ke antarrashtriya sangh kii ranking main anju ko 28 wein sthaan par rakhaa gaya .
 2001 main 61 veen rank se varsh 2003 main chathe sthaan tak kii mahaj do varshon kii unki yah safal yatra, unke kade parishram or gahan yojanabaddh tareeke se kaam karne kaa nateeja he. isaki safalataa kaa shrey unke pati or coach robert bobi jorge ko jaataa he, jo anju ke anusaar unki kshamta ko baahar laane or lakshya praapt karne main unki sabase zyada sahaayta karte hai. bobi jo swayan ek maikenical ejeeniyar or purv rashtriya champion hai, ne anju kaa purnakalik coach banane ke liye 1998 main apane kariyar ko samaapt kar diya. vah ek pratishthit khel parivaar se taalluk rakhte hai or unke chote bhai jimmy jorge ek prasiddh wollyboul khilaadi hai. vishwa atheletics meet se pehle anju bobi ko ye ahsaas hua ki vishwa stareey pratiyogitaaon main mukaable ke liye antarrashtriya expoger hona bahut aavashyak he, isliye unhone vishwa record dharak mike powell ke saath prashikshan lenaa shuru kiya, jisane unhen takanic main moolyawaan expoger pradaan kiya.
 unhone 2004 ke athens olympic main bhaag liya lekin padak jeetne main asafal raheen. unhone 2008 ke beijing olympic main bhi bhaag liya lekin apane teenon prayaason main foul ke chalate vah mahila lambii kud pratiyogita ke liye kwalifai karne main asafal raheen.
 anju kaa vivaah robert bobi jorge se hua jo tripal jump main purv rashtriya champion or unke coach bhi rahe hai. anju custom vibhaag main karyrat he. vah bengalore main rahati hai or vah prashikshan bhi vahi se praapt karti hai. unki khel uplabdhiyon ko dekhte hue chennai customs house ne bina baari ke unako padonnati di. do baar kii olympian anju ko seemaa shulk vibhaag main adheekshak str par padonnati di gai..
