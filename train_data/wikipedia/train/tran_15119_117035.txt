 mishri yaa misari shakkar ke crystalon kaa ek rup he, jise bharat va pakistan main ek mishthann ke rup main prayog kiya jaataa he, athwa doodh ko meetha karne main prayog kiya jaataa he.
 bharat main ise hinduon dwaara devataaon ko bhoga lagane va prasad ke rup main baantne main bhi prayog kiya jaataa he. bhagwaan krishna ko makkhan va mishri priy maani jaati hai. brijabhoomi main likhe gayi kai bhakthigiton main "maakhan or mishri" shabd ek saath aate hai. mishri ko sattvik khaadya kii shrenee main rakhaa jaataa he. uttar bharat main hone vaale bhagwati-jagarnon main praayah aadhi raat kii vela main, upasthit logon ko mishri prasad ke rup main baanti jaati he, tab iske seven se sattviktaa kaa punah sanchaar ho jaataa he, tathaa neend nahin aati he. mishri ko sanskrit main sita, bengali main michari kaha jaataa he.
 bharat main mishri se kai mithaiyaan bhi banaai jaati hai, jaise- mishri-maava, mishri-peda, jinhen praayah uttar-pashchim bharat, uttar pradesh, dilli, rajasthan, gujarat, punjab aadi main khaaya jaataa he.
 mishri swaabhaavik rup se hi mithaas kaa upmaan he. kisi bhi mithi baat yaa saumy vyakti ko mishri kii dali kaa visheshan diya jaataa he. mithi vaani yaa swar bolne ko "kaanon main mishri gholna" kaha jaataa he. isi kaaran se yah uttar bharat main prachlit naam bhi he, jaise- mishrilaal, misaridevi aadi.
