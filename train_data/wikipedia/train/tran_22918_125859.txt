 font yaa tip fes aksharon ke likhane kii alag alag sheliyaan hai jo chaape kii likhaawat ko vibhinnata or saundarya pradaan karte hai.
 hindi font do prakaar ke hote hai:-
 ve font jo unicode maanak ke anusaar hote hai tathaa computer ke prachaalan tantra ke script ingine kaa prayog karke taxt pradarshit karte hai, unicode font kehalaate hai. vartmaan main computer evam internet par tamaam kaaryon main inhen kaa prayog hota he. unicode fonton ke prayog se computer par sabhi jagah hindi evam anya bhartiya bhashaayen vaise hi likhi ja sakati hai jaise angrezi likhi ja sakati he. unicode fonton ke kaaran hi computer kaa sampoorn oparating system hindi bhashaa main uplabdh ho chukaa he.
 kuch prachlit unicode hindi font hai:-
 windos 7 ke hindi font:-
 ye ve font hai jo puraane 8 bit maanakon ke anusaar bane hai. ye unicode ke aane se pehle prachlan main laaye jate the. vartmaan main computer ke saamaanya kaaryon evam internet par inkaa sthaan unicode fonton ne le liya he. vartmaan main ye keval ditipi evam graphics hetu prayog main laaye jate hai kyonki kai ditipi evam graphics software indic unicode kaa samarthan nahi karte. iske atirikt chapai hetu sundar unicode fonton kaa abhaav bhi ek kaaran he jisaki vajah se nonri-unicode font abhi bhi prayog main laaye jate hai.
 kuch prachlit non-unicode hindi font hai:-
 adhiktar non-unicode font remington leout aadhaarit hote hai. remington main tip karne hetu aam taur par kisi anya aujaar kii aavashyaktaa nahin hoti. hindi ke maanak keyboard leout inscript dwaara in fonton main tip karne hetu i-pandit ikhmai naamak aujaar uplabdh he.
 sanskrit hetu kai unicode font uplabdh hai jinamein omkaranand ashram kaa sanskrit 2003 naamak font sarvshereshth he. puraane jamaane kii style main likhi jane vaali sanskrit varnmaala hetu uttara naamak font he.
 sanskrit ke adhiktam varna/chihn chhandas naamak unicode font main hai. yah belarus ke Mihail Bayaryn dwaara viksit kiya gaya he. isamen un varna chihnon hetu bhi gliphs hai jo indic unicode main kutabaddh nahin hai jaise swastik aadi. haalaanki yah font sanskrit 2003 jitna sundar nahin lekin sarvaadhik chihn yukt hai. iske liye ek ikhmai bhi uplabdh he jisase atirikt jode gayi chihnon ko tip kiya ja saktaa he.
 a-unicode sanskrit fonton main omkaranand ashram ke sanskrit 1.2, sanskrit 98, sanskrit new, sanskrit 99 shaamil hai, inamen naveenatam sanskrit 99 he. puraane jamaane kii style main likhi jane vaali sanskrit varnmaala hetu Gudakesa 99 tathaa Santipur 99 naamak font hai. iske alaava kuch anya font xdvng aadi shaamil hai. parantu inamen se kisi main bhi sabhi sanskrit chihn shaamil nahin hai. baraa ke BRH-Devanagari naamak font main kuch atirikt sanskrit chihn shaamil hai.
 xdvng - isamen IS0 LATIN-8859-1 ankoding prayukt huyi he. yah sandeep sibbal ke JTRANS auditor hetu banaaya gaya tha jisamen hindi tathaa sanskrit dastavej banane kii suvidha thi.
 sanskrit ko roman main likhane hetu anterrashtriya sanskrit lipyantaran varnmaala kaa prayog kiya jaataa he. isckon ne iske liye kuch font banaaye hai.
 harek aakaar ke phont ke liye, harek style ke liye, alag alag phont set banane padate the. isse harddisk spes adhik lagata tha. yo saare phont kisi bhi Dot matrick printer yaa PCL lager printer par mudrit ho pate the.
 kintu aajkal Open Type font ke jamaane main unicode ke jamaane main postscript kaa prayog kam hota ja rahaa he.
