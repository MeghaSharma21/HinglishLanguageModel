 bal patrakarita kii sudeergh parampara ko ek aalekh main sametanaa nishchay hi anjali main samudr bhar lene ke samaan he. bal sahitya kii anek patrikas vigat pachaas varshon main prakashit hui hai. in prakaatrion kaa sahi-sahi vivaran de paanaa ek duruh kaary he. bal sahitya kii patrikaon main kuch patrikas to lambe samay se prakashit ho rahi hen, parantu adhiktar patrikas kaal-kavilat ho gayi. unke puraane ank khojna kathin hi nahin, asambhav-sa kaary he. phir bhi jin patrikaon kaa vivaran uplabdh ho sakaa he, unhen is aalekh main samahit kiya ja rahaa he.
 bal patrakarita kaa aupchaarik prarambh bharatendu kaal se mana ja saktaa he. 1882 main bharatendu harishchandr kii vishesh prerana evam abhisrchi se 'bal daparn' patrika kaa elahabaad se prakaashan hua. iske baad bharatendu harishchandr ne 'bala bodhini' patrika kaa prakaashan prarambh kiya. in patrikaon main naitik mulyon ko kendra main rakhkaar upadeshaatmak bal sahitya prakashit hua. in aarmabhik bal patrikaon ke baad nirantar bal sahitya kii patrikas prakashit hoti raheen. 1891 main lakhanu se 'bal hitkar' patrika prakashit hui. 1906 main aligarh se 'chaatr hitaishi' patrika prakashit hui. isi varsh 1906 main hi banaras se 'bal prabhakar' patrika kaa prakaashan hua. 1910 main elahabaad se 'vidhyaarthee' patrika prakashit hui. 1912 main 'monitor' patrika narsinghpur se prakashit hui.
 bal patrikarita ke kshetra main nae yug kaa prarambh 'shishu' patrika se hua. 1914-15 main 'shishu' patrika ke prakaashan prarambh hua. iske sampadan pan.sudarshanacharya the. 'shishu' patrika ke prakaashan ke kuch samay uparant ek or utkrisht patrika ne bal sahitya ke kshetra main padaarpan kiya. is patrika ne bal sahitya ke kshetra main kranti laa di. 1917 main bengaaliyon kii privet limited sanstha 'indian press' ne 'baalasakhaa' patrika kaa prakaashan prarambh kiya. iske sampadak pan.badrinath bhatta the. dvivedi yug kii yah patrika sarvaadhik lokapriy rahi. 'baalasakhaa' sabase adhik samay arthath 53 varsh tak prakashit hoti rahi. is patrika ne bal pathakon ko bahut prabhavit kiya tathaa bal sahityakaron kii acchhee-khaasi sankhya taiyaar kii. isi prakaashan prarambh kiya shrinkhalaa main 1920 main jabalpur se 'chaatr sahodar' kaa prakaashan prarambh hua. 1924 main dilli se madhav ji ke sampadan main 'vir balak' kaa prakaashan hua. patna se 1926 main acharya ramlochan sharan ke sampadan main 'balak' patrika kaa prakaashan prarambh hua. 1927 main pan.raamji laal sharma ke sampadan main elahabaad se 'khilonaa' patrika kaa prakaashan prarambh hua.
 ilahabaad se hi 1913 main 'chamacham' kaa prakaashan hua. iske sampadak the 'vishwa prakaash' 1931 main ek or patrika elahabaad se prakashit hui, jo aage chalakar bal sahitya ke kshetra main mile kaa pathar sabit hui. yah patrika thi pan.ramanaresh treepathi ke sampadakatv main prakashit 'vanar' patrika. pan.ramanaresh treepathi ke shreshth baalagit is patrika ke madhyam se hi bal pathakon tak pahunchakar lokapriy bane. 1932 main kalakankar se kunwar suresh singh se sampadan main 'kumaar' patrika kaa prakaashan hua. ilahabaad se 1934 main 'akshay bhaiya' kaa prakaashan hua. ramashankar jaitali ke sampadan main muradabad se 1936 main 'bal vinod' kaa prakaashan prarambh hua. 1938 main pan.ram dahin mishra se sampadan main patna se 'kishor' patrika kaa prakaashan hua. 1944 main prem narayan tandon se sampadan main lakhanu se 'honhaar' patrika kaa prakaashan hua. 1946 main vyathit hriday se sampadan main ilahabaad se 'titali' kaa prakaashan prarambh hua. 1946 main hi elahabaad se thakur shrinath singh ke sampadan main 'balbodh' patrika kaa prakaashan hua.
 swatantraaa prapti se pehle prakashit in patrikaon ne bal patrakarita main apana sthaan bana liya tha. patrikaon hi sankhya nirantar badhati gai. kuch band huin, to kuch nai patrikas bhi prakashit hoti raheen. in patrikaon main kahani, baalagit, naitik kahaniyaan or hasya kii kahaniyaan pramukhataa se prakashit hoti thim. lok jivan main prachlit bal geet, bal kathaayen aadi bhi in patrikaon main samay-samay par prakashit hoti raheen. swatantraaa prapti se purv tak bal sahitya ke prakaashan kii sthiti santoshjanak kahi ja sakati he. is samay tak bal sahitya kii lagbhag tees mudrit patrikas prakashit ho rahi thi tathaa solah hastlikhat patrikas prakashit hoti thim.
 1947 main desh swatantra hua. vyavastha parivartit hui. aise main sahitya ke kshetra main bhi ek mode aayaa. bal sahitya main desh prem, khushi ore ullas se labalab sahitya patrikaon main prakashit hone laga bharat sarkaar ke prakaashan vibhaag ne swatantraaa prapti se 'balbharat'kaa prakaashan prarambh hua. yah patrika adhyatan nirantar atyant lokapriy bani hui he. 1948 main punjab se 'prakaash' naamak patrika kaa prakaashan prarambh hua. 1949 main dilli se 'amar kahani' evam elahabaad se 'manmohan' patrikaon kaa prakaashan prarambh hua, parantu durbhaagya se kuch ank nikal kar ye donon patrikas kaal ke gaal main samaa gayi.
 hindi bal patrakarita ke kshetra main isi samay ek mahatvapurn ghatana hui, vah yah ki madras se 'chandaamama' kaa prakaashan prarambh hua. 1948 main ahindi bhashi kshetra se apana prakaashan prarambh karne vaali 'chandaamama' aaj tak nirantar prakashit hoti chali ai he. yah patrika samooche desh main atyant lokapriy he. patna se 'chunnu munnu' patrika kaa prakaashan shuru hua or yah bhi apane samay main lokapriy patrika bani rahi. 1951 main prolekharaj ulfat ke sampadan main dehradun se 'nanheen duniya' kaa prakaashan prarambh hua. lakhanu se es.em. samim anhonvi se sampadan main 'kaliyaan' patrika nikalni prarambh hui. 1955 main dilli se lakshmi chandra tii. rup chandani ke sampadan main 'bal mitra' kaa prakaashan prarambh hua. isi kalavadhi main 'vanar' patrika kaa prakaashan jaipur se prarambh hua.
 1957 main tasrnbhai ke sampadan main 'jivan shiksha' patrika kaa prakaashan prarambh hua. isi varsh dilli se 'swatantra balak'kaa prakaashan ytn prakashshil ke sampadan main prarambh hua. ye patrikas bal pathakon ke bich apni khaas pahchaan nahin bana sakin. 1958 main dilli se 'paraag' patrika kaa prakaashan prarambh hua. is patrika ne ati sheeghra bal pathakon evam bal sahitya ke pathakon ke bich apni pahchaan bana lee. lambe samay tak acchhee prasar sankhya hone ke baavjood bhi paraag kaa prakaashan band ho gaya.
 ludhiana se santraam ke sampadan main 1958 main 'shobhaa' patrika kaa prakaashan prarambh hua. 1958 main hi umesh malhotra ke sampadan main jalandhar city se 'nanhen-munne' patrika kaa prakaashan prarambh hua. varanasi se rajkumaar wahi ke sampadan main 1958 main 'raja betaa' kaa prakaashan shuru hua. isi varsh muradabad se shanti prasad dikshit ke sampadan main 'balbandhu' kaa prakaashan prarambh hua. 1959 main 'minu-tinu' patrika ramaakanth pandeya ke sampadan main chakradhar se nikali. dayashankar mishra 'dadda' ne dilli se 'raja bhaiya' patrika nikaali. amritsar se gusrcharan singh saakhi se sampadan main 'bal phulwari' kaa prakaashan hua.
 1960 main bhagavaandas datta se sampadan main karnal se 'bal jivan' kaa prakaashan hua. shrimati suman evam je.en. verma se sampadan main 'hamaara shishu' patrika nikaali. isi varsh bholanath pusrshottam saran ke sampadan main azamgarh se 'vishwa bal kalyaan' patrika kaa prakaashan hua. ye patrikas bal sahitya ke pathakon main adhik lokapriy nahin huinr. bal patrikaon ke prakaashan hi sankhya nirantar badhati rahi. saath-hi-saath anekanek patrikas band bhi hoti raheen. kuch patrikas to apane keval praveshank hi prakashit kar paayin.
 1960 ke uparant prakashit hui bal patrikaon main pramukh is prakaar rhain:
 
 bal-patrakarita ke kshetra main adhyatan prakashit hote rahakar apni pahchaan jin patrikaon ne banaai he, unamem pramukh rhain bal bharti, balahans, nadan, chapank, chandramama, bal yug,bal vaani, bal vatika, devputra, sneh, bal mitaan, bacchon kaa desh, jan samplav, hansati duniya, balmitra aadi. bal patrakarita ke kshetra main un saahitiyik patrikaon ke avdaan ko bhi andekha nahin kiya ja saktaa, jinhone bal sahitya ke visheshank prakashit kiye hai. samay-samay par prakashit vibhinn patrikaon ke visheshankon ne bal sahityakaron ko disha nirdesh to diya hi, saath hi unke dwaara gaye vigat kaal ke bal sahitya kaa samikshatmak moolyankan bhi kiya.
 in patrikaon ke bal sahitya visheshankon kaa yatha upalbadh vivaran is prakaar rhe:
 in visheshankon ke atirikt samay-samay par saahitiyik patrikas bal sahitya par kendrit lekhon ko prakashit karti rahati hai. 'kadambini', 'navaneet', 'harigandha', 'punjab saurabh', 'samakaleen bhartiya sahitya', 'madhumati', 'uttar pradesh' aajkal aadi patrikaon main prakashit lekhon main bal sahitya par vibhinn konon se vichaar kiya gaya he. 'saptahik hindustan' patrika apane samay main bal sahitya par acchhee saamagree prakashit karti rahi he.bal patrakarita ke kshetra main dainik samachar patron ke saptahik parishishton main bal-sahitya prakashit kar rahe hai. bal pathak in ankon kii pratiksha karte rahete hai. aise samachar patron rmen 'amar ujaalaa', 'aaj', 'dainik jaagran', 'rashtriya sahaara', 'jansatta', 'daily milaap', 'swantatra bharat', 'navabharat times', 'dainik navajyoti', 'rajasthan patrika', 'hindustan', 'dakshin samachar', 'nai duniya', aadi pramukh hai.
 bal sahitya kii vibhinn patr-patrikaon ke prakaashan varsh evam unke namollekh karne ke uparant vigat varshon main prakashit hote rahe bal sahitya ke pramukh kavyanshon ko uddhat karnaa prraasangik prathit ho rahaa he, kyonki samay-samay par prakashit bal sahitya ke in anshon se yah sahaj anumaan lagaana sambhav ho sakegaa ki bal sahityakaron ne balman kii samvednaaon ko kahan tak samajha he.
