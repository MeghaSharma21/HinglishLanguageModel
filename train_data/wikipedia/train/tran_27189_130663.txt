 dropbox ), dropbox ink dwaara sanchalit ek fil hosting seva he jo ki cloud sanchayan, fil tulykaalan or graahak software pradaan karata he. dropbox prayek upyogkartaaon ko apane computer par ek vishesh folder banane ki anumati detaa he, jo tulykaalan ke baad internet ke dwaara, saamagree ke saath, kisi bhi computer par upbhogta istmaal kar saktaa he. yahi folder upyogkarta apane mitron ke saath share bhi kar sakate he. dropbox ink, 2007 main emiti snaatakon driu huston or arsh firdausi dwaara sthapit kiya gaya tha. dropbox microsoft windos, appal mike oes eks, linaks, google android, appal ioes, blueberi oes or web brouzar ke liye graahak software pradaan karata he.
 
 dropbox sansthaapak drew huston ke anusaar, jab vah emiti main ek chaatr tha tab baar baar apane yuasabi flash drive bhul jaataa tha, tab use yah vichaar aayaa tha. usne khud ke liye kuch karnaa shuru kar diya, lekin phir use ehsaas hua ki isse dusron ko bhi phayda ho saktaa he. huston ne dropbox ink kii, june 2007 main sthapana kii or uske baad sheeghra hi vaai komabinetre naamak company se beej dhan surakshit kiya.12 navambar 2012 ko, dropbox ne ghoshnaa kii vah 100 lakh upyogkartaaon tak pahunch chukaa he.
 july 2011 main bade parisar main jane se pehle tak, dropbox kaa mukhyalay san francisco main 760 market street ke parisar main tha. us taarikh se dropbox corporate mukhyalay san francisco main china basin landing imaarat kii chauthi manzil par suite 400 main hai. company imaarat ke khand 1991 kii chauthi manzil main 85,600 varg feet or adhik jagah lene ke vikalp ke saath maujuud he.
 dropbox upyogkartaaon kii 32.7%, sabase bada hissaa, sanyukt rajya america se hai, united kingdom se 6.7% or germany se 6.5% he. dropbox upyogkartaaon kii 66.1% keval windos kaa upyog, 20.9% keval mike oes kaa upyog, 2.0% keval linaks upyog or shesh ek se adhik oparating system kaa upyog karte hai.
 dropbox ek freemium vyapaar model kaa upyog karata he, jahaan upyogkartaaon ko ek simit sanchayan kshamta ke saath ek mukt khate kii peshkash karata he. adhik kshamta vaale khaaton ke liye sadsyta kaa bhugtan aavashyak he.
 donon dropbox sarwar or desktop client software mukhya rup se paythan main likhe gayi he. dropbox client, upyogkartaaon ko ek nirdisht folder main kisi bhi file ko drop karne ke liye sakasham banaata he, phir vah folder internet ke jariye upyogkarta ke computer or upkarnon ke saath sincranaise kiya jaataa he. upyogkarta bhi ek web browz ke madhyam se swayan file aplod kar sakate hai. haalanki dropbox ek sanchayan seva ke rup main kaam karata he, is kaa mukhya dhyaan tulykaalan or sahbhaajan karne par kendrit he.
 yah sanshodhan itihaas kaa samarthan karata he, is vajah se, dropbox folder se nasht kar di ja chuki filein sinc kiye gaye computer ke jariye baraamad kii ja sakati hai. dropbox bahu upyogkarta sanskaran niyantran kaa samarthan karata he, yah sanskarnon ko bina adhilekith kiye, kai upyogkartaaon ko filein sampadit karne or phir se post karne ke liye sakasham banaata he. sanskaran itihaas delta ankoding praudyogiki ke upyog ke jariye rakhaa jaataa he. jab ek upyogkarta folder main ek fil badalata he, dropbox tulykaalan ke dauraan keval fil ke badale hue tukade hi aplod karata he. desktop client vyaktigat file ke aakaar par koi pratibandh nahin he, web site ke madhyam se aplod kii gai filon kaa adhiktam aakaar 300 MB prati fil tak simit he.
