 dhaatuon kaa sanksharan rasinic kriya he, jiske falswaroop dhaatuon kaa kshay evam hraas hota he. dhaatuon kii ksharankriya, jinamein yantric karkon ke falswaroop dhaatuon kaa hraas hota he, is kriya se bhinn hoti he. dhaatuon main sanksharan vastut: rasinic kriya, athwa vaidyutracinic kriya, ke rup main hota he. mul aadhaar ke anusaar uparyukt donon prakaar kii sanksharan kriyaayein mul kriya kii vibhinn avasthaaen hai. 
 dhaatuon kii sanksharan kriyapranali kii mukt urja main vishisht evam aavashyak rup main nunata utpann hoti he. pratyaksh rasinic kriya dwaara dhaatuon ke sanksharan main gas, athwa ardratayukta vaataavaran, kaa sansarg sanksharan ke liye upyukt paristhityaan utpann karata he. sanksharan kii vidyutracinic kriya main, dhaatuon ke drav main nimajjit hone se, vidyutdhara utpann hone kii upyukt paristhitiyaan utpann ho jaati hai. is prakaar sanksharan kriya main dhaatuon kaa vidyutracinic hraas hota he. unamem tathaa dravon main nimajjit hone se dhaatuon kii sanksharan keval uparyukt paristhitiyon par nirbhar nahin karti, anya karkon kaa bhi vishesh evam mahatvapurn prabhaav padataa he. 
 saamanyat: dhaatuon kii sanksharan kriya main nirmit honevaala antim utpaada aisa yogic hota he jo prakruti main khanij padaarth ke rup main paaya jaataa he. udaahranarth, taanbe ke patt ko bahut varshon tak aantarasthaleey vaataavaran main, khuli avasthaa main rakhane se patt ke upree tal par ksharak sulphate kii ek parat germ jaati he. taanbe kaa yah ksharak sulphate prakruti main paae janevaale khanij brocatite jaisaa hota he. isi prakaar lohe athwa ispaat ke patt ko lavniya jal main purnat: nimajjit rakhane par varsha main uske tal par jalyojit loh oxide kii kathor parat jam jaati he. jalyojit ferick oxide prakruti main paae janevaale khanij gothit jaisaa hota he. is prakaar dhaatuon kii sanksharan kriya dhaatuon ke madhyasthayee dhatvik avasthaa main sthaayee oxikrut avasthaa main prayaavartan kii kriya he. jo dhaatuen prakruti main apane shuddh rup main pai jaati hai, jaise swarna, unamem saamanyat: prakruti main upasthit karkon dwaara sanksharan kriya nahin hoti or iske falswaroop hi aisi dhaatuen asamyukt avasthaa main pai jaati hai.
 sanrachanaatmak aadhaar par sanksharan kriya nimnaankit vibhinn swaroopon main dhaatuon par prabhaav utpann karti he :
 dhaatuon ke sanksharan nivaaran main saidhantik rup main un sabhi upayon evam karkon ko chhod dena padataa he jo sthaayee avasthaa ke prayaavartan ko protsaahit karte hai. is prakaar ke kaarak vibhinn dhaatuon ke liye bhinn bhinn hote hai, parantu saamaanya rup main oxigen tathaa oxigen mishrit vilyan evam jal main viley padaarth sthaayee avasthaa ke prayaavartan ko protsaahit karte hai. sanksharan ke nivaaran main uparyukt karkon kaa purna bahishkaar praaia: asambhav hota he. at: inaki upasthiti main swayansthirak kriyaaon kaa sahaara liya jaataa he. dhaatusanksharan kii vishesh paristhitiyon main sanksharan kii gati par adhiktam avrodh utpann karanevaale kaarak ko sanksharan kaa niyantrak kaarak kaha jaataa he. audyogic drushti se mahatvapurn praaia: sabhi dhaatuon ke bahya tal par vaataavaran main khule rahane se dikhaai denevaali, athwa na dikhaai denevaali, sookshm parat jam jaati he, jo anuvarthi sanksharan prakriyaayon ko prabhavit karti he. saamanyat: yah parat khuli dhaatu ke oxide se nirmit hoti he. iske gun mul dhaatu ke gunon se bhinn hote hai tathaa khule rahane kii paristhitiyon se bhi vyavahaarabhinnata utpann hoti he. adhikansh paristhitiyon main oxide kii yah parat muldhatu ke sanksharan kaa nivaaran karti he. is prakaar kii moti parat jab prasaaran evam sankuchan ke kaaran kahin kahin se tut jaati he, tab in sthanon par vidyut-rasinic-sanksharan prarambh ho jaataa he. dhaatuon ke sanksharan ko nimnaankit chah bhaagon main vibhakt kiya jaataa he : 
 uparyukt prayek dasha ke sanksharan prakram main visheshta hoti he or iske nivaaran ke liye bhinn bhinn upaay kiye jate hai.
 dhaatuon ke sanksharan nivaaran main vibhinn ritiyon kaa upyog hota he, jinamein nimnaankit pramukh hai : 
 sanksharan nivaaran kii antim reeti sarvaadhik upyog main aati he. is reeti main dhaatu ke svachh bahya tal par oxide jaise prakritik nivarak adhilep kaa sambalan athwa uske sadrush prabhavotpadak kritrim adhilep padaarth kaa upyog kiya jaataa he. adhilep dwaara nivaaran kii is reeti main rangalep evam isi prakaar ke anya lepon kaa upyog hota he, jinamein ksharak chromet padaarth, litharge, rakt sis, laal loh oxide, graphite, vitumini padaarth aadi pramukh hai. sanksharan nivaaran main galvanikaran, dhatukaneekaran, cement karan, vidyut nikshipt adhilep tathaa anya dhaatu adhilep kaa adhikadhik upyog hone laga he.
