 malta fever ek atyant sankramak rog he, jo brusela jati ke jeevanu dwaara utpann hota he. ise medicrenian fever, brucilolis, yaa andulent fever bhi kehete hai. yah ek pashujanarog he. manushyon main paaltu jaanwaron, jaise maveshi kute yaa boar aadi, dwaara iskaa sanchaaran hota he. in sankramit pashuon kaa doodh pine, maance khaane yaa inke sraavon ke sampark main aane se iskaa sankraman ho saktaa he. rog kii tivravastha main fever, pasinaa, susti tathaa sharir main dard rahata he or kabhi-kabhi yah maheenon tak jeern rup main chalata rahata he. rog dwaara mrityu kii sankhya adhik nahin he, kintu rog sheeghra dur nahin hota. rhite ne san 1897 main brucelosis rog ke samoohan pareekshan kaa varnan kiya.
 brusela kii teen kismein gyaat hai, jo jaanwaron kii teen jatiyon main pai jaati he: bakrii main brusela melitansis, boar main brusela sooyee tathaa maveshi main bru0 abartus . sankraman jaanwaron ke doodh pine se manushya main rog kaa sanchaar hota he. udbhavan kaal 5 se 21 din he. kabhi kabhi rog ke lakshan pratyaksh hone main 6 se 9 maah tak lag jate he. ugr rup main fever, thand ke saath canpakanpi tathaa pasinaa hota he. jeern rup main dhire-dhire lakshan prakat hote hai. is rog ke tathaa influenza, malaria, tapedic, motitora aadi rogon ke lakshan aapas main milane ke kaaran vishesh samoohan pariksha tathaa twachaa main tikaa pareekshan se rog nidaan hota he.
 chikitsa main uchit paricharya tathaa salphonamide, streptomicine aadi kaa prayog hota he. rog pratishedh ke liye passchurikrut doodh ko kaam main laana chahie.
 jeevanu janit is rog main gopashuon tathaa bhainson main garbhavastha ke antim traimas main garbhpaat ho jaataa he. jaisaa upar bataaya gaya he, yah rog pashuon se manushyon main bhi aa sakaata he. manushyon main yah utaar-chadhaav vaala bukhaar naamak beemaari paidaa karata he. pashuon main garbhpaat se pehle yoni se apaardarshee padaarth nikalataa he tathaa garbhpaat ke baad pashu kii jer ruk jaati he. iske atirikt yah jodon main aarthrytis paidaa ke saktaa he.
 ab tak is rog kaa koi prabhavakaari ilaaj nahin hai. isaki rokthaam ke liye bachchhiyon main 3-6 maah kii aayu main brushella-aborts strain-19 ke tike lagaae ja sakate hai. pashuon main prajnan kii kritrim garbhaadhan paddati apanaakar bhi is rog se bachaa ja saktaa he.
