 sajiv koshikaon main bhojan ke oxikaran ke falswaroop urja utpann hone kii kriya ko koshikeey shwasan kehete hai. yah ek ketabolic kriya he jo oxygen kii upasthiti yaa anupasthiti donon hi avasthaaon main sampann ho sakati he. is kriya ke dauraan mukt hone vaali urja ko atipi naamak jaev anu main sangrahit karke rakh liya jaataa he jiskaa upyog sajiv apni vibhinn jaivik kriyaaon main karte hai. yah jaev-rasinic kriya paudhon evam jantuon donon kii hi koshikaon main din-raat har samay hoti rahati he. koshikaaye bhojya padaarth ke rup main glucose, amino amla tathaa vasiy amla kaa prayog karti hai jinako oxykrut karne ke liye oxygen kaa parmaanu electron grahan karne kaa kaary karata he.
 koshikeey shwasan evam shwaas kriya main abhinn sambandh he evam ye donon kriyaayein ek-duusare kii poorak hai. shwans kriya sajiv ke shwasan angon evam unke vaataavaran ke bich hoti he. iske dauraan sajiv evam unke vaataavaran ke bich oxygen evam carbon dioxide gas kaa aadan-pradaan hota he tathaa is kriya dwaara oxygen gas vaataavaran se sajivon ke shwasan angon main pahunchati he. oxygen gas shwasan angon se visaran dwaara rakt main pravesh kar jaati he. rakt parivahan kaa madhyam he jo is oxygen ko sharir ke vibhinn bhaagon kii koshikaon main pahunchaa detaa he. vahaan iskaa upyog koshikaaye apane koshikeey shwasan main karti hai.
 shwasan kii kriya prayek jivit koshika ke koshika dravya evam mitocandria main sampann hoti he. shwasan sambandhit praarambhik kriyaayein cytoplasm main hoti he tathaa shesh kriyaayein mitocandriaon main hoti hai. chunki kriya ke antim charan main hi adhikansh urja utpann hoti hai. isliye mitocandria ko koshika kaa shwasanang yaa shakti-grih kaha jaataa he.

 shwasan kriya ke creb chakra ke dauraan pyruvik amla oxeloacetic amla se kriya karata he. isamen teen sah-anzim kaam aate hai. sarvapratham piruvik amla injim kii upasthiti main oxykrut hokar carbon dioxide tathaa hydrogen main badal jaataa he. hydrogen dydrogen grahi enedy ke saath sanyukt hokar enediech tu banaata he tathaa carbon dioxide gas vayumandal main mukt ho jaati he. enediech tu main upasthit hydrogen kai shwasan injimon kii usthiti main oxygen se milakar jal main badal jaataa he tathaa urja mukt hoti he. yah urja edipi se milakar atipi ke rup main sanchit ho jaati he. ek anu glucose ke purna rup se oxikaran ke falswaroop 38 anu atipi kaa nirmaan hota he. atipi urja kaa bhandar he jise urja kii mudra bhi kehete hai. atipi main sanchit urja jeevon ke aavashyaktaanusaar vighatit hokar urja mukt hoti he jisamen jeevon kii vibhinn jaivik kriyaayein sanchalit hoti he.
 anaksic shwasan main oxygen kii aavashyaktaa nahin padati he tathaa yah oxygen kii anupasthiti main hota he. is kriya main bhojan kaa apurna oxikaran hota he. jantuon main is kriya ke falswaroop carbon daai-oxide tathaa lactic amla kaa nirmaan hota he tathaa paudhon main carbon daai-oxide tathaa ithyle alcohol banataa he evam bahut kam maatra main urja mukt hoti he. is prakaar kaa shwasan kuch nimna shrenee ke paudhon, yeast, jeevanu, evam antah paraseete jantuon jaise golkrimi, feetakrimi, monocystis ityadi main hota he. kuch vishesh paristhitiyon main ucch shrenee ke jantuon, paudhon ke utakon, bijon, rasadaar falon aadi main bhi oxygen kii anupasthiti main hota he. manushyon yaa ucch shrenee ke jeevon kii mansapeshiyon ke thakne kii avasthaa main yah shwasan hota he. jin jeevadhariyon main yah shwasan hota he use anerobs kehete hai.

 kinvan anaksi shwasan kii ek vishisht rasinic kriya he. isamen sookshm jeevon tathaa injimon kii upasthiti main jatil carbonic yogicon kaa vighatan saral yogicon ke rup main hota he. kinvan kii kriya khamir, lactobasilas tathaa ankurit bijon aadi main hoti he. kinvan kii kriya ko mukhyatah do bhaagon main baanta ja saktaa he- alcoholic kinvan tathaa amleey kinnav. alcoholic kinvan kii kriya main glucose kaa vighatan ithyle alcohol tathaa carbon daai-oxide main hota he tathaa kuch urja mukt hoti he. yah east tathaa ankurit beej aadi main hoti he.

 prakaash-sanshleshan kii kriya main paudhe prakaash kii upasthiti main vaataavaran se carbon daai-oxide ko grahan karte hai tathaa oxigen gas vaataavaran main mukt karte hai. jisase vaataavaran main oxigen kii sandrata kaa bad janaa tathaa carbon daai-oxide kaa kam hona swaabhaavik hota he, parantu vastav main aisa nahin hota he. shwasan kii kriya main sajiv din or raat har samay oxigen gas grahan karte hai tathaa carbon daai-oxide gas kaa vaataavaran main tyaag karte hai. is prakaar shwasan kriya dwaara vaataavaran main oxigen evam carbon daai-oxide gaison kaa santulan bana rahata he.
 shwasan ek urja-unmochan prakriya he. shwasan kii kriya main bhojya padaarthon kaa oxikaran hota he, jisase unamem sanchit urja mukt hoti he. shwasan ke samay mukt urja kaa kuch bhaag koshika ke mitocondria main atipi ke rup main sanchit ho jaati he. atipi ke rup main sanchit yah urja bhavishya main sajiv jeevadhariyon ke vibhinn jaivik kriyaayon ke sanchaalan main prayukt hoti he. mukt honi vaali ooshmiya urja sajivon ke sharir ke tapakram ko santulit banaae rakhane main sahaayta karti he. kuch kiton tathaa samudri jantuon ke sharir se prakaash utpann hota he. yah prakaash shwasan dwaara utpann urja kaa hi ek roopaantarit rup he. kuch samudri jeevon jaise- electric-re machhlee tathaa tarpeado aadi kaa sharir aatm-raksha ke liye vidyutiya tarange utpann karata he, yah vidyut shwasan dwaara utpann urja ke roopaantaran se utpann hota he.
 prakaash-sanshleshan evam shwasan kii kriyaayein ek duusare kii poorak evam vipareet hoti hai. prakaash-sanshleshan main carbon daai-oxide or jal ke bich rasinic kriya ke falswaroop glucose kaa nirmaan hota he tathaa oxigen mukt hoti he. shwasan main iske vipareet glucose ke oxikaran ke falswaroop jal tathaa carbon daai-oxide banati hai. prakaash-sanshleshan ek rachanaatmak kriya he iske falswaroop sajiv ke shushk bhaar main vruddhi hoti he. shwasan ek nasaatmak kriya he, is kriya ke falswaroop sajiv ke shushk bhaar main kami aati he. prakaash-sanshleshan main saurya urja ke prayog se bhojan banataa he, vikiran urja kaa roopaantaran rasinic urja main hota he. jabaki shwasan main bhojan ke oxikaran se urja mukt hoti he, bhojan main sanchit rasinic urja kaa prayog sajiv apane vibhinn kaaryon main karata he. is prakaar ye donon kriyaayein apane kachche maal ke liye ek duusare ke ant padaarthon par nirbhar rahete hue ek duusare kii poorak hoti hai.
 sajivon ke sharir main prayek koshika main jaev-rasinic kriyaayein hoti rahati hai. in kriyaaon ke sampaadan hetu dainik urja kii aavashyaktaa hoti he. urja kaa mukhya strot bhojan he. saura urja bhojan main sthitij urja ke rup main sanchit rahati he. shwasan kriya main shwasan kaa mukhya padaarth glucose kaa oxigen kii upasthiti yaa anupasthiti main purna yaa apurna oxikaran hota he, jiske falswaroop bhojan main sanchit sthitij urja gatij urja ke rup main mukt hota he, jisase jivan kii dainik kriyaaon ke sampaadan hetu aavashyak urja kaa unmochan tathaa purti hoti he. shwasan main oxikaran ke liye aavashyak glucose, poshan ke madhya hi nirmit hota he. poshan ke liye aavashyak urja, shwasan kriya main utpann urja se praapt hoti he. is prakaar shwasan tathaa poshan ek-duusare se abhinn rup se sambandhit he.
