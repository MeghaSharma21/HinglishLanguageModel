 shri mukut bihari verma ji ,kaa janm 28 disambar 1945 ko bharat-nepal seemaa par sthit gram madariyaa, vikaas khand-mihipurava, janapad-baharich u0pra0 ke ek kisaan parivaar main hua.	apane paitruk gram madariyaa se 03 kilometre dur sthit ayodhya gram main sthit praathmik paathshaala se praarambhik shiksha grahan kiya tatpashchaat varsh 1971 main lakhanu vishvavidhyalay se el0el0bi0 paas karke baharich main vakaalat prarambh kiya. varsh 1962 main hiskool main hi rashtriya swayan sevak sangh ke sampark main aae. baharich va lakhanu main rashtriya swayan sevak sangh ke anek dayitvon ke saath hi sangh ke nagar kaarywah kaa daayitv nirvahan kiya. varsh 1975 main apatkaal ke samay bhumigat rahakar baharich main loktantr raksha aandolan ke sahabhaagi rahe. mukademen main faraar rahe police ne ghar-makaan kurk kiya, lekin giraftaar nahin kar saki. varsh 1977 main mukadmaa waapas hua isi varsh jila kaarywah kaa bhi daayitv nirvahan kiya. varsh 1978 main edijisi ke pad par niyukti huyi. varsh 1985 main bhaajapa ke jila mahaamantri tathaa varsh 1990 main jiladhyaksh kaa daayitv saunpa gaya. varsh 1990 main kisaan mandi samiti ke adhyaksh kaa daayitv rahaa. varsh 1994 main tatkaaleen avagh kshetra jo shravasti se jhonsi tak tha, main sadsyta pramukh kaa daayitv rahaa. varsh 1994 main pravaas ke samay lakhanu jile ke gram bani main gambhir accident hua. varsh 1996 main party ne kaisarganj vidhaan sabha se pratyaashee banaaya lekin 234 maton ke antar se chunaav haar gayi. is interval main party main vibhaag sanyojak/jila prabharee tathaa varsh 2009 main punah jiladhyaksh ke pad kaa bhi daayitv saunpa gaya. 	varsh 2002, 2012 evam 2017 main kaisarganj vidhaan sabha se sadsy vidhaan sabha ke rup main chune gayi. varsh 2013 se lagaataar duusari baar avadh kshetra ke adhyaksh tathaa vartmaan main pradesh sarkaar main sahakaarita vibhaag ke mantri kaa daayitv saunpa gaya he.
