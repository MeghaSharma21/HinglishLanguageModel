 dach bhashaa netherland desh kii mukhyabhasha or rajbhasha he. yah hind-europea bhashaa-parivaar kii jarmaini shakha main aati he. kyonki ye ek nimna germanik bhashaa he, isaliye ye angrezi se kaafii mail khaati he. isaki lipi roman lipi he.
 netherland ke atirikt yah belgium ke uttari aadhe bhaag main, france ke nard jile ke upree hisse main tathaa europe ke baahar dach neugini aadi kshetron main boli jaati he. sanyukt rajya america tathaa canada main rahanevaale dach nagarikon kii bhi yah maatrubhaasha he. dakshin africa union rajya main bhi bahut se dach mul ke naagarik rahete hai or unki bhashaa bhi dach bhashaa se bahulansh main milti-julati he, yadhyapi ab vah ek swatantra bhashaa ke rup main viksit ho gai he.
 prarambh main hollandwalon kii bhashaa ko uttar samudr tathaa baltic samudr ke tat par rahanevaale germanon kii sthaneey boliyon main ek swatantra sthaan praapt tha. pehle yah mukhya rup se pashchimi flanders main prachlit thi kintu 16veen shatee main dach samskruti ke saath-saath iskaa prachaar bhi uttar kii ore badhata gaya. spani aadhipaty se mukt ho jane ke baad holland kii unnati kshipragati se hone lagi jisase dach bhashaa kaa vikaas bhi teji se hone laga. spani satta ke adheen bache hue dakshini praanton se bhagkar aanevaale sharnaarthiyon se bhi isamen sahaayta mili. dach bhashaa main dakshin kaa prabhaav aaj bhi spasht dekh padataa he. holland kii bol-chaal kii tathaa saahitiyik bhashaa main aaj bhi kaaphi antar dekh padataa he, yadhyapi dakshin se aae hue shabdon ke kaaran yah khai adhik gahari nahin hone pai.
 dakshin ke kai bhaagon main, jo ab belgium main shaamil he, aaj kii kai boliyaan prachlit hai. in sabakaa samuhik naam "flemish" he. yadhyapi skulon main saahitiyik bhashaa bhi padhai jaati he, tathaapi saamaanya log praaia: sthaneey boliyon main hi paraspar baatcheet karte hai. brucells main bahut se shikshit logon kii bhashaa aaj bhi french bani hui he kintu "flemish" kaa praadhanya badhata ja rahaa he. kanoon kii drushti se french tathaa flemish donon ko samaan rup se manyata praapt he. jahaan tak vartmaan holland rajya kaa prashn he, bhashaa kii drushti se sthiti utani jatil nahin he. 16veen tathaa 17veen shatiyon main jo nae-nae pradesh holland main shaamil hote gaye, unamem bhi kritrim rup se dach bhashaa kaa prasar hota gaya. kuch sthaneey boliyaan bhi prachlit hai, jinamein sabase pruthak astitv "frigian" kaa he jo freezland pradesh main boli jaati he. amstardam se jaise-jaise ham purv kii ore chalate hai, in boliyon main purvi prabhaav adhik lakshith hone lagata he, jisase samipsth german kshetron kii saamaanya german boliyon se unka saadrishy spasht ho jaataa he.
