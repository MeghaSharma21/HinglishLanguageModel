 forencik data vishleshan digital forencik kii ek shakha he. yah vittiya aparaadh kii ghatnaon ke sambandh main sanrachit data parakh hoti he. forencik data vishleshan kaa uddeshy kii khoj or dhokhadhadi gatividhiyon ke pattern kaa vishleshan karne ke liye hai.
 data kii badi maatra kaa vishleshan aam taur par vishleshan team dwaara chalaae ja rahe ek alag databes system main kiya jaataa he. live pranaaliyon aamtaur par niyamit rup se upyogkartaaon ko prabhavit kiye bina vyapak alag-alag vishleshan chalaane ke liye aayam nahin kar rahe hai.duusari ore, yah prakriya se alag system par data kaa vishleshan pratiyaan or mul data main pherbadal ke aarope ke khilaaf vishleshan tiimon kii raksha ke liye behatar he.
 data kii prakruti ke kaaran, vishleshan database main nihit he par kii tulana main data kii saamagree ke bare main adhik aksar kendrit he. agar database main hi byaaj kii he to databes forencik lagu kar rahe hai.
 aadesh main vittiya aparaadh se yah team main visheshgyata ke liye kam se kam teen prakaar letaa he kaa pataa lagane ke iraade se bade sanrachit data set kaa vishleshan karne main:ek data vishleshak takneekee charnon kaa pradarshan or prashnon, prakriyaaon or jaanch kii company ke sambandhit kshetra or ek nyaayalayik vaigyanik, jo dhokhadhadi vyavhaar ke pattern se parichit he main aantarik niyantran kii vyapak anubhav ke saath ek team ke sadsy likhane ke liye.
