 dhruv tara, jiskaa bair naam "alfaa ursaae minoris" he, dhruvamatsay taramandal kaa sab se roshan tara he. yah prithvi se dikhne vaale taron main se 45waan sab se roshan tara bhi he. yah prithvi se lagbhag 434 prakaash varsh kii duuri par he. haalanki kii prithvi se yah ek tara lagata he, yah vastav main ek bahu tara mandal he, jiskaa mukhya tara F7 shrenee kaa roshan daanav tara yaa mahadaanav tara he. vartmaan yug main dhruv tara khagoleey gole ke uttari dhruv ke nitak sthit he, yani duniya main adhiktar jagahon se dhruv tara prithvi ke uttari dhruv ke upar sthit prathit hota he. is kaaran se taron se maargdarshan lete hue samudr yaa registan jaisi jagahon se nikalne vaale yatri aksar dhruv taare kaa prayog karte hai.
 prithvi ke ghurnan se ratri main akaash ke lagbhag sabhi taare dhire-dhire ghumte hue lagate hai, lekin dhruv tara uttar kii ore sthir lagata he. agar kisi camere kaa lens lambe arase tak khulaa rakh kar raat main aasmaan kaa chitra khinchaa jaae, to tasviir main aisa prathit hota he kii saare taare dhruv ke ird-gird gum rahe hai.
 dhruv taare ko angrezi main "polaris", "polster", "norg star" yaa "lodustar" kehete he. arabi main ise "nazoom al-shumaal" kehete he, jiskaa arth he "uttar kaa tara". farsi main ise "qutabi sitara" kehete hai.
 dhruv mandal ke taare is prakaar hai -
 prithvi kaa aksh ek dhimi lekin nirdhaarit gati se dagamagaataa he, jis vajah se koi bhi tara sthir rup se uske dhruv ke upar sthit nahin rahata. dhruv tara is samay lagbhag uttari dhruv ke upar he lekin yah hazaron saal ke kaal main badal jayega. prithvi ke is aksh ke rujhaan main badlaav aisa he ki har 26,000 saal main uskaa ek puraa chakra puraa ho jaataa he. iskaa arth he ki haalanki dhruv tara prithvi ke dhruv se hatataa hua prathit hoga, lekin aaj se 26,000 varshon baad dhruv tara phir se dhruv ke lagbhag upar hoga. is samay yah aksh aynansh dhruv taare ko uttari dhruv ke or bhi sameep laa rahaa he. san 2100 main yah apni sab se sameepi sthiti main hoga or uske baad dhruv se hatane lagegaa.
