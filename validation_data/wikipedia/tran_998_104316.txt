 vigyaan, hindi main vaigyanik vishyon par nikalne vaali masik patrika he. iskaa prakaashan san 1915 ke april maas se armbh hua tha. is patrika kaa prakaashan vigyaan parishad prayag dwaara kiya jaataa he. iske pratham prakaashak lala karmchand bhalla evam sampadak lala seetaram evam pandit shridhar pathak the. pichley lagbhag sau varshon se iskaa anwarat prakaashan hota aa rahaa he. armbh main iskaa shulk 3 rupaye vaarshik tha. brijraj, dow satyaprakash, yudhisthir bhargav prabhruti sampadakon ne iskaa safaltaapurvak sampaadan kiya. samprati dow shivaprakash mishra iske sampadak hai.
 vigyaan parishad prayag ke sansthaapakon main se ek ’ dow ganganath jha ‘ ji kaa janm 15 sitambar 1872 ko darbhanga main hua tha. aapane elahabaad vishvavidhyalay se sanskrit main em0 e0 tathaa di0 lit kii oopaadhi praapt kii. 1920 tak aap myor centrral college prayag main sanskrit ke vibhaagadhayaksh rahe. 1920 se 1923 tak queens oriental sanskrit kalege banaras ke pratham bhartiya principal ke pad par rahe. aapane 1923 se 1932 tak elahabaad vishvavidhyalay ke kulapati ke rup main kaary kiya. aap prantiya legislative council ke manonith sadsy bhi rahe.
 aapane sanskrit, hindi, maithili tathaa angreji bhashaaon main 20 se adhik moulik granthon kii rachana kii. iske atirikt aapane sanskrit ke 19 granthon kaa hindi anuwaad tathaa 11 granthon kaa sampadan bhi kiya. aapaki vidwata ke liye aapako mahamahopadhyay, vidyasagar or el0 el0 di0 kii maanad oopaadhiyon se vibhooshit kiya gaya.
 aap 1927 se 1930 tak vigyaan parishad prayag ke sabhapati rahe. aap hindi sahitya sammelan ke bhi sabhapati rahe. 1941 main aapkaa nidhan ho gaya.
 vigyaan parishad ke sansthaapakon main se ek ’pro saligram bhargav‘ kaa janm 12 disambar 1888 ko gudgaanv jile ke khori naamak sthaan par hua tha. 12 varsh kii alpaayu main hi aapke pita munshi lachchhi ram ji bhargav kaa dehavsaan ho gaya. aapane san 1905 main alvar hai skool se metric kii pariksha pratham shrenee main uteern kii. 1909 main agra cholez se bi0 es0 si0 tathaa 1912 main myor central cholez elahabaad se bhautik vigyaan main em0 es0 si0 kii oopaadhi praapt karne ke baad aap myor central cholez main demastrator pad par niyukt hue. 1919 main bhautik vigyaan ke sahaayak adhyapak niyukt hue or tatkaaleen sanyukt praant kii prantiya ajukational sarvice main bhi rahe. prayag vishvavidhyalay kaa punarsanghathan hone par aap bhautik vigyaan vibhaag main reader bane tathaa 1946 se 1949 tak vibhaagadhayaksh bhi rahe. 1 may 1949 ko aapane avkaash grahan kiya.
 aap abhi 24 varsh ke hi the jab aapane dow ganganath jha, shri ramdaas goud tathaa pro hamiduddin ke saath milakar 10 march 1913 ko vigyaan parishad prayag kii sthapana kii. aapane aaajeevan hindi bhashaa main vigyaan ke prachaar- prasar evam lekhan main apana yogadan diya. aapane ’vigyaan praveshika‘, ’chumbak‘ aadi pustakon kii rachana kii. april 1915 main aapane hindi kii sarvapratham masik vaigyanik patrika ’vigyaan‘ kaa prakaashan armbh karavaaya. jan saadhaaran main vigyaan ke prachaar ke liye aapane anek avasaron par suprasiddh vaigyanikon ke dwaara vaigyanik vishyon par bhaashan yojit karwaae tathaa swayan bhi vyaakhyaan die. vigyaan parishad prayag ke mantri, pradhanmantri tathaa upasabhapati ke rup main aap aaajeevan isse jude rahe. 16 sitambar 1953 ko aapkaa swargvaas ho gaya.
 vigyaan parishad ke sansthaapakon main se ek ’ shri ramdaas goud ‘ kaa janm san 1881 main jonpur main hua tha. aapaki praranmbhik shiksha jonpur va varanasi main hui. 1907 main myor central cholez se aapane bi0 e0 kii padhai kii. aap central hindu cholez main rasayan ke sahaayak adhyapak rahe. 1908 se 1910 tak kysth paathshaala prayag main rasayan ke adhyapak rahe. rasayan se em0 es0 si0 karke aap prayag vishvavidhyalay main rasayan ke demastrator bhi rahe. 1918 main aap kaashi hindu vishvavidhyalay main professor rahe parantu 1921 main asahyog aandolan ke dauraan naukari chhod di. aap congress kii prantiya samiti ke sadsy tathaa hindi sahitya sammelan kii sthaayee samiti ke bhi sadsy rahe. dedh varsh tak aap jail main bhi rahe. aapane 23 pustaken likhin tathaa 200 se adhik lekh likhe. aapke ’ vigyaan hastamalak ‘ ko mangala prasad puraskaar 1936 main pradaan kiya gaya. 1913 main aapane vigyaan parishad kii sthapana kii. 
 1933-1937 tak aap ’vigyaan‘ patrika ke sampadak rahe tathaa 1915-1916 tak parishad ke pradhanmantri rahe. aap ’living encychlopedia‘ ke naam se prasiddh the. 13 sitambar 1938 ko aapkaa nidhan ho gaya.
