 sar shanti swaroop bhatnagar, OBE, FRS jane maane bhartiya vaigyanik the. inkaa janm shahpur main hua tha. inke pita parameshwari sahay bhatnagar kii mrityu tab ho gayi thi, jab ye keval aath maheene ke hi the. inkaa bachapan apane nanihaal main hi beetaa. inke naanaa ek ejeeniyar the, jinase inhen vigyaan or abhiyantriki main ruchi jaagi. inhen yantric khilone, electranik batariyaan or tarayukt telephone banane kaa shauk rahaa. inhen apane nanihaal se kavita kaa shauk bhi mila or inkaa urdu ekanki karamaati pratiyogita main pratham sthaan paaya tha.
 bharat main snatakottar degree purna karne ke uparaant, shodh fellowship par, ye england gayi. inhone university kalege, landan se 1921 main, rasayan shaastra ke profaisar fraderick ji donnan kii dekh rekh main, vigyaan main doctorat kii oopaadhi praapt kii. bharat lautne ke baad, unhen banaras hindu vishvavidhyalay se profaisar pad hetu aamantran mila. san 1941 main british sarkaar dwaara inaki shodh ke liye, inhen nisehud se sammanit kiya gaya. 18 march 1943 ko inhen felo aaf royal sosayadi chunaa gaya. inke shodh vishay main emlzan, kolayds or audyogic rasayan shaastra the. parantu inke mul yogadan chumbakeey-rasiniki ke kshetra main the. inhone chumbakatv ko rasinic kriyaaon ko adhik jaanane ke liye aujaar ke rup main prayog kiya tha. inhone pro aar.en.maathur ke saath bhatnagar-maathur interfrence santulan kaa pratipaadan kiya tha, jise baad main ek british company dwaara utpaadan main prayog bhi kiya gaya. inhone ek sundar kulgit naamak vishvavidhyalay geet kii rachana bhi kii thi. iskaa prayog vishvavidhyalay main kaaryakramon ke pehle hota aayaa he.
 bharat ke pradhan mantri jawahar laal nehru vaigyanik prasar ke prabal samarthak the. 1947 main, bhartiya swatantraaa ke uparaant, vaigyanik evam audyogic anusandhan parishad kii sthapana, shri bhatnagar kii adhyakshata main kii gayi. inhen si.es.ai.aar kaa pratham maha-nideshak banaaya gaya. inhen shodh prayogshaalaon kaa janak kaha jaataa he va bharat main anekon badi rasinic prayogshaalaon ke sthaapan hetu samaran kiya jaataa he. inhone bharat main kul baarah rashtriya prayogshaalaayein sthapit kin, jinamein pramukh is prakaar se hai:
 inaki mrityu ke uparaant, si.es.ai.aar ne kushal vaigyanikon hetu, inke sammaan main; bhatnagar puraskaar kii shuruyaat kii ghoshnaa kii. shanti swaroop bhatnagar ko vigyaan evam abhiyantriki kshetra main padam bhushan se 1954 main sammanit kiya gaya.
