 badri dutt pandey bhartiya swatantraaa senaanee the. saat varsh kii aayu main unke mata-pita kaa nidhan ho gaya. badri dutt pandey mul rup se almoda ke rahane vaale the. isliye mata-pita ke nidhan ke baad vah almoda aa gaye. almoda main hi unhone padhai kii. 1903 main unhone nainital main ek skool main shikshan kaary kiya. kuch samay baad dehradun main unki sarkari naukari lag gai, lekin jaldi hi unhone naukari se tyaagptr de diya or patrakarita main aa gaye.
 unhone 1903 se 1910 tak dehradun main leader naamak akhbaar main kaam kiya. 1913 main unhone almoda akhbaar kii sthapana kii. unhone is akhbaar ke jariye swatantraaa andolan ko gati dene kaa kaam kiya. isi kaaran kai baar angrej afasar is akhbaar ke prakaashan par roak laga dete the. almoda akhbaar ko hi unhone shakti akhbaar kaa rup diya. shakti saptahik akhbaar aaj bhi lagaataar prakashit ho rahaa he. 1921 main kuli begaar andolan main bidi pandey kii bhoomika ko hameshaa yaad kiya jaataa he. unhen kumaun kesari kii oopaadhi se bhi nawaja gaya. badri dutt pandey 1921 main ek saal, 1930 main 18 maah, 1932 main ek saal, 1941 main teen maah jail main rahe. 1942 ke bharat chodo andolan main bhi unhen jail bheja gaya. aazaadee ke baad bhi almoda main rahakar vah saamaajik kaaryon main sakriyata se hissaa lete rahe. 1957 main duusari loksabhaa ke liye hue chunaav main swatantraaa sangram senaanee hargovind pant chune gaye, lekin kuch hi maah main unka nidhan ho gaya. iske baad sitambar 1957 main hue upachunaav main congress ne badri dutt pandey ko pratyaashee banaaya or vah vijayi hue. badri dutt pandey bahut bebaak maane jate the. unhone swatantraaa sangram senaniyon ko milane vaali pension aadi kaa laabh bhi nahin liya. 1962 ke china yuddh ke samay apane saare medal, puraskaar aadi sarkaar ko bhent kar die. 13 farvari 1965 ko pandit badridatt pandeya kaa nidhan ho gaya.
