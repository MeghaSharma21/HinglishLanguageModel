 dilli public skool sosayadi bharat main schuli str par shiksha uplabdh karaane kii sabase bade sansthaanon main se ek he. isaki bharat main or videshon main apane niyantran main skulon kii ek badi sankhya he. dilli public skool sosayadi apane binar ke tahat sabhi sansthaanon ke liye prashasnic adhikaar he. kuch dilli public skool nai dilli main kendriya madhyamik shiksha board se sambhddha hai.aligarh, agra, elahabaad, asanasol, hyderabad, bikaner, jaipur, dhanbad, bokaro, bhopal, sonipat, mathura road, aar ke puram, ahmedabad, patna, vadodara, jodhpur, kanpur, udaipur main sthit sabase bade skulon ko milakar bharat main 150 se bhi zyda dilli public skool hai. 
 
 13 dilli public skool, bharat se baahar 10 vibhinn deshon main - sanyukt arab amirat, bahrain, indonesia, kuwait, nepal, katar, saudi arab, singapore, uganda or ghana main sthapit hai. 1949 main sthapit mathura road main sthit dilli public skool, sabase pehla dilli public skool tha. yah shikshavid reverend james duglas titalar dwaara sthapit kiya gaya tha. iske baad 1972 main aar ke puram main dilli public skool sthapit kiya gaya jise vyapak rup se samaaj ke pramukh deepies ke rup main manyata praapt he. deepies sosayadi kaa ek ghatak, deepies greater faridabad, faridabad or or uske aas paas ke ilaakon main bacchon ko ucch shiksha pradaan kar rahaa he. deepies ne sibisai kii sisies paathyakram par apni shiksha pranali kii modelling kii he. yah iti sector 81, faridabad main sthit he. 
 deepies nigaahi, dilli public skool singrouli ne agast 1998 se kaary karnaa aarambh kiya. yah dilli public skool sosayadi, nai dilli or uttari kolfields limited, nigaahi pariyojna ke bich ek sanyukt uddyam he. yah skool nau acre zamin par nigaahi pariyojna kii awasaiya colony main sthit he, evam ise paas ke stadium ke nau acre zamin kaa phayda bhi he. skool sibiesai, nai dilli se sambaddh he. yah skool dilli public skool sosayadi ke tatwadhan main sanchalit he, 1860 ke sosayadi panjikaran adhinium ke tahat darj. 
 dilli public skool, hyderabad 2001-2002 main shuru hua. yah khajaguda gram, golkunda main sthit he. skool ko 1860 ke sosayadi panjikaran adhinium ke tahat tatwadhan main chalaaya jaataa he. yahan ek varisht block, ek kanishtha block or uske paas sthit ek prashasnic block he. yahan juniurs ke liye ek chota sa khel kaa maidan bhi he. chaatro kii vibhinn ruchiyon ko dhyaan main rakhte hue yahan tennis court, basketbal court, swimming pool, shatranj, pustkaalay aadi maujuud hai. yahan kii pradhanacharya rekha agarwal hai evam is skool main nursery se barahavin tak kakshaayen hai.
