 abhrak ek bahupayogi khanij he jo aagney evam kayantarit chattaanon main khandon ke rup main paaya jaataa hai. ise bahut patali-patali paraton main chira ja saktaa he. yah rangarahit yaa halake pile, hare yaa kaale rang kaa hota he.
 abhrak ek jatil silicket yogic he. isamen potacium, sodium or lithium jaise kshareey padaarth bhi mili rahete hai. aagney chattaanon main praaia: abhrak paaya jaataa he. vaayu tathaa dhoop aadi se prabhavit hokar kabhi-kabhi silicket khanij bhi abhrak main badal jaataa he.
 abhrak ko do virgon main vibhajit kiya jaataa he: muscowite varg, biotite varg.
 in donon jatiyon ke mukhya khanij kramash: shwetabhrak tathaa krishnabhrak hai.
 purvokt donon prakaar ke khanijon ke gun lagbhag ek se hi hai. rasinic sanghathan main thodaa sa bhed hone ke kaaran inke rang main antar paaya jaataa he. shwetabhrak ko potacium abhrak tathaa krishnabhrak ko meganecium or louh abhrak kehete hai. shwetabhrak main jal kii maatra 4 se 6 pratishat tak vidyaman rahati he.
 abhrak varg ke sabhi khanij monoclinics samudaay main sphaty hote hai. adhiktar ye paratdaar akruti main paae jate hai. shwotabhrak kii paraten rangahin, athwa halke katthi yaa hare rang kii hoti hai. lohe kii vidyamanata ke kaaran krishnabhrak kaa rang kaalaapan liye hota he. in khanijon kii sataha chikani tathaa moti ke samaan chamakdaar hoti he. ek disha main in khanijon kii parton ko badi suvidha se alag kiya ja saktaa he. ye paraten bahut namya tathaa pratystha hoti hai. iskaa anumaan isi se lagaaya ja saktaa he ki yadi ham ek inch ke hajaarwein bhaag ke baraabar motaai kii parat len or use ek chauthai inch vyas ke belan ke aakaar main mode daalen to apni pratyasthata ke kaaran vah pun: falkar samtal ho jaaegi. in khanijon kii kathorta 2 se 3 tak he. thode se dabaab se yah nakhun se khurache ja sakate hai. inkaa aapekshik ghanatv 2.7 se 3.1 tak hota he.
 abhrak varg ke khanijon par amlon kaa koi prabhaav nahin padataa. abhrak aluminium tathaa potacium ke jatil silicket hai, jinamein vibhijh maatra main meganecium tathaa louh evam sodium, calcium, lithium, titanium, chromium tathaa anya tatv bhi praaia: vidyaman rahete hai. muscowite sarvaadhik mahatvapurn abhrak he. yadhyapi muscowite sarvaadhik saamaanya shilanirmaataa khanij he tathaapi iske nikshep, jinase upyogi abhrak praapt hota he, keval bharat tathaa brazil ke kuch simit kshetron main pigmetic patticoon main hi vidyaman hai. sampurna sansar kii aavashyaktaa kaa 80 pratishat abhrak bharat main hi miltaa he.
 abhrak ke utpaadan main china agraganya desh he, is ke pehle bharat kaa pratham sthaan ta yadhyapi yah canada, brazil, aadi deshon main bhi prachur maatra main praapt hota he, tathaapi vahaan kaa abhrak adhikanshat: chote aakaar kii paraton main athwa chure ke rup main miltaa he. badi staronvaale abhrak ke utpaadan main bharat ko hi ekaadhikar praapt he.
 abhrak kii patali-patali paraton main bhi vidyut rokane kii shakti hoti he or isi prakritik gun ke kaaran iskaa upyog anek vidyutyantron main anivaary rup se hota he. iske atirikt kuch anya udyogon main bhi abhrak kaa prayog hota he. biotite abhrak katipay aushadhiyon ke nirmaan main prayukt hota he.
 bihar kii abhrakapetica pashchim main gaya jile se hazaribagh tathaa munger hoti hui purab main bhagalpur jile tak lagbhag 90 mile kii lambaai or 12-16 mile kii chaudaai main faili hui he. iskaa sarvaadhik utpaadak kshetra koderma tathaa aasapaas ke kshetron main simit he. bhartiya abhrakashilaayein subhaj hai, gimen anek parivartan hue hai. abhrak mukhyat: pustak ke rup main praapt hota he. is samay bihar kshetra main 600 se bhi adhik choti badi abhrak kii khaane hai. in khaanon main anek kii gaharaai 700 foot tak chali gai he. bihar main atyuttam jati kaa laal abhrak paaya jaataa he jiske liye yah pradesh sampurna sansar main prasiddh he:.
 andhra main nellore jile kii abhrak petica dur tathaa sangam ke madhya sthit he. isaki lambaai 60 tathaa chaudaai 8-10 mile he. is petica main anek sthanon par abhrak kaa khanan hota he. yadhyapi adhikansh abhrak kaa varna haraa hota he, tathaapi kuch sthanon par "bengal rubi' ke samaan laal varna kaa kuch abhrak bhi praapt hota he.
 bhartiya abhrak ke utpaadan main rajasthan kaa dwitiya sthaan he. rajasthan kii abhramaky petica jaipur se udaipur tak faili he tathaa usamen pigmetic milte hai. rajasthan se praapt abhrak main se keval alpaansh hi ucch koti kaa hota he; adhikansh main yaa to dhabbe hote hai athwa paraten tuuti yaa mudi hoti hai.
 bihar, rajasthan or andhra ke vishaal abhrakakshetron ke atirikt kuch muscowite bihar ke manbhoom, singhbhum tathaa palamau jilon main bhi miltaa he. isi prakaar adhovarg kaa kuch abhrak urisa ke sambalpur, aangul tathaa dhenkanal main paaya gaya he. andhra main kadapa, tathaa madras main salem, malabar tathaa nilgiri jilon main bhi abhrak ke nikshep hai, kintu ye adhik mahatv ke nahin. mysore ke hasan tathaa mysore or pashchim bangal ke medinipur tathaa bankura jilon main bhi alp maatra main abhrak paaya gaya he.vartmaan main abhrak utpaadan main andhrapradesh,rajasthan agrani he.
 yadhyapi desh main abhrak prachur maatra main paaya jaataa he, tathaapi iskaa adhikansh kachche maal ke rup main videshon ko bhej diya jaataa he. hamaare apane udyog main isaki khapat praaia: nahin ke baraabar he. isamen andeh nahin ki adhik maatra niryaat ke kaaran is khanij dwaara videshi mudra kaa uparjan yathesht ho jaataa he, kintu yadi isko desh main hi parishkrut padaarth ke rup diya ja sake to or bhi adhik aay hone kii sambhaavna he.
 vyapaar kii drushti se abhrak ke do khanij shwetabhrak or flogopite adhik mahatvapurn hai. abhrak kaa prayog badi-badi chaadaron ke rup main tathaa chote-chote yaa churna ke rup main hota he. badi-badi paratonvala abhrak mukhyatya vidyut udyog main kaam aata he. vidyut kaa asanvahak hone ke kaaran iskaa upyog condencer, commutator, telephone, dianemo aadi ke kaam main hota he. paardarshak tathaa taparodhak hone ke kaaran yah lamp kii chimney, stove, bhatthiyon aadi main prayukt hota he. abhrak ke chote-chote tukdon ko chipkaakar micanite banaaya jaataa he. abhrak ke chote-chote tukade rabad udyog main, rang banane main, mashiinon main chiknayee dene ke liye tathaa maanapatron aadi kii sajaavat ke kaam aate hai.
 abhrak ooshmaa tathaa vidyut kaa kuchaalak he. yahi gun iske vyapaarik mahatv kaa aadhaar he. pavanmaapi yantra tathaa uttam koti ke darpan abhrak kii sahaayta se banaae jate hai. byler ke jacket ke aavaran banane main bhi iskaa upyog hota he. vidyutyantra tathaa upkaran, jaise dianamo, armechar, heater, telephone ke dial banane main bhi iskaa upyog hota he. radio, vayuyan tathaa motor ingine ke purjon main bhi abhrak kaa upyog badhata ja rahaa he. isse khaad bhi banaai jaati he.
 abhrak paardarshak hota he. saath hi taap ke aakasmik utaar-chadhaav kaa bhi asapar adhik asar nahin hota he. isliye yah bhattiyon main agninirodhak plasters karne ke kaam aata he. rangahin paardarshak kaagaj, vibhijh prakaar ke khilone, rangamanch ke paradon kii sajaavat tathaa chamkile paint color bhi abhrak kii sahaayta se banaae jate hai.
 ayurveda chikitsa main abhrak bhasma kaaphi prachlit aushadhi he jo kshay, prameh, pathari aadi rogon ke nidaan main prayukt hoti he.
 rasshastra main abhrak ko 'maharas' kaha gaya he. kaale rang kaa abhrak ayurvedic aushadhi ke kaam main lene kaa aadesh he. saadhaarant: agni kaa isapar prabhaav nahin hota, phir bhi ayurveda main iskaa bhasma banane kii reetiyaan hai. yah bhasma sheethal, dhatuvardhak or tridosh, vishavikar tathaa krimidosh ko nasht karanevaala, deh ko drudh karanevaala tathaa apurv shaktidaayak kaha gaya he. kshay, prameh, bavasir, pathari, mutraghaat ityadi rogon main yah laabhadaayak kaha gaya he.
