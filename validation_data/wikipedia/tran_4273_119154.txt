 1936 ke baad se prayek sheetakaaleen olympic main alpine skiing kaa chunaav hua he, jab ek sanyukt aayojan germany ke garmish-partenkirchen main yojit kiya gaya tha.
 1948 se 1980 tak, sheetakaaleen olympic ne olympic varshon main vishwa championships ke rup main bhi kaam kiya tha, yahaan tak ​​ki alag-alag gaiyr-olympic khelon main yojit vibhinn pratiyogitaaon ke saath. is avadhi ke dauraan, olympic padak vijetaon ko antarrashtriya ski federation se ek hi rang kaa ek atirikt padak mila.
 1950 vishwa championships or 1952 main olympic main vishaal slalam kii shuruyaat hui thi; donon kaaryakramon ne sanyukt aayojan ko chhod diya, lekin 1954 main vishwa championships main slalam, vishaal slalam or downhill ke parinaamon kaa upyog karte hue "paper" res ke rup main laut aayaa. 1956 se 1980 tak olympic main, vishwa championships padak sanyukt kaarykram main exs dwaara sammanit kiya gaya. yah 1988 main olympic main ek stand-alon ghatana ke rup main laut aayaa, jisane ek ran super-ji bhi shuru kiya. sanyukt aayojan 1992 ke madhyam se olympic main exs ank pranali par chalaaya gaya tha, phir teen ranon ke kul samay main badal diya gaya tha. sanyukt super 2010, jo ek ran or ek din ke liye ghatana ke liye slalam bhaag kam main pehali baar aayaa.
 1985 se, shatranj olympic se swatantra hone ke baavjood vishwa championships ko har ajeeb-ginati varsh nirdhaarit kiya gaya he. vishwa championships main, sanyukt 1982 main ek stand-alon samaaroh ke rup main laut aayaa or 1987 main super-ji kii shuruyaat hui. sanyukt ghatana 1996 main kul samay se, or 2007 main super sanyukt ke liye badal gai.

 ^ germany ne vibhinn naamon ke tahat sheetakaaleen olympic khelon main hissaa liya he; jinamein se sabhi ko alag se soochibaddh kiya gaya he. 1948 ke khelon se pratibandhit, ve 1952 main germany ke rup main pratispardhaa karte the, haalanki keval pashchim germany dwaara iskaa pratinidhitiv kiya jaataa tha. 1956 se 1964 tak, unhen germany kii uniphid team ke naam se janaa jaataa tha, jisamen purvi germany, german loktantric ganraajya shaamil tha. 1968 se 1988 tak, alag olympic tiimon ko maidan main utaara gaya or pashchim germany ne sangheey ganraajya germany ke rup main pratispardhaa kii. purv germany main alpine skiing main keval ek pratiyogi the, jinhone padak nahin kiya tha. 1990 ke punarmilan ke baad, unhone 1992 se shuru hone vaale olympic main germany main hissaa liya he. 2014 ke madhyam se germany main alpine skiing main kul 40 padak hai.
