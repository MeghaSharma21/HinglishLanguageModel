 kishan maharaj yaa pandit kishan maharaj vikhyath tabalaa vadak the.
 kishan maharaj kaa janm kaashi ke kabirchaura muhalle main 1923 main paaramparik rup se ek sangitajna ke parivaar main hua. krishna janmashtami par aadhi raat ko janm hone ke kaaran unka naam kishan padaa. unhone apane praarambhik varshon main pita pandit hari maharaj se shastreey sangeet kii shiksha praapt kii. pita ke dehanth ke baad unke chachaa evam pandit baldev sahay ke shishy pandit kanthe maharaj ne unki shiksha kaa kaarybhaar sambhala.
 kishan maharaj ne table kii thap kii yatra shuru karne ke kuch saal ke andar hi ustad faiyyaaj khan, pandit onkaar thakur, ustad bade gulam ali khan, pandit bhimsen joshi, vasant raay, pandit ravi shankar, ustad ali akbar khan jaise bade naamon ke saath sangat kii. kai baar unhone sangeet kii mehfil main ekal tabalaa vaadan bhi kiya. itnaa hi nahin nrity kii duniya ke mahaan hastakshar shambhu maharaj, sitara devi, nataraj gopi krishna or birju maharaj ke kaaryakramon main bhi unhone table par sangat kii.unhone edinburgh or varsh 1965 main britain main kaamanwealth kala samaaroh ke saath hi kai avasaron par apane kaarykram prastut kar pratishtha arjeet kii. unke shishyon main vartmaan samay ke janemaane tabalaa vadak pandit kumaar bose, pandit baalkrishna ayyar, sukhwinder singh namdhari sahit anya naam shaamil hai.
 kishan maharaj ko varsh 2002 main padmavibhushan se sammanit kiya gaya tha. unhen varsh 1973 main padmashri, varsh 1984 main kendriya sangeet naatak puraskaar, varsh 1986 main ustad inayat ali khan puraskaar, deenanath mangeshkar puraskaar or taal vilaas ke alaava uttarapradesh ratna, uttarapradesh gaurav bhojpuri ratna, bhagirath sammaan or life time achievement sammaan se bhi nawaja gaya.
