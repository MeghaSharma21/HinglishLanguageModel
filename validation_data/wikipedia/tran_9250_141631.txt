 bharat kii nadiyon kaa desh ke aarthik evam saanskrithik vikaas main prachinkaal se hi mahatvapurn yogadan rahaa he. sindhu tathaa ganga nadiyon kii ghaatiyon main hi vishwa kii sarvaadhik pracheen sabhyataaon - sindhu ghati tathaa arya sabhyataa kaa arvibhaav hua. aaj bhi desh kii sarvaadhik jansankhya evam krishi kaa sankendran nadi ghati kshetron main paaya jaataa he. pracheen kaal main vyapaarik evam yatayaat kii suvidha ke kaaran desh ke adhikansh nagar nadiyon ke kinare hi viksit hue the tathaa aaj bhi desh ke lagbhag sabhi dharmik sthal kisi na kisi nadi se sambaddh he.
 nadiyon ke desh kahe jane vaale bharat main mukhyatah chaar nadi pranaliyaan he hai. uttari bharat main sindhu, madhya bharat main ganga, uttar-purv bharat main brahmaputra nadi pranali he. prayadwipiya bharat main narmada kaveri mahanadi aadi nadiyaan vistrut nadi pranali kaa nirmaan karti hai.
