 wiliam morris davis amareeki bhugolvetta tathaa bhuvaigyanik the. davis ne bhu-akritivigyaan ke kshetra main gaveshanaon or siddhanton par kai pustaken tathaa kai sau shodhpatr likhe the. vah apradanchakr ke siddhanth kaa janmdata tha.
 wiliam morris davis kaa janm sanyukt rajya america ke pensiolwania rajya main philadelphia nagar main 2 farvari 1850 main hua. 1869 i0 main harvard vishvavidhyalay main shiksha samaapt karke san 1770 se 73 tak ajretina kii carbona sthit rashtriya vedhashala main jyotirvid ke rup main inhone kaary kiya. 1876 i0 main inhone harvard vishvavidhyalay main adhyayapan kaary prarambh kiya or 1890 i0 main bhautik bhugol ke acharya niyukt hue. tatpashchaat 1899-1912 i0 tak inhone vahiin par bhautakritiktatv ke acharya ke rup main kaary kiya. 1903, 1904 tathaa 1909 i0 main ye amareeki bhugolvettaon ke santhaan ke adhyaksh rahe. san 1911 main inhen amareeki bhuvigyaan kaa bhi adhyaksh banaaya gaya. inhone berlin vishvavidhyalay tathaa sarbon main bhi adhyaapan kaary kiya.
 davis amareeki bhautakritikttatv vigyaan ke janak maane jate hai. inhone uparyukt vigyaan main pehle kiye gaye kaaryo ko saidhantik praturup diya tathaa vaigyanik shabdaawalee pradaan kii. inhone bhautakritik tatwon ke vikaas ke adhyayan main sanrachana, prakriya tathaa avasthaa ko mahatvapurn bataaya he or apaksharan chakra ke prakaran main bhautakritik tatwon ke adyopant vikaas ke kramon ko kishoravastha, praudhavastha tathaa vriddhavastha kii sangyaayen di hai. inke anusaar is kramik vikaschakr kaa kabhi ant nahin hota. at: uttar vriddhavastha main samtal saddrishy jis bhautakruti kaa vikaas hota he, use inhone praaia: samabhumi tal kii sangya di. inhone jal apaksharan ke atirikt himandiyon tathaa shuddh pradeshy bhautakritik tatwon par bhi shodhapurna niabandh likhe or pravalanirmaan ke prakaran main mahatvapurn siddhant pratipaadit kiye. apane jivan ke lambey kaal main inhone lagbhag 400 pustaken tathaa niabandh likhe. bhautakritik tatv vigyaan ke atirikt, jise ve bhugol kii ek shakha maanate the, bhugarbha vigyaan main bhi inhone mahatvapurn kaary kiye. bhaimyakriti siddhant par unki pratham pustak german bhashaa main prakashit hui. iske atirikt unki rachanaaon main "bhugolic niabandh", "praarabhik ritu vigyaan" bhautik bhugol tathaa coral rif problem mahatvapurn hai.
