 
  hindi kii ek vigyaan patrika he. vigyaan pragati ne hindi vigyaan patrakarita main ek mahatvapurn mile kaa pathar sthapit kiya he.
 'vigyaan pragati' hindi main vaigyanik evam takneekee vishyon par adhyatan saamagree dene vaali masik patrika he. iskaa prakaashan san 1952 se armbh hua. iske prakaashan kaa mul uddeshy siesaiaar kii prayogshaalaon main hone vaale vaigyanik anusandhanon ke bare main soochna dena tha. 
 swatatrata prapti ke baad bharat main jab hindi ko samuchit sthaan dilaane ke prayaas kae ja rahe the tab vaigyanik tathaa audyogic anusandhan parishad ne jan-maanas main hindi bhashaa main vigyaan ke prachaar-prasar ke liye ek mahatvapurn evam doordarshee nirnay liya jiske falswaroop varsh 1952 main vigyaan pragati naamak lokapriy vigyaan patrika ke prakaashan kaa kaary prarambh hua. aarambh main yah patrika ek newzlator ke rup main, siesaiaar kii prayogshaalaon main kiye ja rahe vaigyanik anusandhanon ke bare main suchanaaen prakashit karti thi. baad main iskaa prakaashan jan-maanas main vaigyanik drushtikon jagaane tathaa andhvishwaas se chhutkaaraa dilaane ke liye ek lokapriy vigyaan patrika ke rup main kiya jane laga. tab se lekar aaj tak apane naam ke anuroop vigyaan pragati bharat ke logon ko vigyaan kii pragati ke bare main jaankaree detii aa rahi he. agast 2010 main patrika ne apana 675waan ank prakashit kiya. janvari 2011 yah lokapriy vigyaan patrika apane prakaashan ke 60wein varsh main pravesh kar gayi. 
 apana uddeshy 'jan-maanas ke liye vigyaan' ke anusaar vigyaan pragati bhartiya vigyaan evam praudyogiki ke yogadan ko maanav kalyaan tathaa uske samaaj par prabhaav ke vibhinn kshetron par jaankaree pradaan karti he. is patrika main apana amuly sahyog dene waalon main shri shivaraj patil, purv loksabhaa adhyaksh; dow es. jed. kasim, purv sadsy yojana ayog; dow aar ek marshalakar, purv mahanideshak siesaiaar; pro. e. rahamaan, purv nideshak nistads; pro. yashpal; tathaa dow en. vitthal sammilit hai.
 vigyaan pragati vibhinn samayik vishyon par visheshankon kaa prakaashan bhi karti he. vigyaan pragati kii lokapriyata kaa anumaan is baat se lagaaya ja saktaa he ki isaki prasar sankhya lokapriy vigyaan patrikaon main se sarvaadhik he. desh ke dur-daraaj ke kshetron main is patrika ke 10-11 lakh se zyada pathak honge. videshon main bhi vigyaan prageet ke pathak uplabdh hai.
