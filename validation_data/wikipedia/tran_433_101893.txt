 bhartiya cricket team bharat kii rashtriya cricket team he. bhartiya cricket niyantran board dwaara sanchalit bhartiya cricket team antarrashtriya cricket parishad kii purnakalik sadsy he. bhartiya team do baar cricket vishwacap apane naam kar chuki he. vartmaan main bhartiya cricket team ke coach ravi shastri hai.
 yadhyapi bharat main cricket 18 veen sadi main europea vyapaaree navikon dwaara laayaa gaya tha, or bharat main pehla cricket club 1792 main kalakatta main sthapit kiya gaya tha parantu rashtriya cricket team ne apana pehla match lords main 25 june 1932 ko khelaa. apane pehle test match khilne ke saath hi vishwa main test team kii haisiyat paane vaali chhathvi team ban gayi. apane pehle 50 varshon main team ne bahut hi kamjor pradarshan kiya, 196 test maichon main se keval 35 match main hi jeet darj kara pai. 1970 ke dashak se bhartiya cricket team ek shaktishaalee team banakar ubhari. 1983 main kapil dev ke netritv main westeindies ko haraakar vishwacap apane naam kiya. saurav ganguli kii captaini main 2003 main upavijeta rahi evam 2011 main mahendra singh dhoni kii captaini main bharat ne duusari baar vishwacap jeeta.

 sachin tendulkar jinhone 16 varsh kii umr main 1989 main bhartiya cricket team ke liye cricket khelnaa prarambh kiya tha jo baad main sabase zyada ran banane vaale, sabase zyada shatak lagane vaale tathaa ek divasiy antarrashtriya cricket main bhi pehle khilaadi bane jinhone doharaa shatak lagaaya ho. vanade tathaa test cricket main sabase zyada record bhi inhen ke naam he. bhartiya cricket team kii ore se test cricket main ek pari main sabase zyada ran banane kaa record virendra sahvaag kaa he jinhone chennai main 319 kii pari kheli thi jo ki pehla bhartiya tihara shatak tha. sahvaag ne inase pehle 309 ranon kii pari kheli thi jo pakistan ke khilaaf banaaya tha. test cricket main bharat kaa sabase bada score 726/9 he jo 2009 main mumbai ke breaborn stadium main srilanka ke khilaaf banaaya tha jabaki sabase nunatam score 42 ran he jo england ke khilaaf 1974 main banaaya tha.
 ek divasiy antarrashtriya main bharat kaa sabase bada score 418/5 he jo 2011-2022 main indore main west indies ke khilaaf banaae the. jabaki pehali baar 400 yaa isse zyada ran 2007 cricket vishwa cap main west indies main bermuda ke khilaaf 413/5 ran banaae the. isi match ko 257 ranon se jitkar ranon se sabase badi jeet hasil kii thi. bharat kii ore se vanade cricket main ek match main sabase zyada ran banane kaa record openar ballebaaj rohit sharma kaa he jinhone srilanka ke khilaaf 2012 main kolkata ke eden gardens maidan main 264 ranon kii pari kheli thi inke alaava rohit sharma ve pehle khilaadi he jinhone 3 baar dohare shatak lagaae ho. 
 gendbaaji ke maamle main bhi bharat ke kai gendbaajon ne record banaae he. anil kumble sheersh teen gendbaajon main shaamil he jinhone 600+ wicket liye ho. 1999 main gym lekar ke anil kumble test match kii ek pari main 10 ke 10 wicket lene vaale khilaadi bane jo ki pehle bhartiya he. yah record kumble ne 1999 main dilli ke firoz shah kotla main pakistan ke khilaaf banaaya tha.

 bhartiya cricket team main lagbhag athaees captain ho chuke he jinhone kam se kam ek test cricket main captaini kii ho. jinamein se sirf chah aise khilaadi he jinhone kam se kam 25 yaa inase adhik maichon main captaini kii ho, inke alaava paanch aise bhi he jinhone keval vanade cricket main captaini kii he. bharat kaa sabase pehla captain sike naidu the jinhone pehla match england cricket team ke khilaaf khelaa tha. inke alaava 1933-34 main 3 test maichon kii shrinkhalaa hui thi. lala amarnath bharat ke chauthe captain the inhone pehla match bhartiya swatantraaa ke pashchaat khelaa tha. lala amarnath kii captaini main hi bharat ne pehla test match or pehali test shrinkhalaa jiti thi jo 1952-53 main pakistan ke khilaaf khele gaye the. 
 bhartiya cricket team ke nawab mansoor ali khan pataudi jinhone 1961 se 1962 tathaa 1939 se 1970 tak 36 test maichon main captaini kii thi, inke baad inhen 4 maichon ke liye 1979 main waapas captain banaaya tha. ajit wadekar kii captaini main bharat ne pehla ek divasiy antarrashtriya match 1984 main jeeta tha.
