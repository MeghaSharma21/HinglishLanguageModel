 takneekee drushti se software teen prakaar ke hote hai. 
 “system software” yah ek aisa program hota he, jinka kaam system arthath computer ko chalaanaa tathaa use kaam karne laayak banaae rakhana he. system software hi hardware main jaan daalata he. oparating system, compiler aadi system sofyware ke mukhya bhaag hai.
 ‘aplicoshan software’ aise prograamon ko kaha jaataa he, jo hamaare computer par aadhaarit mukhya kaamon ko karne ke liye likhe jate hai. aavashyaktaanusaar bhinn-bhinn upyogon ke liye bhinn-bhinn software hote hai. vetan kii gananaa, len-dein kaa hisaab, vastuon kaa stock rakhana, bikri kaa hisaab lagaana aadi kaamon ke liye likhe gaye program hi aplicoshan software kahe jate hai.
 software kanoon main bauddhik smapda adhikaar ke andar surakshit hota he. trips main saat prakaar ke bauddhik smapda adhikaar ke bare main charchaa kii gayi he isamen teen prakaar ke adhikaar, yani kii copirite, trade sikret, tathaa patent, computer software ko prabhavit karte hai. software ko patent karaane kaa mudda vivaadaaspad he tathaa kuch kathin bhi. isaki charchaa ham alag se patent evam kamputer software ke andar kii gai he.
 object code computer ko chalaata he or yah sadaa prakashit hota he, parantu kya yah kisi chiz kaa varnan he athwa nahin is bare main shankaa thi. trips ke samjhaute ke andar yah kaha gaya ki computer program ko copirite ke samaan surakshit kiya jaay. isaliye object code hamaare desh main tathaa sansar ke anya deshon main isi prakaar se surakshit kiya gaya he.
 computer program ke object code to prakashit hoten hai par sabake sorce code prakashit nahin kiye jate hai. jin kamputer program ke sorce code prakashit kiye jate hai unamem to ve copirite se surakshit hote hai. par jin computer program ke sorce code prakashit nahin kiye jate hai ve trade sikret kii tarah surakshit hote hai.
 sorce code se object code compail karnaa aasaan he; yah hameshaa kiya jaataa he or isi tarah program likhaa jaataa he. par iskaa ulta yani ki object code se sorce code maaloom karnaa asambhav to nahin par bahut mushkil tathaa mahangaa he. is par reverse ingeneering kaa kanoon bhi lagu hota he. isi liye sorce code prakashit nahin kiya jaataa he. ise gopneey rakh kar, ise zyada aasaani se surakshit kiya ja saktaa he. reverse ingeneering bhi rochak vishay he, iske bare par phir kabhi.
