 kaaman getwe interphes CGI ek bahari aplicoshan software kaa soochna dene vaale sarwar ke saath sampark saadhane ke liye ek maanak protocol he, jo ki aamtaur par ek web sarwar hota he.
 is soochna dene vaale sarwar kaa kaary, output/parinaam ke dwaara anurodh kaa jawaab dena he. har baar ek anurodh praapt hone par, sarwar anurodh kaa vishleshan karata he or uchit uttar detaa he. aisa karne ke liye sarwar dwaara nimnlikhit do buniyadi tareeke prayukt kiye jate hai:
 CGI duusare tareeke ko karne kaa maanak dhang paribhaashit karata he. yah paribhaashit karata he ki kaise sarwar ke bare main jaankaree or anurodh, tark tathaa sthiti ke anusaar veriable ke rup main bheje jate hai or kaise cmand hader ke rup main parinaam kii atirikt jaankaree bhej sakate hai.
 1993 main world wide web simit, kintu teji se bad rahaa tha. WWW software developers or web site developers www-tok meling suchi ke dwaara ek duusare ke sampark main rahete the, isliye is tarah sampark saadhane ke liye cmand line chalaane ke ek maanak par sahmati hui. CGI vivarani main vishesh rup se varnit yogadanakarta nimnlikhit hai:
 rob mckul ne praarambhik visheshtaon kaa masauda taiyaar kiya or NCSA ise ab bhi host karata he. yah bahut se sarvaron main teji se lagu kiya gaya tha.
 CGI kaarykram kaa ek udhaaharan ek waiki ko lagu karnaa he. yuzar agent ek pravishti ke naam kaa anurodh karata he; sarwar pravishti ke prishth se uskaa strot praapt karegaa, ise HTML main badalegaa or parinaam bhejega.
 web sarwar ke nazariye se, kuch loketor, jaise http://www.example.com/wiki.cgi, ek program main CGI dwaara chalane ke liye paribhaashit kiye gayi hai. jab URL ke liye anurodh praapt hota he to is se sambandhit kaarykram chalaaya jaataa he.
 data sthiti ke anusaar veriable kaa prayog kar ke program main daalaa jaataa he. yah vishisht chaalan ke vipareet he, jahaan cmand line tark prayog kiye jate hai. HTTP PUT yaa POST, ke maamle main, upyogkarta dwaara daalaa gaya data maanak input ke madhyam se program ko uplabdh karaayaa jaataa he.
 web sarwar main diarectory tree ke aadhaar par ek cgi-bin diarectory hoti he jo CGI dwaara maangi jane vaali chalaane yogya filon kaa sangrahan rakhti he.
 program web sarwar par maanak uttar ke rup main parinaam bhejtaa he, jiske shuru main ek hader or ek rikt pankti hoti he.
 hader usee tarah se kutabaddh hota he jaise ek HTTP hader or isamen bheje gayi dastauez kaa MIME prakaar bhi shaamil hona chahie. hader aamtaur par parinaam ke saath upyogkarta ko aage bheje jate hai, jise web sarwar dwaara pradaan kiya jaataa he.
 yadi kisi maamle main CGI kol main scrypting bhashaa prayukt hoti he jaise ki csh yaa parl, to koding trutiyon ke parinamswaroop code injection jokhim kii sambhaavna bad jaati he.
 cmand colling kaa aam taur par arth he ek nav nirmit prakriya kaa aahvaan. prakriya ko shuru karne main, parinaam utpann karne ke vaastavik kaary kii apeksha adhik samay tathaa memory kharch ho sakati he, khaskar jab program ko abhi interpritate yaa compail karnaa baaki he. yadi program ko aksar chalaaya jaataa he to kaam ke boj se web sarwar jaldi nasht ho sakate hai. baharhaal, is boj ko compail kiye hue CGI prograamon kaa prayog kar ke kam kiya ja saktaa he jaise ki CGI ke saath parl yaa anya scrypting bhashaaon ke sthaan par C/C++, kaa prayog kiya ja saktaa he. pradarshan se sambandhit aise mudde aam taur par keval un mashiinon par ek chintaa kaa vishay he jin par atyaadhik kaarybhaar hota he.
 iskaa ilaaj karne ke liye kai drushtikon apanaae ja sakate hai:
 kisi bhi web aplicoshan ke liye sarvshereshth seoup aplicoshan ke vishisht vivaran, yatayaat kii maatra or lenden kii jatilta par nirbhar karata he, kisi bhi die gayi kaam or samay ke bajet kaa kriyanvayan nirdhaarit karne ke liye in vikalpon kaa vishleshan karne kii aavashyaktaa he.
