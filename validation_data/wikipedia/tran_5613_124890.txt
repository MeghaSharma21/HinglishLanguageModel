  brajabhasha mulat: braj kshetra kii boli he. | jiske pashchim se yah rajasthani kii up-bhashaaon main jaakar mil jaati he. 
 haryana main yah dilli ke dakshini ilaakon main boli jaati he- faridabad jila or gudgaanv or mevaat jilon ke purvi bhaag.
 iskaa vikaas mukhyat: pashchimi uttar pradesh or usase lagate rajasthan va madhya pradesh main hua. mathura, bharatpur,hindun city, dhaulpur, agra, gwalior aadi ilaakon main aaj bhi yah mukhya samvaad kii bhashaa he. is ek puure ilaake main brijbhasha yaa to mul rup main yaa halke se parivartan ke saath vidyaman he. isiliye is ilaake ke ek bade bhaag ko brijanchal yaa brijbhoomi bhi kaha jaataa he. 
 bhartiya aryabhashaon kii parampara main viksit honevaali "brajabhasha" shauraseni apbhransh kii kokh se janmi he.jab se gokul vallabh sampradaay kaa centre bana, brajabhasha main krishna vishyak sahitya likhaa jane laga. isi ke prabhaav se braj kii boli saahitiyik bhashaa ban gai. bhaktikaal ke prasiddh mahakavi mahatma surdaas se lekar aadhunik kaal ke vikhyath kavi shri viyogi hari tak brajabhasha main prabandh kavy tathaa muktak kavy samay samay par rache jate rahe.
 janapadiya jivan ke prabhaav se brajabhasha ke kai rup hamen drushtigochar hote hai. kintu thode se antar ke saath unamem ekarupata kii spasht jhalak hamen dekhne ko milti he.
 brajabhasha kii apni rupagat prakruti aukarant he arthath isaki ekvachaneey puling sangyaayen tathaa visheshan praaia: aukarant hote hai; jaise khurpu, yamrau, maanjhu aadi sangya shabd aukarant hai. isi prakaar karau, goru, saanwarau aadi visheshan pad aukarant he. kriya kaa saamaanya bhootkaalik ekvachan puling rup bhi brajabhasha main pramukhroopen aukarant hi rahata he. yah baat alag he ki uske kuch kshetron main "y" shruti kaa aagam bhi paaya jaataa he. jila aligarh kii tehsil kol kii boli main saamaanya bhootkaaleen rup "y" shruti se rahit miltaa he, lekin jila mathura tathaa dakshini bulandshahr kii tahseelon main "y" shruti avashya pai jaati he. jaise :
 ""karau chhora bolau"" -.
 ""karau chhora bolgau"" -.
 ""karau launda bolgau"" -.
 kannauji kii apni prakruti okarant he. sangya, visheshan tathaa kriya ke roopon main brajabhasha jahaan aukarantata lekar chalati he vahaan kannauji okarantata kaa anusaran karti he. jila aligarh kii jalapadiya brajabhasha main yadi ham kahen ki- ""karau chhora bolau"" to ise hi kannauji main kahenge ki-""kaaro larika bolo. bhavishyatkaleen kriya kannauji main tidhntaroopini hoti he, lekin brajabhasha main vah kridantarupini pai jaati he. yadi ham "ladkaa jayega" or "ladki jaaegi" vaakyon ko kannauji tathaa brajabhasha main roopantarit karke bolen to nimnaankit rup pradaan karenge :
 kannauji main - larika jaihai. bitiyaa jaihai.
 brajabhasha main - chhora jaigau. chhori jaigi.
 uparyukt udaahranon se spasht he ki brajabhasha ke saamaanya bhavishyat kaal rup main kriya karta ke ling ke anusaar parivartit hoti he, jab ki kannauji main ek rup rahati he.
 iske atirikt kannauji main avadhi kii bhanti vivruti kii pravriti bhi pai jaati he jiskaa brajabhasha main abhaav he. kannauji ke sangya, sarvnaam aadi vaakyapadon main sandhirahitya praaia: miltaa he, kintu brajabhasha main ve pad sandhigat avasthaa main milte hai. udhaaharan :
  kannauji -""bau gao"" .
  brajabhasha -""bo gayu"" .
 uparyukt vaakyon ke sarvnaam pad "bau" tathaa "bo" main sandhirahitya tathaa sandhi kii avasthaaen donon bhashaaon kii prakrutiyon ko spasht karti hai.
 brajabhasha kshetra kii bhashaagat vibhinnata ko drushti main rakhte hue ham uskaa vibhaajan nimnaankit rup main kar sakate hai :
  kendriya braj arthath adarsh brajabhasha - aligarh, mathura tathaa pashchimi agre kii brajabhasha ko "adarsh brajabhasha" naam diya ja saktaa he.
  budeli prabhavit brajabhasha - gwalior ke uttar pashchim main boli janevaali bhashaa ko yah naam pradaan kiya ja saktaa he.
  rajasthan kii jaipuri se prabhavit brajabhasha - yah bharatpur tathaa uske dakshini bhaag main boli jaati he.
  sikarwari brajabhasha - brajabhasha kaa yah rup gwalior ke uttar purv ke anchal main prachlit he jahaan sikarwaad rajputon kii bastiyaan pai jaati hai.
  jaadobati brajabhasha - karauli ke kshetra tathaa chambal nadi ke maidan main boli janevaali brajabhasha ko "jadobari" naam se pukaara gaya he. yahan jaadu rajputon kii bastiyaan hai.
  kannauji se prabhavit brajabhasha - jila eta tathaa tehsil anupshahar evam atarauli kii bhashaa kannauji se prabhavit he.
 brajbhaashi kshetra kii janapadiya brajabhasha kaa rup pashchim se purv kii ore kaisaa hota chalaa gaya he, iske liye nimnaankit udhaaharan drashtavy hai :
 jila gudgaanv main - ""tamaso dekhne ku gaye. aaps main jhagro ho rahyu ho. tab gaano band ho gayo.""
 jila bulandshahr main -""launda gom ku ayu or bahu su bolgau kai main naukri ku jangau.""
 jila aligarh main - ""chhora gaanm kuun ayu auru bau te bolau kai main naukri kuun jango.""
 jila eta main - ""chhora gom kuun aao or bau te bolo kai main naukri kuun jaaungo.""
 isi prakaar uttar se dakshin kii ore kaa parivartan drashtavy he-
 jila aligarh main -""gu chhora mere ghar te chalau gayu.""
 jila mathura main -""bu chhora mere ghar tain chalyu gayu.""
 jila agra main -""muktau rupaia apni bari kuun bheji dayu.""
 gwalior main - "baanain ek bokara paal lao. tab bau anand sai raibe lago.""
 brajabhasha ek saahitiyik evam swatantra bhashaa he or isaki apni kai boliyaan hai.
