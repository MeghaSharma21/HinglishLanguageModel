 sarasadaas sarasadaas haridasi sampradaay ke bhakt kavi evam pancham acharya the. 
 sarasadaas nagaridas ke chote bhai tathaa bengal ke raja ke mantri kamalapati ke duusare putr the. inkaa janm  vi 0 san 0 1611 main goud brahman kul main hua tha.ye swabhaav se virakt the atah apane bade bhai nagaridas ke saath hi vrindavan aa gaye or biharindev se diksha grahan kii. diksha ke uparant shesh jivan vrindavan main hi vyateet hua. vi 0 san 0 1670 main apane bade bhai nagaridas ke paralok-gaman ke uparant ye tatti santhaan kii gaddi par baithe. vi 0 san 0 1683 main inkaa dehavsaan ho gaya.
 inke pad ,kavitt or savaiye sfoot rup main uplabdh hai. inaki samast vaani siddhant ke pad or shringaar ke pad do  bhaagon main vibhakt he.
 sarasadaas ke upasya shyama-shyaam aanandanidhi ,sukhnidhi gunanidhi or lavanya-nidhi hai. ye sadaa  prem main matt main leen rahete hai. radha-krishna ke isi vihaar kaa dhyaan sarasadaas kii upasna he. kintu yah upasna sabhi ko sulabh nahin he. swami haridas jis par krupa karen vahi praapt kar saktaa he.
